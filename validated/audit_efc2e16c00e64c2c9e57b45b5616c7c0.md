### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Contract Freezing — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS accepts any arbitrary felt value as the new class hash without verifying that the hash corresponds to a declared class. A contract can call `replace_class` with an undeclared or zero class hash, permanently rendering itself unexecutable. Any funds held in that contract are irreversibly frozen, because no future transaction calling the contract can be included in a provable block.

---

### Finding Description

`execute_replace_class` reads the requested class hash directly from the syscall request and writes it into `contract_state_changes` with no validation:

```cairo
// execute_replace_class (syscall_impls.cairo, lines 877–916)
let class_hash = request.class_hash;

// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
``` [1](#0-0) 

The in-code TODO comment explicitly acknowledges the missing check. [2](#0-1) 

When a subsequent transaction attempts to call the affected contract, `execute_entry_point` performs:

```cairo
let (compiled_class_hash: felt) = dict_read{dict_ptr=contract_class_changes}(
    key=execution_context.class_hash   // now the invalid/zero hash
);
// ...
let (compiled_class_fact: CompiledClassFact*) = find_element(
    array_ptr=compiled_class_facts_bundle.compiled_class_facts,
    elm_size=CompiledClassFact.SIZE,
    n_elms=compiled_class_facts_bundle.n_compiled_class_facts,
    key=compiled_class_hash,           // 0 or undeclared → not found → hard proof failure
);
``` [3](#0-2) 

`find_element` is a Cairo assertion-level lookup; if the key is absent the proof is invalid. The sequencer therefore cannot include any transaction that calls the frozen contract in a provable block. Because `replace_class` is a committed, irreversible state change, the contract is permanently unexecutable.

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Once `replace_class` is called with an undeclared class hash:

1. The contract's `class_hash` field in `contract_state_changes` is set to an invalid value.
2. Every future call to the contract causes `find_element` to fail inside `execute_entry_point`, making the OS proof invalid.
3. The sequencer cannot include any such call in a block.
4. The contract can never call `replace_class` again to self-repair (it cannot execute).
5. All ERC-20 tokens, ETH, or other assets held in the contract's storage are permanently inaccessible.

---

### Likelihood Explanation

The `replace_class` syscall is callable by any contract during its own execution. A realistic attack path:

- A malicious deployer creates a contract that accepts user deposits (mimicking a pool or vault) and exposes a public function that calls `replace_class` with an attacker-controlled or hardcoded zero class hash.
- After attracting user deposits, the attacker invokes that function.
- The OS accepts the state transition (no validation), the contract's class hash becomes invalid, and all deposited funds are frozen.

Alternatively, any legitimate contract that passes user-supplied data to `replace_class` without its own validation is vulnerable to the same attack from an unprivileged caller. The attacker needs no special role, no tokens, and no privileged key — only the ability to submit a transaction.

---

### Recommendation

Inside `execute_replace_class`, before writing the new class hash to `contract_state_changes`, assert that the hash is present in `contract_class_changes` (i.e., it has been declared):

```cairo
// Verify the new class hash is a declared class.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);
```

This mirrors the guard already applied implicitly in `execute_entry_point` and makes the invariant explicit and enforced at the point of mutation, consistent with the existing `assert_not_zero(compiled_class_hash)` pattern used in `execute_declare_transaction`. [4](#0-3) 

---

### Proof of Concept

1. **Deploy attacker contract** with a public entry point `freeze_self()` that executes the `replace_class` syscall with `class_hash = 0` (or any undeclared felt).
2. **Attract victim deposits** into the contract (e.g., advertise it as a yield vault).
3. **Call `freeze_self()`** from any account. The OS processes `execute_replace_class`:
   - `request.class_hash = 0`
   - No validation is performed (the TODO check is absent).
   - `dict_update` writes `class_hash=0` into `contract_state_changes` for the contract address.
   - The syscall returns success; the transaction is committed.
4. **Any subsequent call** to the contract reaches `execute_entry_point`:
   - `dict_read(contract_class_changes, key=0)` returns `0` (uninitialized).
   - `find_element(..., key=0)` fails — no compiled class with hash `0` exists.
   - The OS proof is invalid; the sequencer cannot include the transaction.
5. **Result**: The contract is permanently unexecutable. All victim funds are frozen with no recovery path. [5](#0-4) [6](#0-5)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
