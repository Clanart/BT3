### Title
Flat Hardcoded Gas Cost for `send_message_to_l1` Ignores Variable Payload Size — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

`execute_send_message_to_l1` charges a single flat constant `SEND_MESSAGE_TO_L1_GAS_COST = 14470` regardless of the message payload size, while the OS itself performs `O(payload_size)` work via `memcpy`. An unprivileged contract can send arbitrarily large L2→L1 payloads at a fixed gas cost, causing the OS to do unbounded proportional work without proportional compensation. This is a fee/accounting bug that can be exploited to bloat OS output and degrade or halt block processing.

---

### Finding Description

In `execute_send_message_to_l1`, the gas deduction is:

```cairo
let success = reduce_syscall_gas_and_write_response_header(
    total_gas_cost=SEND_MESSAGE_TO_L1_GAS_COST, request_struct_size=SendMessageToL1Request.SIZE
);
```

where `SEND_MESSAGE_TO_L1_GAS_COST = 14470` is a compile-time constant. [1](#0-0) 

After the flat gas deduction succeeds, the OS computes the actual payload size and copies it unconditionally:

```cairo
tempvar payload_size = request.payload_end - payload_start;
...
memcpy(dst=outputs.messages_to_l1 + MessageToL1Header.SIZE, src=payload_start, len=payload_size);
``` [2](#0-1) 

The `payload_size` is attacker-controlled (it is the difference between two pointers from the contract's syscall segment) and can be up to `SIERRA_ARRAY_LEN_BOUND - 1 = 2^32 - 1` felts. [3](#0-2) 

**Contrast with correctly scaled syscalls.** `execute_deploy` and `execute_meta_tx_v0` both apply a per-element factor to their gas cost:

```cairo
let specific_base_gas_cost = DEPLOY_GAS_COST + DEPLOY_CALLDATA_FACTOR_GAS_COST *
    constructor_calldata_size;
``` [4](#0-3) 

`DEPLOY_CALLDATA_FACTOR_GAS_COST = 4850` and `META_TX_V0_CALLDATA_FACTOR_GAS_COST = 4850` exist precisely to account for variable-length data. [5](#0-4) 

`send_message_to_l1` has no analogous per-element factor, making it the only variable-payload syscall that charges a flat fee.

---

### Impact Explanation

**Impact: High — Network not being able to confirm new transactions (total network shutdown).**

The OS must copy the entire payload into the `messages_to_l1` output segment for every `send_message_to_l1` call. If an attacker fills a block with transactions that each call `send_message_to_l1` with a maximally large payload, the OS must perform a proportionally enormous `memcpy` for each transaction while the gas charged per call remains constant at 14470 units. This creates a gas-to-work ratio that can be exploited to make block proving computationally infeasible, stalling the network's ability to confirm new transactions. [6](#0-5) 

---

### Likelihood Explanation

**Likelihood: Medium.**

Any unprivileged contract can call `send_message_to_l1` with an arbitrarily large payload. The attacker only needs to deploy a contract and submit invoke transactions. The only implicit cost is the L2 gas consumed by writing the payload into the contract's memory segment (memory hole costs), but this is far smaller than the OS-side `memcpy` work for large payloads. The attack is straightforward and requires no special privileges. [7](#0-6) 

---

### Recommendation

Apply a per-element gas factor to `send_message_to_l1`, consistent with how `execute_deploy` and `execute_meta_tx_v0` handle variable-length data:

```cairo
// Add to constants.cairo:
const SEND_MESSAGE_TO_L1_PAYLOAD_FACTOR_GAS_COST = 4850;  // or calibrated value

// In execute_send_message_to_l1:
let payload_size = request.payload_end - request.payload_start;
let specific_gas_cost = SEND_MESSAGE_TO_L1_GAS_COST +
    SEND_MESSAGE_TO_L1_PAYLOAD_FACTOR_GAS_COST * payload_size;
let success = reduce_syscall_gas_and_write_response_header(
    total_gas_cost=specific_gas_cost, request_struct_size=SendMessageToL1Request.SIZE
);
``` [8](#0-7) 

---

### Proof of Concept

1. Deploy a Cairo 1 contract with an entry point that calls `send_message_to_l1` with a payload array of length `N` (e.g., `N = 1_000_000`).
2. Submit an invoke transaction targeting that entry point.
3. The OS charges exactly `SEND_MESSAGE_TO_L1_GAS_COST - SYSCALL_BASE_GAS_COST = 14470 - 10000 = 4470` net gas units for the syscall.
4. The OS then executes `memcpy(..., len=1_000_000)`, doing 1,000,000 felt-copy operations for 4470 gas units.
5. Repeat across many transactions in a block to make the OS proving time grow super-linearly relative to the gas consumed, eventually making block proving infeasible. [9](#0-8) [10](#0-9)

func execute_send_message_to_l1{range_check_ptr, syscall_ptr: felt*, outputs: OsCarriedOutputs*}(
    contract_address: felt
) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, SendMessageToL1Request*);
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=SEND_MESSAGE_TO_L1_GAS_COST, request_struct_size=SendMessageToL1Request.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    tempvar payload_start = request.payload_start;
    tempvar payload_size = request.payload_end - payload_start;

    assert [outputs.messages_to_l1] = MessageToL1Header(
        from_address=contract_address, to_address=request.to_address, payload_size=payload_size
    );
    memcpy(
        dst=outputs.messages_to_l1 + MessageToL1Header.SIZE, src=payload_start, len=payload_size
    );
    let (outputs) = os_carried_outputs_new(
        messages_to_l1=outputs.messages_to_l1 + MessageToL1Header.SIZE + payload_size,
        messages_to_l2=outputs.messages_to_l2,
    );

    return ();
}

// Reduces the base amount of gas for the current syscall.
// In case of out-of-gas failure, writes the corresponding ResponseHeader and FailureReason
// objects to syscall_ptr.
// Returns 1 if the gas reduction succeeded and 0 otherwise, along with the remaining gas.
func reduce_syscall_base_gas{range_check_ptr, syscall_ptr: felt*}(
    specific_base_gas_cost: felt, request_struct_size: felt
) -> (success: felt, remaining_gas: felt) {
    alloc_locals;
    let request_header = cast(syscall_ptr, RequestHeader*);
    // Advance syscall pointer to the response header.
    let syscall_ptr = syscall_ptr + RequestHeader.SIZE + request_struct_size;

    // Refund the pre-charged base gas.
    let required_gas = specific_base_gas_cost - SYSCALL_BASE_GAS_COST;
    tempvar initial_gas = request_header.gas;
    local initial_ge_required_gas;
    %{ InitialGeRequiredGas %}
    if (initial_ge_required_gas != FALSE) {
        tempvar remaining_gas = initial_gas - required_gas;
        assert_nn(remaining_gas);
        return (success=1, remaining_gas=remaining_gas);
    }

    // Handle out-of-gas.
    assert_lt(initial_gas, required_gas);
    write_failure_response(remaining_gas=initial_gas, failure_felt=ERROR_OUT_OF_GAS);

    return (success=0, remaining_gas=initial_gas);
}
