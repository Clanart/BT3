### Title
Missing Declared Class Hash Validation in `execute_replace_class` Allows Permanent Freezing of Contract Funds — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler updates `contract_state_changes` with an arbitrary new class hash supplied by the calling contract, but performs **no validation** that the new class hash has been declared (i.e., exists in `contract_class_changes`). The OS itself acknowledges this with a TODO comment. Any contract can therefore replace its class with an undeclared hash, rendering itself permanently uncallable and freezing all funds it holds.

---

### Finding Description

`execute_replace_class` in `syscall_impls.cairo` (lines 878–916) is the OS-level handler for the `replace_class` syscall. Its job is to update the calling contract's class hash in the global state. The function:

1. Reads `request.class_hash` — fully attacker-controlled, no range or existence check.
2. Writes a new `StateEntry` with that hash into `contract_state_changes`.
3. Logs the old class hash in `revert_log`.

What it **does not do** — and the code explicitly acknowledges — is verify that the new class hash is declared:

```cairo
let class_hash = request.class_hash;

// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
``` [1](#0-0) 

The function signature structurally prevents the check: `contract_class_changes` is not an implicit argument, so there is no way to look up whether the supplied hash is declared: [2](#0-1) 

The downstream state commitment pipeline — `compute_contract_state_commitment` and `compute_class_commitment` — hashes whatever is in the dicts without cross-validating that every class hash appearing in `contract_state_changes` also appears in `contract_class_changes`: [3](#0-2) [4](#0-3) 

The post-execution compiled-class-fact validation (`validate_compiled_class_facts_post_execution`) only checks classes that were **executed** during the block; a contract that replaces its class and is never called again escapes that check entirely.

**Analogy to M-28**: M-28's `addBridgeAgentFactory` updated the array but omitted the `isBridgeAgentFactory` mapping update, leaving the registry in an inconsistent state. Here, `execute_replace_class` updates `contract_state_changes` but omits the mandatory cross-check against `contract_class_changes`, leaving the contract state pointing at a non-existent class — an equally inconsistent registry.

---

### Impact Explanation

A contract whose class hash is set to an undeclared value becomes permanently uncallable: no compiled class fact exists for that hash, so every future call to the contract will fail at the OS level. Any ERC-20 balance, vault deposit, or other asset stored in that contract's storage is irrecoverably frozen. This satisfies **Critical — Permanent freezing of funds**.

---

### Likelihood Explanation

The entry path requires only that a contract issue a `replace_class` syscall with an arbitrary hash value. This is reachable by:

- A malicious contract deployer who accepts user deposits and then triggers `replace_class(0xdeadbeef)` to freeze them.
- A contract with a publicly callable upgrade path (e.g., a DAO or multisig) where one participant can supply the new class hash — if the OS does not reject undeclared hashes, a malicious participant can supply a garbage hash.
- Any contract bug that allows an attacker to influence the argument to `replace_class`.

No privileged role, operator key, or network-level attack is required. The syscall is available to every deployed contract.

---

### Recommendation

Add `contract_class_changes` as an implicit argument to `execute_replace_class` and assert that the requested class hash is present (i.e., has a non-zero compiled class hash entry):

```cairo
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    contract_class_changes: DictAccess*,   // add this
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    ...
    let class_hash = request.class_hash;

    // Verify the class is declared.
    let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
    assert_not_zero(compiled_class_hash);
    ...
}
```

This mirrors how `execute_declare_transaction` enforces `prev_value=0` to guarantee a class is declared exactly once before it can be used. [5](#0-4) 

---

### Proof of Concept

1. Deploy contract `Vault` that holds user ETH/ERC-20 balances.
2. `Vault` exposes an `upgrade(new_class_hash)` entry point callable by its owner.
3. Owner (or attacker who controls the owner key) calls `upgrade(0x404040)` — a hash never declared on-chain.
4. The OS processes the `replace_class` syscall: `contract_state_changes[Vault_address].class_hash = 0x404040`. No error is raised.
5. The block is proven and accepted; the global state root now encodes `Vault` with class hash `0x404040`.
6. Any subsequent call to `Vault` (withdraw, transfer, etc.) fails: the OS cannot find a compiled class fact for `0x404040`.
7. All user funds inside `Vault` are permanently frozen with no recovery path.

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
