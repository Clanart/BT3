### Title
Stale `feemarketParams` Snapshot in `NewDynamicFeeChecker` Allows Cosmos Txs to Bypass EIP-1559 Base Fee Enforcement — (File: `ante/evm/fee_checker.go`)

---

### Summary

`newCosmosAnteHandler` captures a one-time snapshot of `feemarketParams` at app startup and passes a pointer to it into `NewDynamicFeeChecker`. Because the EIP-1559 base fee is updated every block but the snapshot is never refreshed, all Cosmos (non-EVM) transactions are fee-checked against a permanently stale base fee. This allows any unprivileged user to submit Cosmos transactions whose fees are below the current block's required base fee, bypassing the fee market entirely for the Cosmos tx path.

---

### Finding Description

In `evmd/ante/handler_options.go`, `newCosmosAnteHandler` is called **once** at app startup (from `setAnteHandler` in `evmd/app.go`). It reads `feemarketParams` from the KV store at that single moment and passes `&feemarketParams` — a pointer to a local variable — into `NewDynamicFeeChecker`:

```go
// evmd/ante/handler_options.go lines 179-187
func newCosmosAnteHandler(ctx sdk.Context, options HandlerOptions, ...) sdk.AnteHandler {
    evmParams := options.EvmKeeper.GetParams(ctx)
    feemarketParams := options.FeeMarketKeeper.GetParams(ctx)   // snapshot at startup
    ...
    txFeeChecker = evm.NewDynamicFeeChecker(ethCfg, &evmParams, &feemarketParams)
``` [1](#0-0) 

The returned `TxFeeChecker` closure captures this pointer and, on every subsequent Cosmos tx, calls:

```go
// ante/evm/fee_checker.go line 56
baseFee := types.GetBaseFee(ctx.BlockHeight(), ethCfg, feemarketParams)
``` [2](#0-1) 

`types.GetBaseFee` reads `feemarketParams.GetBaseFee()` → `feemarketParams.BaseFee`, which is the value frozen at startup:

```go
// x/evm/types/utils.go lines 244-254
func GetBaseFee(height int64, ethCfg *params.ChainConfig, feemarketParams *feemarkettypes.Params) *big.Int {
    ...
    baseFee := feemarketParams.GetBaseFee()   // reads the stale struct field
    ...
}
``` [3](#0-2) 

Meanwhile, the base fee is updated every block via `BeginBlock` → `CalculateBaseFee` → `SetBaseFee`, which writes the new value into the KV store params:

```go
// x/feemarket/keeper/params.go lines 72-78
func (k Keeper) SetBaseFee(ctx sdk.Context, baseFee *big.Int) {
    params := k.GetParams(ctx)
    params.BaseFee = ethermint.SaturatedNewInt(baseFee)
    err := k.SetParams(ctx, params)
    ...
}
``` [4](#0-3) 

The local `feemarketParams` variable captured by the closure is **never updated** after `newCosmosAnteHandler` returns. The KV store is updated, but the in-memory snapshot is not.

By contrast, the EVM ante handler (`newEthAnteHandler`) correctly reads the current base fee on every tx via `EVMBlockConfig`:

```go
// evmd/ante/handler_options.go lines 88-95
blockCfg, err := options.EvmKeeper.EVMBlockConfig(ctx, options.EvmKeeper.ChainID())
...
baseFee := blockCfg.BaseFee   // always current
``` [5](#0-4) 

This is the exact analog of the external report: a getter/checker function uses a different (stale) value than what the actual execution path uses, causing incorrect accounting.

---

### Impact Explanation

When the base fee rises above its startup value (the normal case on a busy network), `NewDynamicFeeChecker` accepts Cosmos txs whose `feeCap` is between the stale (lower) base fee and the actual current base fee. The effective fee deducted from the sender is also computed using the stale base fee:

```go
// ante/evm/fee_checker.go lines 91-98
effectivePrice := sdkmath.NewIntFromBigInt(types.EffectiveGasPrice(baseFeeInt.BigInt(), feeCap.BigInt(), maxPriorityPrice.BigInt()))
effectiveFee := sdk.Coins{{Denom: denom, Amount: effectivePrice.Mul(sdkmath.NewIntFromUint64(gas))}}
``` [6](#0-5) 

The result: Cosmos transactions (governance, staking, IBC, etc.) are committed with fees below the current EIP-1559 minimum, mis-accounting user funds and bypassing the fee market. This matches the allowed High impact: *"ante handler bug that permits invalid transactions to commit or valid user funds/fees to be mis-accounted."*

---

### Likelihood Explanation

The base fee changes every block. After the very first block following startup, the in-memory snapshot diverges from the live value. Any unprivileged user who submits a Cosmos tx after the base fee has risen can exploit this with no special access. The discrepancy compounds over time and across node restarts (each restart resets the snapshot to the then-current value, but divergence resumes immediately).

---

### Recommendation

`NewDynamicFeeChecker` must read the live base fee from the store on every invocation, not from a startup snapshot. Pass the `FeeMarketKeeper` into the checker and call `keeper.GetBaseFee(ctx)` inside the closure, mirroring the pattern already used by `newEthAnteHandler`:

```go
func NewDynamicFeeChecker(ethCfg *params.ChainConfig, evmParams *types.Params, fmKeeper FeeMarketKeeper) authante.TxFeeChecker {
    return func(ctx sdk.Context, tx sdk.Tx) (sdk.Coins, int64, error) {
        ...
        baseFee := fmKeeper.GetBaseFee(ctx)   // live value, not snapshot
        ...
    }
}
```

The same fix applies to `NewMinGasPriceDecorator`, which also captures `&feemarketParams` at construction time.

---

### Proof of Concept

**Setup:**
- App starts at block 1 with base fee = 1,000,000,000 (1 Gwei). `feemarketParams.BaseFee` snapshot = 1 Gwei.
- Network is busy; after 100 blocks, `CalculateBaseFee` has raised the live base fee to 2,000,000,000 (2 Gwei) in the KV store.

**Attack:**
1. Attacker submits a Cosmos governance `MsgVote` with `fee = 1,500,000,000 * gasLimit` (1.5 Gwei × gas).
2. `NewDynamicFeeChecker` reads `feemarketParams.BaseFee` = 1 Gwei (stale snapshot).
3. Check: `feeCap = 1.5 Gwei >= baseFee = 1 Gwei` → **passes**.
4. `effectivePrice = min(1 Gwei + tip, 1.5 Gwei)` computed with stale base fee.
5. Tx is committed; attacker pays 1.5 Gwei × gas instead of the required 2 Gwei × gas.
6. The fee market is bypassed for all Cosmos txs until the next node restart.

func GetBaseFee(height int64, ethCfg *params.ChainConfig, feemarketParams *feemarkettypes.Params) *big.Int {
	if !IsLondon(ethCfg, height) {
		return nil
	}
	baseFee := feemarketParams.GetBaseFee()
	// should not be nil if london hardfork enabled
	if baseFee == nil {
		return new(big.Int)
	}
	return baseFee
}
