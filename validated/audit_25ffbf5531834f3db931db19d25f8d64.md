### Title
Unbounded Recursion in EIP-712 Type Encoding Enables Ante Handler DoS / Chain Halt — (`ethereum/eip712/types.go`)

### Summary

`recursivelyAddTypesToRoot` in `ethereum/eip712/types.go` recurses into every nested JSON object field with no depth cap. An unprivileged user can submit a Cosmos transaction carrying the `ExtensionOptionsWeb3Tx` extension whose message payload contains arbitrarily deep JSON nesting (e.g., via nested `MsgExec` authorizations). Processing such a transaction in the ante handler triggers unbounded recursion, consuming unbounded CPU/stack memory. Because Go's goroutine-stack-overflow is a fatal, unrecoverable error, the `doRecover` guard present in the caller cannot protect the node, and a sufficiently deep payload crashes the validator process, halting the chain.

### Finding Description

**Root cause — `ethereum/eip712/types.go`, `recursivelyAddTypesToRoot` (line 158)**


For every JSON object field the function calls itself with no depth counter or limit. The only guard in the call-chain is `doRecover` in the caller `addMsgTypesToRoot` (line 107):

```go
func addMsgTypesToRoot(...) (err error) {
    defer doRecover(&err)          // catches panic(), NOT stack overflow
    ...
    recursivelyAddTypesToRoot(...)
}
```

`doRecover` calls `recover()`, which is **ineffective against a goroutine stack overflow**. In Go, exceeding the maximum goroutine stack size (default 1 GB) is a fatal runtime error (`runtime: goroutine stack exceeds ...`) that terminates the entire process unconditionally.

**Call chain from unprivileged transaction to vulnerable code:**

1. User broadcasts a Cosmos tx with extension type `/ethermint.types.v1.ExtensionOptionsWeb3Tx`.
2. `newLegacyCosmosAnteHandlerEip712` routes it to `LegacyEip712SigVerificationDecorator.AnteHandle` (`ante/cosmos/eip712.go` line 73).
3. `VerifySignature` calls `eip712.LegacyWrapTxToTypedData` (line 248) **or** the ethsecp256k1 key's `VerifySignature` calls `GetEIP712BytesForMsg` → `GetEIP712TypedDataForMsg` → `decodeAminoSignDoc` / `decodeProtobufSignDoc` → `WrapTxToTypedData`.
4. `WrapTxToTypedData` → `createEIP712MessagePayload` → `createEIP712Types` → `addMsgTypesToRoot` → **`recursivelyAddTypesToRoot`** (unbounded).

The JSON processed at step 4 is derived directly from the Amino encoding of the messages the attacker placed in the transaction body.

**Attacker-controlled nesting depth:**

Cosmos SDK's `authz.MsgExec` wraps an arbitrary list of inner messages, each encoded as an `Any`. When Amino-serialised, each nesting level produces a JSON object inside the previous one:

```json
{
  "type": "cosmos-sdk/MsgExec",
  "value": {
    "grantee": "...",
    "msgs": [{
      "type": "cosmos-sdk/MsgExec",
      "value": { "grantee": "...", "msgs": [{ ... }] }
    }]
  }
}
```

Each level adds ~100–200 bytes. Within a 1 MB transaction the attacker can embed ~5 000–10 000 nesting levels, consuming tens to hundreds of megabytes of goroutine stack. At the default 1 GB limit this is a severe resource exhaustion; with a slightly larger payload or a lower per-frame estimate it becomes a fatal stack overflow.

The only existing bound is `maxDuplicateTypeDefs = 1000` (line 45 of `types.go`), which limits **duplicate type-name collisions**, not recursion depth.

### Impact Explanation

* **Worst case (stack overflow):** The validator process crashes with a fatal runtime error. Because all validators processing the same block execute the same ante handler, a single crafted transaction included in a block causes every honest validator to crash simultaneously → **deterministic chain halt** (Critical).
* **Near-worst case (resource exhaustion without crash):** The ante handler consumes seconds of CPU and hundreds of MB of memory per malicious transaction. An attacker can flood the mempool with such transactions, starving the node of resources, causing missed blocks and liveness failure.

Both outcomes match the Critical allowed impact: *"Valid unprivileged transaction … can halt the chain … or cause deterministic validator consensus failure."*

### Likelihood Explanation

* No special privilege is required; any account can broadcast a Cosmos tx with `ExtensionOptionsWeb3Tx`.
* `MsgExec` (authz) is a standard, always-registered message type; no governance action is needed to enable it.
* The attack payload is straightforward to construct programmatically.
* The vulnerability is triggered deterministically during ante handler execution, not probabilistically.

### Recommendation

Add an explicit depth parameter to `recursivelyAddTypesToRoot` and return an error when it exceeds a safe constant (e.g., 10–15):

```go
const maxEIP712TypeDepth = 15

func recursivelyAddTypesToRoot(
    typeMap apitypes.Types,
    rootType, prefix string,
    payload gjson.Result,
    depth int,           // ← new parameter
) (string, error) {
    if depth > maxEIP712TypeDepth {
        return "", errorsmod.Wrap(errortypes.ErrInvalidRequest,
            "EIP-712 type nesting exceeds maximum depth")
    }
    ...
    fieldTypeDef, err := recursivelyAddTypesToRoot(
        typeMap, rootType, fieldPrefix, field, depth+1)
    ...
}
```

Apply the same guard to `legacyTraverseFields` in `eip712_legacy.go` (line 347).

### Proof of Concept

1. Construct a `MsgExec` wrapping a `MsgExec` wrapping … (N ≈ 5 000 levels) with a trivial innermost message (e.g., `MsgSend`).
2. Amino-encode the outermost `MsgExec` as the sole message in a `StdSignDoc`.
3. Wrap it in a Cosmos tx with `ExtensionOptionsWeb3Tx` extension and broadcast to any validator's RPC endpoint.
4. The ante handler calls `recursivelyAddTypesToRoot` N levels deep; at sufficient N the goroutine stack is exhausted and the validator process terminates with a fatal runtime error.

```go
func recursivelyAddTypesToRoot(
	typeMap apitypes.Types,
	rootType string,
	prefix string,
	payload gjson.Result,
) (string, error) {
	typesToAdd := []apitypes.Type{}

	// Must sort the JSON keys for deterministic type generation.
	sortedFieldNames, err := sortedJSONKeys(payload)
	if err != nil {
		return "", errorsmod.Wrap(err, "unable to sort object keys")
	}

	typeDef := typeDefForPrefix(prefix, rootType)

	for _, fieldName := range sortedFieldNames {
		field := payload.Get(fieldName)
		if !field.Exists() {
			continue
		}

		// Handle array type by unwrapping the first element.
		// Note that arrays with multiple types are not supported
		// using EIP-712, so we can ignore that case.
		isCollection := false
		if field.IsArray() {
			fieldAsArray := field.Array()

			if len(fieldAsArray) == 0 {
				// Arbitrarily add string[] type to handle empty arrays,
				// since we cannot access the underlying object.
				emptyArrayType := "string[]"
				typesToAdd = appendedTypesList(typesToAdd, fieldName, emptyArrayType)

				continue
			}

			field = fieldAsArray[0]
			isCollection = true
		}

		ethType := getEthTypeForJSON(field)

		// Handle JSON primitive types by adding the corresponding
		// EIP-712 type to the types schema.
		if ethType != "" {
			if isCollection {
				ethType += "[]"
			}
			typesToAdd = appendedTypesList(typesToAdd, fieldName, ethType)

			continue
		}

		// Handle object types recursively. Note that nested array types are not supported
		// in EIP-712, so we can exclude that case.
		if field.IsObject() {
			fieldPrefix := prefixForSubField(prefix, fieldName)

			fieldTypeDef, err := recursivelyAddTypesToRoot(typeMap, rootType, fieldPrefix, field)
			if err != nil {
				return "", err
			}

			fieldTypeDef = sanitizeTypedef(fieldTypeDef)
			if isCollection {
				fieldTypeDef += "[]"
			}

			typesToAdd = appendedTypesList(typesToAdd, fieldName, fieldTypeDef)

			continue
		}
	}

	return addTypesToRoot(typeMap, typeDef, typesToAdd)
}
```

```go
// doRecover attempts to recover in the event of a panic to
// prevent DOS and gracefully handle an error instead.
func doRecover(err *error) {
	if r := recover(); r != nil {
		if e, ok := r.(error); ok {
			e = errorsmod.Wrap(e, "panicked with error")
			*err = e
			return
		}

		*err = fmt.Errorf("%v", r)
	}
}
```

```go
func legacyTraverseFields(
	cdc codectypes.AnyUnpacker,
	typeMap apitypes.Types,
	rootType string,
	prefix string,
	t reflect.Type,
	v reflect.Value,
) error {
	n := t.NumField()

	if prefix == typeDefPrefix {
		if len(typeMap[rootType]) == n {
			return nil
		}
	} else {
		typeDef := sanitizeTypedef(prefix)
		if len(typeMap[typeDef]) == n {
			return nil
		}
	}

	for i := 0; i < n; i++ {
		var (
			field reflect.Value
			err   error
		)

		if v.IsValid() {
			field = v.Field(i)
		}

		fieldType := t.Field(i).Type
		fieldName := jsonNameFromTag(t.Field(i).Tag)

		if fieldType == cosmosAnyType {
			// Unpack field, value as Any
			if fieldType, field, err = unpackAny(cdc, field); err != nil {
				return err
			}
		}

		// If field is an empty value, do not include in types, since it will not be present in the object
		if field.IsZero() {
			continue
		}

		for {
			if fieldType.Kind() == reflect.Ptr {
				fieldType = fieldType.Elem()

				if field.IsValid() {
					field = field.Elem()
				}

				continue
			}

			if fieldType.Kind() == reflect.Interface {
				fieldType = reflect.TypeOf(field.Interface())
				continue
			}

			if field.Kind() == reflect.Ptr {
				field = field.Elem()
				continue
			}

			break
		}

		var isCollection bool
		if fieldType.Kind() == reflect.Array || fieldType.Kind() == reflect.Slice {
			if field.Len() == 0 {
				// skip empty collections from type mapping
				continue
			}

			fieldType = fieldType.Elem()
			field = field.Index(0)
			isCollection = true

			if fieldType == cosmosAnyType {
				if fieldType, field, err = unpackAny(cdc, field); err != nil {
					return err
				}
			}
		}

		for {
			if fieldType.Kind() == reflect.Ptr {
				fieldType = fieldType.Elem()

				if field.IsValid() {
					field = field.Elem()
				}

				continue
			}

			if fieldType.Kind() == reflect.Interface {
				fieldType = reflect.TypeOf(field.Interface())
				continue
			}

			if field.Kind() == reflect.Ptr {
				field = field.Elem()
				continue
			}

			break
		}

		fieldPrefix := fmt.Sprintf("%s.%s", prefix, fieldName)

		ethTyp := typToEth(fieldType)

		if len(ethTyp) > 0 {
			// Support array of uint64
			if isCollection && fieldType.Kind() != reflect.Slice && fieldType.Kind() != reflect.Array {
				ethTyp += "[]"
			}

			if prefix == typeDefPrefix {
				typeMap[rootType] = append(typeMap[rootType], apitypes.Type{
					Name: fieldName,
					Type: ethTyp,
				})
			} else {
				typeDef := sanitizeTypedef(prefix)
				typeMap[typeDef] = append(typeMap[typeDef], apitypes.Type{
					Name: fieldName,
					Type: ethTyp,
				})
			}

			continue
		}

		if fieldType.Kind() == reflect.Struct {
			var fieldTypedef string

			if isCollection {
				fieldTypedef = sanitizeTypedef(fieldPrefix) + "[]"
			} else {
				fieldTypedef = sanitizeTypedef(fieldPrefix)
			}

			if prefix == typeDefPrefix {
				typeMap[rootType] = append(typeMap[rootType], apitypes.Type{
					Name: fieldName,
					Type: fieldTypedef,
				})
			} else {
				typeDef := sanitizeTypedef(prefix)
				typeMap[typeDef] = append(typeMap[typeDef], apitypes.Type{
					Name: fieldName,
					Type: fieldTypedef,
				})
			}

			if err := legacyTraverseFields(cdc, typeMap, rootType, fieldPrefix, fieldType, field); err != nil {
				return err
			}

			continue
		}
	}

	return nil
}
```
