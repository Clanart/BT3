### Title
DataLayer `inner_build_blob_from_node_list` Accepts Caller-Supplied Internal Node Hashes Without Recomputation, Enabling Forged MerkleBlob Root - (`crates/chia-datalayer/src/merkle/blob.rs`)

---

### Summary

`inner_build_blob_from_node_list` blindly stores the map key (`node_hash`) as the stored hash of every `Internal` node it writes, without ever calling `internal_hash(left_child_hash, right_child_hash)`. Because the nodes are written with `dirty: false`, neither `calculate_lazy_hashes` nor `check_integrity` will recompute or detect the mismatch in a production release build. A caller who controls the `NodeHashToDeltaReaderNode` map — reachable via the Python-exposed `DeltaReader` — can produce a `MerkleBlob` whose root hash is an arbitrary value H while the actual tree contents hash to a completely different value H′.

---

### Finding Description

In `inner_build_blob_from_node_list`, the `Internal` branch constructs the block as:

```rust
node: Node::Internal(InternalNode {
    hash: node_hash,   // ← map key, never verified
    parent: Parent(None),
    left: left_index,
    right: right_index,
}),
``` [1](#0-0) 

`node_hash` is the key used to look up the entry in the `NodeHashToDeltaReaderNode` map — it is **not** computed as `internal_hash(left_child_hash, right_child_hash)`. The block is then written with `dirty: false`: [2](#0-1) 

`calculate_lazy_hashes` only recomputes hashes for nodes flagged `dirty`: [3](#0-2) 

So a non-dirty node with a forged hash is never touched. `check_just_integrity` (the core of `check_integrity`) verifies only structural parent-child relationships and cache counts — it does **not** verify that any internal node's stored hash equals `internal_hash(left, right)`: [4](#0-3) 

`check_integrity` calls `calculate_lazy_hashes` on a clone before re-running `check_just_integrity`, but since all nodes are `dirty: false`, no hash is recomputed and the mismatch is invisible: [5](#0-4) 

The `Drop`-based integrity check is compiled only under `#[cfg(any(test, debug_assertions))]` — absent in production release builds: [6](#0-5) 

`create_merkle_blob_and_filter_unused_nodes` — the production entry point — returns the blob directly without calling `check_integrity`: [7](#0-6) 

The `DeltaReader` Python binding accepts arbitrary `internal_nodes: Mapping[bytes32, tuple[bytes32, bytes32]]` from the caller with no hash-consistency validation: [8](#0-7) [9](#0-8) 

---

### Impact Explanation

An attacker who can supply the `internal_nodes` dict to `DeltaReader` (e.g., a malicious DataLayer sync peer whose data is forwarded by the chia-blockchain Python layer) can craft a map entry `{H: (left_fake, right_fake)}` where `internal_hash(left_fake, right_fake) ≠ H`. The resulting `MerkleBlob` will report root hash H while its actual tree contents hash to a different value. If H equals the on-chain DataLayer root commitment, the victim node accepts the blob as authentic. The victim can then be made to:

- Accept forged **inclusion proofs** for keys not present in the real tree (via `get_proof_of_inclusion`, which reads stored hashes verbatim from the blob).
- Accept forged **exclusion proofs** or incorrect key-value mappings.

This matches the High impact category: *DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state.*

---

### Likelihood Explanation

The `DeltaReader` is explicitly designed for peer-to-peer DataLayer delta sync. Its Python binding accepts `internal_nodes` as a plain dict, and the chia-blockchain sync protocol populates this dict from data received from remote peers. No hash-consistency check exists anywhere between receipt of peer data and use of the resulting `MerkleBlob`. The attack requires only the ability to act as a DataLayer sync peer — no privileged access, no key material, no chain reorg.

---

### Recommendation

In `inner_build_blob_from_node_list`, replace the verbatim use of `node_hash` for internal nodes with a recomputed hash:

```rust
// After recursing into left and right children:
let left_hash  = self.get_hash(left_index)?;
let right_hash = self.get_hash(right_index)?;
let computed   = internal_hash(&left_hash, &right_hash);
if computed != node_hash {
    return Err(Error::InternalHashMismatch(node_hash, computed));
}
```

Alternatively, always write internal nodes with `dirty: true` and call `calculate_lazy_hashes` before returning from `build_blob_from_node_list`, so the hash is always derived from children rather than trusted from the caller.

Additionally, extend `check_just_integrity` to verify `stored_hash == internal_hash(left_child_hash, right_child_hash)` for every internal node, so the invariant is enforced defensively at all call sites.

### Proof of Concept

```rust
use chia_datalayer::{
    Hash, KeyId, MerkleBlob, NodeHashToDeltaReaderNode, ValueId,
    merkle::{blob::internal_hash, deltas::DeltaReaderNode},
};
use std::collections::HashSet;

fn forged_root_poc() {
    // Real leaf hashes
    let left_leaf  = Hash([1u8; 32]);
    let right_leaf = Hash([2u8; 32]);
    // Correct root would be internal_hash(&left_leaf, &right_leaf)
    let real_root  = internal_hash(&left_leaf, &right_leaf);

    // Attacker claims the root is some arbitrary H ≠ real_root
    let forged_root = Hash([0xdeu8; 32]);
    assert_ne!(forged_root, real_root);

    let mut nodes = NodeHashToDeltaReaderNode::new();
    // Forged internal node: key = forged_root, children = real leaves
    nodes.insert(forged_root, DeltaReaderNode::Internal {
        left:  left_leaf,
        right: right_leaf,
    });
    nodes.insert(left_leaf,  DeltaReaderNode::Leaf { key: KeyId(1), value: ValueId(1) });
    nodes.insert(right_leaf, DeltaReaderNode::Leaf { key: KeyId(2), value: ValueId(2) });

    let mut all_used = HashSet::new();
    let blob = MerkleBlob::build_blob_from_node_list(
        &nodes, forged_root, &HashSet::new(), &mut all_used,
    ).unwrap();

    // Root hash stored in blob is forged_root, not real_root
    let stored_root = blob.get_hash(chia_datalayer::TreeIndex(0)).unwrap();
    assert_eq!(stored_root, forged_root);   // passes — forged hash accepted
    assert_ne!(stored_root, real_root);     // passes — invariant violated

    // check_integrity does NOT catch this
    blob.check_integrity().unwrap();        // passes — no hash recomputation for dirty:false nodes
}

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }

impl DeltaReader {
    pub fn new(internal_nodes: InternalNodesMap, leaf_nodes: LeafNodesMap) -> Result<Self, Error> {
        let mut nodes = NodeHashToDeltaReaderNode::new();

        for (hash, (left, right)) in internal_nodes {
            nodes.insert(hash, DeltaReaderNode::Internal { left, right });
        }
        for (hash, (key, value)) in leaf_nodes {
            nodes.insert(hash, DeltaReaderNode::Leaf { key, value });
        }

        Ok(Self { nodes })
