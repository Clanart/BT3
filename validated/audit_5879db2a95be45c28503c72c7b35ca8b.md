### Title
Unvalidated Class Hash in `execute_replace_class` Allows Contracts to Bind Arbitrary Class Hashes, Permanently Freezing Funds — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary
The `execute_replace_class` syscall implementation in the StarkNet OS does not verify that the replacement class hash corresponds to a previously declared class. Any deployed contract can call `replace_class` with an arbitrary, undeclared hash. The OS accepts the state transition and generates a valid proof. The contract is then permanently bound to a non-existent class, making it uncallable and freezing any funds it holds.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function reads the new class hash directly from the syscall request and writes it into `contract_state_changes` with no check that the hash exists in the set of declared classes (`contract_class_changes`). The code itself acknowledges this gap with an explicit TODO:

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
``` [1](#0-0) 

The `class_hash` value is taken verbatim from `request.class_hash` with no cross-reference against `contract_class_changes` (the dict tracking declared classes) or the compiled class facts bundle validated at the end of the block by `validate_compiled_class_facts_post_execution`. [2](#0-1) 

This is the direct analog of the "Battle fishing attack": just as `createBattle` creates a battle without initialization and allows a fake manager to be bound, `execute_replace_class` replaces a contract's class without validation and allows a contract to be bound to an arbitrary (non-existent) class hash. The OS then commits this invalid binding into the global state root via `state_update` → `compute_contract_state_commitment`, producing a valid proof over a corrupted state. [3](#0-2) 

The end-of-block validation (`validate_compiled_class_facts_post_execution`) only checks class facts that were **executed** during the block. A class hash that is merely stored in the state but never executed in the same block is invisible to this check. [4](#0-3) 

---

### Impact Explanation

**Permanent freezing of funds (Critical).**

Once a contract's class hash is set to an undeclared value and committed to the state root:

- Every future call to that contract requires the OS to locate and execute the class. Because the class does not exist in the compiled class facts, execution cannot proceed.
- The contract becomes permanently uncallable. Any ERC-20 tokens, ETH, or other assets held in its storage are irrecoverably locked.
- The state root itself is valid (the OS correctly recorded the transition), so the proof passes L1 verification. The corruption is canonical and irreversible without a protocol-level upgrade.

---

### Likelihood Explanation

The entry path requires only a deployed contract — the lowest possible privilege level on StarkNet. The attacker:

1. Deploys any contract (permissionless).
2. Attracts user deposits (social engineering, or via a legitimate-looking DeFi interface).
3. Submits a transaction invoking the `replace_class` syscall with an undeclared hash.
4. The sequencer includes the transaction; the OS processes it without error.

No privileged key, operator access, or external dependency is required. The attack is fully on-chain and deterministic.

---

### Recommendation

Inside `execute_replace_class`, after reading `request.class_hash`, assert that the hash is present in `contract_class_changes` (i.e., it was declared in the current block) **or** that it exists in the pre-block state's class trie. The simplest Cairo-level guard is a `dict_read` on `contract_class_changes` asserting the returned compiled class hash is non-zero, mirroring the check already enforced in `execute_declare_transaction`:

```cairo
// Proposed guard (pseudocode):
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);
``` [5](#0-4) 

---

### Proof of Concept

1. **Attacker deploys** contract `C` with a valid, declared class hash `V`. `C` exposes a `deposit` entry point.
2. **Users deposit** tokens into `C`; `C`'s storage now holds user balances.
3. **Attacker submits** an invoke transaction from `C` that calls the `replace_class` syscall with `class_hash = 0xDEAD` (never declared).
4. **OS executes** `execute_replace_class`: reads `request.class_hash = 0xDEAD`, skips the missing existence check, writes `StateEntry(class_hash=0xDEAD, ...)` into `contract_state_changes`.
5. **`state_update`** squashes and commits the change; the new global state root encodes `C → 0xDEAD`. A valid STARK proof is produced and verified on L1.
6. **Next block**: any user attempts to call `C` (e.g., `withdraw`). The OS looks up class `0xDEAD` in the compiled class facts — it does not exist. The block cannot be proven; the sequencer must drop the call. The funds are permanently frozen with no recovery path.

// Replaces the class.
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
