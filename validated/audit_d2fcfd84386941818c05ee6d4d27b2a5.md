### Title
Missing Internal-Hash Verification in `build_blob_from_node_list` Allows Forged DataLayer Tree Root — (`crates/chia-datalayer/src/merkle/blob.rs`)

---

### Summary

`inner_build_blob_from_node_list` blindly stores the map key as the `InternalNode.hash` field without ever asserting that it equals `internal_hash(left_child_hash, right_child_hash)`. Because `DeltaReader::new` / `py_add_internal_nodes` accept arbitrary `InternalNodesMap` input with no hash verification, and because `check_integrity()` never validates internal-node hash correctness, an attacker who controls the delta input can produce a `MerkleBlob` whose stored root hash is completely fabricated.

---

### Finding Description

**`DeltaReader::new` — no hash verification on ingestion**

`DeltaReader::new` iterates over the caller-supplied `InternalNodesMap` and inserts each `(hash, (left, right))` tuple verbatim into `self.nodes` without checking that `hash == internal_hash(left, right)`. [1](#0-0) 

The Python binding exposes this path directly to untrusted callers: [2](#0-1) 

**`inner_build_blob_from_node_list` — map key used as stored hash, no recomputation**

For an `Internal` node, the function allocates a new blob slot, recursively builds the left and right subtrees, then writes an `InternalNode` whose `hash` field is set to `node_hash` — the map key supplied by the caller — with `dirty: false`. There is no call to `internal_hash(left_child_hash, right_child_hash)` and no assertion comparing the two values. [3](#0-2) 

Because the node is written with `dirty: false`, `calculate_lazy_hashes()` will never revisit it to recompute the correct hash. [4](#0-3) 

**`check_integrity()` — does not verify internal-node hash correctness**

`check_just_integrity` validates parent-child pointer consistency, key-to-index cache lengths, and total node counts. It never computes `internal_hash(left, right)` and never compares it to the stored `InternalNode.hash`. A blob with a completely fabricated root hash passes this check without error. [5](#0-4) 

`check_integrity` calls `calculate_lazy_hashes` on a clone before the second `check_just_integrity` call, but because all nodes were written with `dirty: false`, no hashes are recomputed there either. [6](#0-5) 

---

### Impact Explanation

An attacker who controls the `InternalNodesMap` fed to `DeltaReader` can supply:

```
fake_root_hash → (left_hash, right_hash)
```

where `fake_root_hash ≠ internal_hash(left_hash, right_hash)`.

`create_merkle_blob_and_filter_unused_nodes(fake_root_hash, {})` will produce a `MerkleBlob` whose index-0 block stores `fake_root_hash` as the root. [7](#0-6) 

The resulting blob:
- Reports `fake_root_hash` as its root via `get_hash_at_index(TreeIndex(0))`
- Passes `check_integrity()` without error
- Generates inclusion/exclusion proofs whose intermediate `combined_hash` values are the attacker-chosen fake hashes, not the real Merkle hashes

Any downstream system that trusts the blob's stored root hash (rather than recomputing it from the leaf set) will accept a tree whose committed root is entirely fabricated. This matches the **High** impact category: *DataLayer Merkle proof/blob/delta logic corrupts tree roots or lets untrusted input prove invalid state.*

---

### Likelihood Explanation

The `DeltaReader` Python binding is the primary sync interface for DataLayer delta data received from peers. A malicious peer can supply a crafted `InternalNodesMap` through the normal P2P sync protocol. No special privilege is required — any peer that participates in DataLayer sync can trigger this path. The Rust library provides zero protection; the entire burden of hash verification falls on the Python caller, which has no Rust-level enforcement to rely on.

---

### Recommendation

In `inner_build_blob_from_node_list`, after recursively resolving the left and right child indexes, retrieve the actual stored hashes of those children and assert:

```rust
let computed = internal_hash(&actual_left_hash, &actual_right_hash);
if computed != node_hash {
    return Err(Error::InternalHashMismatch(node_hash, computed));
}
```

This check should be placed immediately before writing the `InternalNode` block, at the point where both child hashes are already available. [8](#0-7) 

Additionally, `check_just_integrity` should be extended to verify the hash invariant for every internal node it visits. [9](#0-8) 

---

### Proof of Concept

```rust
use chia_datalayer::{
    Hash, InternalNodesMap, LeafNodesMap, KeyId, ValueId,
    merkle::deltas::DeltaReader,
};
use std::collections::{HashMap, HashSet};

// Arbitrary hashes — fake_root is NOT internal_hash(left, right)
let fake_root = Hash::from([0xAA; 32]);
let left_hash  = Hash::from([0x01; 32]);
let right_hash = Hash::from([0x02; 32]);

let mut internal_nodes: InternalNodesMap = HashMap::new();
internal_nodes.insert(fake_root, (left_hash, right_hash));

let mut leaf_nodes: LeafNodesMap = HashMap::new();
leaf_nodes.insert(left_hash,  (KeyId(1), ValueId(10)));
leaf_nodes.insert(right_hash, (KeyId(2), ValueId(20)));

let mut reader = DeltaReader::new(internal_nodes, leaf_nodes).unwrap();
let blob = reader
    .create_merkle_blob_and_filter_unused_nodes(fake_root, &HashSet::new())
    .unwrap();

// Root hash is the attacker-chosen fake value, not internal_hash(left, right)
let root = blob.get_hash_at_index(chia_datalayer::TreeIndex(0)).unwrap().unwrap();
assert_eq!(root, fake_root);   // passes — forged root accepted

// check_integrity() does not catch the hash mismatch
blob.check_integrity().unwrap();  // passes — no hash verification performed
```

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
