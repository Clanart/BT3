### Title
Forged Internal Node Hash Accepted Without Verification in `DeltaReader::new` / `build_blob_from_node_list` — (`crates/chia-datalayer/src/merkle/blob.rs`, `crates/chia-datalayer/src/merkle/deltas.rs`)

### Summary
`DeltaReader::new()` accepts an `InternalNodesMap` where the map key (the claimed hash of an internal node) is never verified to equal `internal_hash(left_child_hash, right_child_hash)`. `inner_build_blob_from_node_list()` then writes the forged key directly into the `InternalNode.hash` field of the resulting `MerkleBlob` without recomputation. The resulting blob's root hash does not reflect the actual tree contents.

### Finding Description
`InternalNodesMap` is defined as `HashMap<Hash, (Hash, Hash)>` — a map from a claimed internal-node hash to its (left, right) child hashes. [6](#0-5) 

In `DeltaReader::new()`, every entry is inserted verbatim: [7](#0-6) 

In `inner_build_blob_from_node_list()`, when processing an `Internal` node, the code recurses into left and right children, then writes the block with `hash: node_hash` — the map key — without ever calling `internal_hash(&left_hash, &right_hash)`: [8](#0-7) 

Compare this to the correct path in `insert_second` / `insert_third_or_later` / `batch_insert`, which all call `internal_hash(...)` to compute the hash from children: [9](#0-8) 

### Impact Explanation
An attacker who can supply delta data (via the Python binding `DeltaReader(internal_nodes, leaf_nodes)` or any caller of `DeltaReader::new`) can produce a `MerkleBlob` whose root hash is an arbitrary attacker-chosen value, completely disconnected from the actual leaf contents. Any downstream DataLayer proof-of-inclusion or proof-of-exclusion built from this blob will be cryptographically invalid — the root hash does not commit to the actual key/value data. This matches the allowed High impact: "DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."

### Likelihood Explanation
The Python binding is a direct, unprivileged entry point. No special privileges are required — any caller constructing a `DeltaReader` with a crafted `InternalNodesMap` triggers the bug. The existing test suite itself uses `HASH_ZERO` as a forged internal-node key (not the real `internal_hash(HASH_ONE, HASH_TWO)`), confirming the path is reachable and the invariant is not enforced.

### Recommendation
In `DeltaReader::new()` (or at the start of `inner_build_blob_from_node_list()` for the `Internal` branch), recompute the expected hash and reject mismatches:

```rust
// In DeltaReader::new, for each internal node:
let expected = internal_hash(&left, &right);
if expected != hash {
    return Err(Error::InternalHashMismatch(hash, expected));
}
```

Alternatively, discard the caller-supplied hash entirely in `inner_build_blob_from_node_list()` and recompute it from the children's hashes after recursing, analogous to how `batch_insert` does it.

### Proof of Concept

```rust
use std::collections::{HashMap, HashSet};
use chia_datalayer::{
    DeltaReader, InternalNodesMap, LeafNodesMap, MerkleBlob,
    Hash, KeyId, ValueId, TreeIndex,
    merkle::blob::internal_hash,
};

let forged_root = Hash([0u8; 32]); // all-zeros, NOT internal_hash(leaf1, leaf2)
let leaf1_hash = Hash([1u8; 32]);
let leaf2_hash = Hash([2u8; 32]);

let mut internal_nodes: InternalNodesMap = HashMap::new();
internal_nodes.insert(forged_root, (leaf1_hash, leaf2_hash)); // key ≠ internal_hash(l,r)

let mut leaf_nodes: LeafNodesMap = HashMap::new();
leaf_nodes.insert(leaf1_hash, (KeyId(1), ValueId(1)));
leaf_nodes.insert(leaf2_hash, (KeyId(2), ValueId(2)));

let mut reader = DeltaReader::new(internal_nodes, leaf_nodes).unwrap();
let blob = reader
    .create_merkle_blob_and_filter_unused_nodes(forged_root, &HashSet::new())
    .unwrap();

let stored_root = blob.get_hash_at_index(TreeIndex(0)).unwrap().unwrap();
assert_eq!(stored_root, forged_root); // passes — forged hash accepted
let correct_root = internal_hash(&leaf1_hash, &leaf2_hash);
assert_ne!(stored_root, correct_root); // passes — invariant violated
```
