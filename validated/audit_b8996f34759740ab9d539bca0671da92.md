### Title
Missing Backward-Compatibility Branch in `calculate_global_state_root` Produces Incorrect Global State Root — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/state/commitment.cairo`)

---

### Summary

`calculate_global_state_root` in `commitment.cairo` documents three distinct cases for computing the global state root, but only implements two of them. The missing case — when the contract class tree is empty (`contract_class_root == 0`) but the contract state tree is non-empty — causes the OS to emit an incorrect `initial_root` in the proof output. Because L1 stores the old-format root (`contract_state_root` directly), the proof fails L1 verification, permanently halting the ability to confirm new transactions.

---

### Finding Description

The function `calculate_global_state_root` in `commitment.cairo` carries this specification comment:

```
// If both the contract class and contract state trees are empty, the global root is set to 0.
// If the contract class tree is empty, the global state root is equal to the
// contract state root (for backward compatibility);
// Otherwise, the global root is obtained by:
//     global_root = H(GLOBAL_STATE_VERSION, contract_state_root, contract_class_root).
```

The actual implementation is:

```cairo
func calculate_global_state_root{poseidon_ptr: PoseidonBuiltin*, range_check_ptr}(
    contract_state_root: felt, contract_class_root: felt
) -> (global_root: felt) {
    if (contract_state_root == 0 and contract_class_root == 0) {
        return (global_root=0);
    }
    tempvar elements: felt* = new (GLOBAL_STATE_VERSION, contract_state_root, contract_class_root);
    let (global_root) = poseidon_hash_many(n=3, elements=elements);
    return (global_root=global_root);
}
``` [1](#0-0) 

The code collapses the three documented cases into two. When `contract_class_root == 0` and `contract_state_root != 0` (the backward-compatibility case), the code falls through to the `poseidon_hash_many` branch and returns:

```
H(GLOBAL_STATE_VERSION, contract_state_root, 0)
```

instead of the documented:

```
contract_state_root
```

These are cryptographically distinct values. The old StarkNet state format (pre-class-tree) stored the global root as the bare `contract_state_root`. L1 holds this value. When the OS processes the first block that transitions from the old format (class root = 0) to the new format, it computes the wrong `initial_root` in the `CommitmentUpdate` struct. [2](#0-1) 

This `CommitmentUpdate` is propagated through `state_update` → `get_block_os_output_header` → `OsOutputHeader.state_update_output` → `serialize_output_header`, where `initial_root` and `final_root` are written directly to the proof output segment. [3](#0-2) 

L1 rejects any proof whose `initial_root` does not match the on-chain stored state root. Because the OS emits `H(GLOBAL_STATE_VERSION, contract_state_root, 0)` while L1 holds `contract_state_root`, every proof generated during this transition is invalid.

---

### Impact Explanation

**High — Network not being able to confirm new transactions (total network shutdown).**

The L1 StarkNet core contract verifies that the `initial_root` in the proof matches the current on-chain state root before accepting any state update. A mismatch causes the L1 call to revert. No block can be finalized on L1, and the network cannot advance its state. This is a total network shutdown for any deployment whose class tree root is 0 at the time the OS runs.

---

### Likelihood Explanation

The trigger condition — `contract_class_root == 0` with `contract_state_root != 0` — is precisely the state of any StarkNet deployment that existed before the contract class tree was introduced, or any fresh deployment that has contract state but no declared classes yet. The comment's explicit mention of "backward compatibility" confirms this was a known, expected scenario. Any sequencer running this OS against such a state will produce an unprovable block, halting the chain. No privileged access is required; the condition arises from the normal state of the network.

---

### Recommendation

Restore the missing backward-compatibility branch in `calculate_global_state_root`:

```cairo
func calculate_global_state_root{poseidon_ptr: PoseidonBuiltin*, range_check_ptr}(
    contract_state_root: felt, contract_class_root: felt
) -> (global_root: felt) {
    if (contract_state_root == 0 and contract_class_root == 0) {
        return (global_root=0);
    }
    // Backward compatibility: old state format has no class tree.
    if (contract_class_root == 0) {
        return (global_root=contract_state_root);
    }
    tempvar elements: felt* = new (GLOBAL_STATE_VERSION, contract_state_root, contract_class_root);
    let (global_root) = poseidon_hash_many(n=3, elements=elements);
    return (global_root=global_root);
}
```

Additionally, add an integrity assertion after computing the global root to verify it matches the value expected by the L1 verifier, analogous to the `CheckBlockHashConsistency` hint pattern used in `get_block_hashes`. [4](#0-3) 

---

### Proof of Concept

1. Deploy a StarkNet instance (or use any existing deployment) whose on-chain global state root equals `contract_state_root` (old format, no class tree — `contract_class_root = 0`).
2. Submit any transaction that causes the OS to execute `state_update`. The OS calls `compute_class_commitment`, which guesses `initial_root = 0` for the class tree (correct, since no classes exist yet).
3. `calculate_global_state_root` is called with `contract_state_root = S` (non-zero) and `contract_class_root = 0`.
4. The code skips the missing backward-compatibility branch and computes `global_root = poseidon_hash_many([GLOBAL_STATE_VERSION, S, 0])` — call this value `X`.
5. The OS outputs `initial_root = X` in `OsOutputHeader.state_update_output`.
6. L1 holds `S` as the current state root. It checks `initial_root == S`, finds `X != S`, and reverts.
7. No proof can be accepted by L1. The network cannot finalize any further blocks. [5](#0-4) [6](#0-5) [7](#0-6)

// Calculates and returns the global state root given the contract state root and
// the contract class state root.
// If both the contract class and contract state trees are empty, the global root is set to 0.

func calculate_global_state_root{poseidon_ptr: PoseidonBuiltin*, range_check_ptr}(
    contract_state_root: felt, contract_class_root: felt

    if (contract_state_root == 0 and contract_class_root == 0) {
        // The state is empty.
        return (global_root=0);

    tempvar elements: felt* = new (GLOBAL_STATE_VERSION, contract_state_root, contract_class_root);
    let (global_root) = poseidon_hash_many(n=3, elements=elements);
    return (global_root=global_root);
}
