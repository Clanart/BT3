### Title
Missing Declared-Class Invariant in `execute_replace_class` Allows Permanent Contract Freezing — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall implementation in the StarkNet OS allows any contract to replace its class hash with an **arbitrary, undeclared felt value** without verifying that the new class hash corresponds to a previously declared class. This bypasses the invariant enforced by `execute_declare_transaction` and `execute_entry_point`, which require that a contract's class hash must map to a known compiled class. An unprivileged contract deployer can exploit this to permanently freeze all funds held by any contract they control, matching the "invariant bypass via alternative setter" vulnerability class from the external report.

---

### Finding Description

The StarkNet OS enforces a critical invariant: every contract's `class_hash` field must correspond to a class that has been declared and whose compiled form is present in the `compiled_class_facts_bundle`. This invariant is enforced in two places:

**1. Declaration path** (`execute_declare_transaction`, `transaction_impls.cairo` lines 814–819): A class is only written to `contract_class_changes` after the Sierra class hash is verified and the compiled class hash is confirmed non-zero. [1](#0-0) 

**2. Execution path** (`execute_entry_point`, `execute_entry_point.cairo` lines 154–166): When executing a contract, the OS looks up the class hash in `contract_class_changes` to get the compiled class hash, then calls `find_element` to locate the compiled class in the bundle. If the class hash is not declared, `find_element` fails at the prover level. [2](#0-1) 

**The bypass** — `execute_replace_class` (`syscall_impls.cairo` lines 878–916) sets a contract's class hash to any caller-supplied felt without any check that the new hash is declared. The code itself acknowledges this with a TODO comment: [3](#0-2) 

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
```

The `class_hash` value comes directly from `request.class_hash` — a caller-controlled field — with no validation against `contract_class_changes` or `compiled_class_facts_bundle`.

---

### Impact Explanation

After `replace_class` is called with an undeclared hash `X`:

1. The contract's `StateEntry.class_hash` is set to `X` in `contract_state_changes`.
2. In any subsequent block, a transaction targeting this contract causes `execute_entry_point` to call `dict_read{dict_ptr=contract_class_changes}(key=X)`, which returns `0` (the default for an absent key).
3. `find_element` is then called with `key=0` on the `compiled_class_facts_bundle`. If no compiled class with hash `0` exists, the hint inside `find_element` fails, making it **impossible to generate a valid proof** for any block containing a transaction to this contract.
4. If the contract holds ERC-20 balances or other assets, those assets are **permanently frozen** — no transfer, withdrawal, or interaction can ever be proven again.

**Impact**: Critical — Permanent freezing of funds held by the affected contract.

---

### Likelihood Explanation

- Any user can deploy a contract (via `deploy_account` transaction or `deploy` syscall).
- Any contract can call `replace_class` — it is a standard, unprivileged syscall available to all Cairo 1 contracts.
- The attacker only needs to write a contract whose constructor or any entry point calls `replace_class(0xdeadbeef)` with an arbitrary undeclared felt.
- No special role, leaked key, or privileged access is required.
- The attack is deterministic and requires a single transaction.

---

### Recommendation

In `execute_replace_class`, before updating `contract_state_changes`, verify that the new `class_hash` is present in `contract_class_changes` (i.e., it has been declared in the current or a prior block). Concretely:

```cairo
// Verify the new class hash is declared.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);
```

This mirrors the invariant already enforced in `execute_declare_transaction` (line 816: `assert_not_zero(compiled_class_hash)`) and closes the bypass path.

---

### Proof of Concept

1. **Attacker deploys a contract** whose `__execute__` entry point contains:
   ```
   replace_class(0x123456789abcdef)  // arbitrary undeclared felt
   ```
2. **Attacker sends an invoke transaction** to their contract, triggering `replace_class`.
3. `execute_replace_class` in `syscall_impls.cairo` (lines 896–910) updates the contract's `StateEntry.class_hash` to `0x123456789abcdef` with no validation.
4. The block is proven successfully (the replace itself is valid Cairo execution).
5. **In the next block**, any transaction targeting the attacker's contract (or any contract the attacker controls that holds funds) causes `execute_entry_point` to call `dict_read` on `contract_class_changes` with key `0x123456789abcdef`, returning `0`.
6. `find_element` is called with `key=0` on `compiled_class_facts_bundle` — this element does not exist, causing the prover hint to fail.
7. **The block cannot be proven.** All funds in the contract are permanently frozen. [4](#0-3) [5](#0-4) [6](#0-5)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
