### Title
Missing Declared-Class Validation in `execute_replace_class` Enables Re-Deployment and Constructor Re-Execution — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

`execute_replace_class` in `syscall_impls.cairo` accepts any arbitrary felt as the new class hash without verifying it is a declared class. This allows a contract to set its own class hash to `0` — the sentinel value `UNINITIALIZED_CLASS_HASH` — making the OS treat the contract as never-deployed. A subsequent `deploy` syscall with the original deployment parameters then passes all guards in `deploy_contract`, re-executing the constructor against the existing storage. This enables unlimited re-initialization of contract state (e.g., repeated token minting), constituting a direct loss of funds.

---

### Finding Description

**Root cause — missing validation in `execute_replace_class`:** [1](#0-0) 

```cairo
let class_hash = request.class_hash;
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
```

The function reads `class_hash` directly from the syscall request and writes it unconditionally into the contract's `StateEntry` with no check that the value corresponds to a declared class. The value `0` is explicitly accepted.

**The sentinel value `UNINITIALIZED_CLASS_HASH = 0` is the guard used by `deploy_contract`:** [2](#0-1) [3](#0-2) 

```cairo
assert state_entry.class_hash = UNINITIALIZED_CLASS_HASH;
assert state_entry.nonce = 0;
```

`deploy_contract` uses `class_hash == 0` as the sole proof that an address has never been deployed. After `replace_class(0)`, this invariant is violated: the address appears undeployed even though it has existing storage and history.

**`deploy_contract` then re-initializes the contract with the old storage:** [4](#0-3) 

```cairo
tempvar new_state_entry = new StateEntry(
    class_hash=constructor_execution_context.class_hash,
    storage_ptr=state_entry.storage_ptr,   // ← old storage preserved
    nonce=0,
);
```

The constructor executes against the pre-existing storage, not a blank slate.

### Impact Explanation

**Direct loss of funds (Critical):** If a contract's constructor mints tokens, initializes balances, or grants ownership, re-executing it against existing storage allows those operations to run again. An attacker who controls the contract can repeat the cycle — `replace_class(0)` → re-deploy → constructor runs — indefinitely, minting unbounded tokens or duplicating any financial initialization.

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}

func deploy_contract{
    range_check_ptr,
    remaining_gas: felt,
    builtin_ptrs: BuiltinPointers*,
    contract_state_changes: DictAccess*,
    contract_class_changes: DictAccess*,
    revert_log: RevertLogEntry*,
    outputs: OsCarriedOutputs*,

    retdata_size: felt, retdata: felt*
) {
    alloc_locals;

    local contract_address = constructor_execution_context.execution_info.contract_address;

    // Assert that we don't deploy to one of the reserved addresses.
    assert_not_zero(
        (contract_address - ORIGIN_ADDRESS) * (contract_address - BLOCK_HASH_CONTRACT_ADDRESS) * (
            contract_address - ALIAS_CONTRACT_ADDRESS

    );

    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}
    assert state_entry.class_hash = UNINITIALIZED_CLASS_HASH;
    assert state_entry.nonce = 0;

    tempvar new_state_entry = new StateEntry(
        class_hash=constructor_execution_context.class_hash,
        storage_ptr=state_entry.storage_ptr,
        nonce=0,
    );

        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    // Entries before this point belong to the caller.
    // Note the caller is the deployer (not the caller of the deployer).
    assert [revert_log] = RevertLogEntry(

    let (is_reverted, retdata_size, retdata, _is_deprecated) = select_execute_entry_point_func(

    // Entries before this point belong to the deployed contract.
    assert [revert_log] = RevertLogEntry(selector=CHANGE_CONTRACT_ENTRY, value=contract_address);
    let revert_log = &revert_log[1];

    // The deprecated deploy syscalls do not support reverts.
    assert is_reverted = 0;
    return (retdata_size=retdata_size, retdata=retdata);
