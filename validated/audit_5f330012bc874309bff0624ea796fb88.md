### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Contract Freezing — (`File: crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS does not verify that the caller-supplied `class_hash` corresponds to a previously declared class before writing it into `contract_state_changes`. An unprivileged contract can therefore replace its own class hash with an arbitrary, undeclared felt value. Any funds held by that contract become permanently unrecoverable because no valid class exists to execute against.

---

### Finding Description

The `replace_class` syscall is handled by `execute_replace_class` in `syscall_impls.cairo`. After deducting gas, the function reads the caller-supplied `class_hash` directly from the request and writes it into `contract_state_changes` with no check that the hash exists in `contract_class_changes`:

```cairo

    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),

```

### Impact Explanation

**Impact: Critical — Permanent Freezing of Funds.**

Once the OS commits a `contract_state_changes` entry whose `class_hash` field points to an undeclared class, the global state root is updated to reflect that invalid state. Any subsequent transaction targeting that contract address will fail at class-lookup time because no Sierra/CASM class exists for the stored hash. Storage (and any token balances held in that storage) is permanently inaccessible. Because the OS is the proof-generating component of the ZK-rollup, the invalid state transition is cryptographically finalized and cannot be rolled back.

---

### Likelihood Explanation

The `replace_class` syscall is callable by any deployed contract on itself — no privileged role is required. A malicious contract author can:

1. Deploy a contract that accepts user deposits (e.g., a vault or staking contract).
2. Attract user funds.
3. Issue a `replace_class` syscall with an arbitrary undeclared felt (e.g., `0x1`).
4. The OS processes the call through `execute_replace_class`, writes the invalid class hash, and the proof is generated.

The attack requires only the ability to deploy a contract and submit a transaction — both are unprivileged operations available to any StarkNet user.

---

### Recommendation

Before writing the new `class_hash` into `contract_state_changes`, verify that it exists as a key in `contract_class_changes` (i.e., it has been declared in the current or a prior block). The check should assert `class_hash != UNINITIALIZED_CLASS_HASH` and that a corresponding entry with a non-zero `compiled_class_hash` exists in the class tree. This is precisely what the existing `TODO` comment in `syscall_impls.cairo` line 898 calls for. [3](#0-2) 

---

### Proof of Concept

1. **Deploy attacker vault contract** — a Cairo contract that accepts `deposit()` calls and stores balances in its own storage.
2. **Users deposit funds** — multiple users call `deposit()`, transferring tokens; balances are recorded in the vault's storage slot.
3. **Attacker calls `replace_class(0xdeadbeef)`** — the vault contract issues the `REPLACE_CLASS` syscall with class hash `0xdeadbeef`, which has never been declared.
4. **OS executes `execute_replace_class`** — the function at `syscall_impls.cairo:878` reads `request.class_hash = 0xdeadbeef`, skips the (missing) declared-class check, and calls `dict_update` to write `StateEntry(class_hash=0xdeadbeef, ...)` into `contract_state_changes`.
5. **State is committed** — `state_update` in `state.cairo` squashes and commits the changes; `compute_contract_state_commitment` hashes the new `StateEntry` (including the invalid class hash) into the Patricia tree; the global state root is updated.
6. **Funds are permanently frozen** — any future `call_contract` or `invoke` targeting the vault address causes the OS to look up class `0xdeadbeef` in the class tree, find nothing, and fail. The vault's storage (user balances) is permanently inaccessible. [4](#0-3) [5](#0-4) [6](#0-5)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,

    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    let revert_log = &revert_log[1];

    return ();
}
