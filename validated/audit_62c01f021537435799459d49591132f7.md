### Title
`batch_insert` Bypasses Duplicate Key/Hash Validation, Corrupting DataLayer Merkle Tree Roots - (File: `crates/chia-datalayer/src/merkle/blob.rs`)

### Summary

`MerkleBlob::batch_insert` skips the duplicate-key and duplicate-hash guards that `MerkleBlob::insert` enforces for every item beyond the first two in a batch. Any caller — including an untrusted DataLayer peer — can supply a batch of three or more entries where entries at index ≥ 3 repeat an already-present key or hash. The repeated entries are written directly into the blob without updating the `block_status_cache` consistently, silently corrupting the tree structure and producing an incorrect root hash. Subsequent `get_proof_of_inclusion` calls against the corrupted tree can return proofs that pass `valid()` for the wrong state, or fail for legitimately inserted keys.

### Finding Description

`MerkleBlob::insert` (lines 362–413) enforces two invariants before touching the blob:

```rust
// crates/chia-datalayer/src/merkle/blob.rs  lines 369-374
if self.block_status_cache.contains_key(key) {
    return Err(Error::KeyAlreadyPresent());
}
if self.block_status_cache.contains_leaf_hash(hash) {
    return Err(Error::HashAlreadyPresent());
}
```

`batch_insert` (lines 570–657) calls `insert` only for the first two items when the tree has ≤ 1 existing leaf. For every subsequent item it calls `insert_entry_to_blob` directly, with **no duplicate check**:

```rust
// crates/chia-datalayer/src/merkle/blob.rs  lines 587-603
for ((key, value), hash) in keys_values_hashes {
    let new_leaf_index = self.get_new_index();
    let new_block = Block {
        ...
        node: Node::Leaf(LeafNode {
            parent: Parent(None),
            hash,
            key,
            value,
        }),
    };
    self.insert_entry_to_blob(new_leaf_index, &new_block)?;
    indexes.push(new_leaf_index);
}
```

A batch of `[A, B, C, A_dup]` (where `A_dup` shares the key of `A`) passes items `A` and `B` through the guarded `insert` path, then writes `C` and `A_dup` directly. The blob now contains two leaf nodes with the same key. The internal-node hashes built in the subsequent `while indexes.len() > 1` loop incorporate the duplicate leaf's hash, producing a root hash that does not correspond to any valid tree state. The `block_status_cache` is left inconsistent with the blob, so `get_proof_of_inclusion` traverses stale index mappings and can return a proof whose `valid()` check passes against the corrupted root.

The Python binding `py_batch_insert` (lines 1503–1519) exposes this path directly to any Python caller:

```rust
// crates/chia-datalayer/src/merkle/blob.rs  lines 1503-1519

pub fn py_batch_insert(
    &mut self,
    keys_values: Vec<(KeyId, ValueId)>,
    hashes: Vec<Hash>,

    ...
    self.batch_insert(zip(keys_values, hashes).collect())?;
    Ok(())

```

No privilege check, role check, or whitelist guards either entry point.

### Impact Explanation

Corrupting the DataLayer Merkle tree root falls squarely within the allowed High impact: *"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."*

A corrupted root hash means:
- `ProofOfInclusion::valid()` can return `true` for a key/value pair that was never legitimately inserted (forged inclusion).
- Legitimate keys may become unprovable (denial of valid state).
- Any downstream system that trusts the root hash (e.g., on-chain DataLayer coin commitments) receives an incorrect commitment.

### Likelihood Explanation

`batch_insert` is the standard bulk-sync path used by the DataLayer service when receiving data from a remote peer. A malicious DataLayer peer that the victim is syncing from can craft a sync payload containing ≥ 3 entries where the third or later entry duplicates an earlier key. No cryptographic material, privileged role, or key compromise is required — only the ability to act as a DataLayer data provider.

### Recommendation

Move the duplicate-key and duplicate-hash guards into `batch_insert` itself, before any item is written to the blob, or call the guarded `insert` for every item in the batch rather than only the first two:

```rust
for ((key, value), hash) in keys_values_hashes {
    if self.block_status_cache.contains_key(key) {
        return Err(Error::KeyAlreadyPresent());
    }
    if self.block_status_cache.contains_leaf_hash(&hash) {
        return Err(Error::HashAlreadyPresent());
    }
    // ... existing direct-write path
}
```

Alternatively, pre-validate the entire input vector for uniqueness before touching the blob.

### Proof of Concept

```rust
use chia_datalayer::{Hash, InsertLocation, KeyId, MerkleBlob, ValueId};

fn main() {
    let mut blob = MerkleBlob::new(vec![]).unwrap();

    // Pre-populate with 2 leaves so batch_insert skips the guarded path
    for i in 0i64..2 {
        blob.insert(KeyId(i), ValueId(i), &sha256_num(i), InsertLocation::Auto {}).unwrap();
    }

    // Batch with a duplicate key at position 3 (index 2 in the batch)
    let dup_key = KeyId(0);  // already present
    let batch = vec![
        ((KeyId(10), ValueId(10)), sha256_num(10)),
        ((KeyId(11), ValueId(11)), sha256_num(11)),
        ((dup_key,   ValueId(99)), sha256_num(99)),  // duplicate — no error raised
    ];
    blob.batch_insert(batch).unwrap();  // succeeds silently

    blob.calculate_lazy_hashes().unwrap();

    // check_integrity will now fail, confirming tree corruption
    blob.check_integrity().expect_err("tree is corrupted");
}

    pub fn check_integrity(&self) -> Result<(), Error> {
        self.check_just_integrity()?;

        let mut clone = self.clone();
        clone.check_integrity_on_drop = false;
        clone.calculate_lazy_hashes()?;
        clone.check_just_integrity()
    }

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
