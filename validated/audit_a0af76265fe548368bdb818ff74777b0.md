### Title
Missing Internal Hash Consistency Validation in `MerkleBlob::new` Allows Forged Tree Root — (`crates/chia-datalayer/src/merkle/blob.rs`)

---

### Summary

`MerkleBlob::new` and `BlockStatusCache::new` accept a raw blob with arbitrary internal node hash bytes without verifying that each internal node's stored hash equals `internal_hash(left_hash, right_hash)`. A crafted blob with `dirty = false` on all nodes passes both construction and `check_integrity()` unchanged, causing `get_root_hash()` to return a forged value that differs from the root derivable from the actual leaf hashes.

---

### Finding Description

**Entry point — Python binding:**

`MerkleBlob.__new__(cls, blob: ReadableBuffer)` calls `py_init`, which calls `Self::new(Vec::from(slice))`. [1](#0-0) 

**`MerkleBlob::new` — only checks blob length alignment:** [2](#0-1) 

The only structural check is `length % BLOCK_SIZE != 0`. It then delegates to `BlockStatusCache::new`.

**`BlockStatusCache::new` — only validates leaf-level uniqueness:** [3](#0-2) 

For each node visited, it checks for duplicate `key` and duplicate `leaf.hash` among leaf nodes. For internal nodes it does nothing — **no check that `node.hash == internal_hash(left_child_hash, right_child_hash)`**.

**`check_integrity` — does not recompute hashes for non-dirty nodes:** [4](#0-3) 

`check_integrity` calls `calculate_lazy_hashes()` on a clone, but that function only processes nodes where `metadata.dirty == true`: [5](#0-4) 

A crafted blob with `dirty = false` on all internal nodes causes `calculate_lazy_hashes()` to be a no-op. The subsequent `check_just_integrity()` only validates parent-child index relationships and cache counts — **it never compares any stored hash against a recomputed value**: [6](#0-5) 

**`get_root_hash` — returns stored (forged) hash directly:** [7](#0-6) 

It reads the hash field from the block at index 0 without recomputation.

**`ProofOfInclusion::valid()` — correctly recomputes, but is not called during construction:** [8](#0-7) 

`valid()` does recompute hashes bottom-up and would detect the forgery — but it is only called explicitly by the caller, not during `MerkleBlob::new` or `check_integrity`.

---

### Impact Explanation

Any code path that calls `get_root_hash()` on a `MerkleBlob` loaded from untrusted bytes will receive a forged root hash. In the DataLayer context, root hashes are committed to the blockchain to represent the state of a key-value store. A forged root hash that does not correspond to the actual leaf data corrupts the committed state, enabling a malicious peer to misrepresent the contents of a DataLayer store. This falls within the allowed High impact: **"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."**

---

### Likelihood Explanation

The Python API `MerkleBlob(blob=bytearray(...))` is a documented, exposed constructor. Any DataLayer code path that receives blob bytes from a peer and constructs a `MerkleBlob` from them is directly exploitable. The crafted blob requires only correct block-size alignment and valid leaf key/hash uniqueness — both trivially satisfiable. No cryptographic knowledge is needed; the attacker simply writes arbitrary bytes into the internal node hash fields.

---

### Recommendation

Add hash consistency validation during construction (or at minimum in `check_integrity`). After `BlockStatusCache::new` succeeds, traverse all internal nodes and assert:

```rust
let left_hash = get_hash(node.left)?;
let right_hash = get_hash(node.right)?;
if node.hash != internal_hash(&left_hash, &right_hash) {
    return Err(Error::IntegrityHashMismatch(index));
}
```

This traversal must be done bottom-up (leaves first) so child hashes are available before parent hashes are checked — matching the existing `LeftChildFirstIterator` traversal order already used in `calculate_lazy_hashes`.

---

### Proof of Concept

```python
import struct
from chia_rs.datalayer import MerkleBlob, KeyId, ValueId
from hashlib import sha256

# 1. Build a valid two-leaf blob via normal API
blob = MerkleBlob(blob=bytearray())
blob.insert(KeyId(1), ValueId(1), bytes(range(32)))
blob.insert(KeyId(2), ValueId(2), bytes(range(32, 64)))
blob.calculate_lazy_hashes()

valid_root = blob.get_root_hash()

# 2. Mutate the internal node hash bytes (index 0 = root internal node)
#    Internal node hash occupies bytes [2:34] within the block (after 2 metadata bytes)
raw = bytearray(blob.blob)
BLOCK_SIZE = 116  # from BLOCK_SIZE constant
# Overwrite the hash field of the root internal node with all-0xff
raw[2:34] = b'\xff' * 32

# 3. Construct MerkleBlob from mutated bytes — no exception raised
forged = MerkleBlob(blob=bytes(raw))

forged_root = forged.get_root_hash()

# 4. Demonstrate divergence
assert forged_root != valid_root, "root hashes should differ"
assert forged_root == bytes([0xff] * 32), "forged hash accepted as root"

# 5. check_integrity() also passes (dirty=false, no recomputation)
forged.check_integrity()  # does not raise

print("PASS: forged root accepted:", forged_root.hex())
```

    fn new(blob: &[u8]) -> Result<Self, Error> {
        let index_count = blob.len() / BLOCK_SIZE;

        let mut seen_indexes: BitVec<u64, bitvec::order::Lsb0> = BitVec::repeat(false, index_count);
        let mut key_to_index: HashMap<KeyId, TreeIndex> = HashMap::default();
        let mut leaf_hash_to_index: HashMap<Hash, TreeIndex> = HashMap::default();

        for item in LeftChildFirstIterator::new(blob, None) {
            let (index, block) = item?;
            seen_indexes.set(index.0 as usize, true);

            if let Node::Leaf(leaf) = block.node {
                if key_to_index.insert(leaf.key, index).is_some() {
                    return Err(Error::KeyAlreadyPresent());
                }
                if leaf_hash_to_index.insert(leaf.hash, index).is_some() {
                    return Err(Error::HashAlreadyPresent());
                }
            }
        }

        let mut free_indexes: IndexSet<TreeIndex> = IndexSet::new();
        for (index, seen) in seen_indexes.iter().enumerate() {
            if !seen {
                free_indexes.insert(TreeIndex(index as u32));
            }
        }

        Ok(Self {
            free_indexes,
            key_to_index,
            leaf_hash_to_index,
        })
    }

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
