### Title
`execute_meta_tx_v0` Creates New Transaction Data Layer Without Enforcing Call Stack Depth Limit - (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

`execute_meta_tx_v0` is the StarkNet analog to the Tezos FA bridge proxy contract described in the external report. It creates a new transaction data layer (`TxInfo`) and calls `contract_call_helper` — exactly the pattern the original report flags — but does **not** enforce any call stack depth limit in Cairo code before doing so. A malicious transaction sender can deploy a contract whose `__execute__` function recursively issues `META_TX_V0` syscalls, creating unbounded nested execution contexts. The only protection is gas deduction, which is not a hard Cairo-level assertion and can be bypassed if gas costs are miscalibrated relative to actual prover resource consumption.

---

### Finding Description

`execute_meta_tx_v0` in `syscall_impls.cairo` (lines 286–400) is a syscall that simulates a v0 top-level transaction: it constructs a fresh `TxInfo` with `version=0`, `caller_address=ORIGIN_ADDRESS`, and a newly computed `meta_tx_hash`, then calls `contract_call_helper` to execute the target contract's `__execute__` entry point. [1](#0-0) 

The function deducts gas (lines 301–310) and then unconditionally calls `contract_call_helper` at line 389 with no Cairo assertion checking the current call stack depth: [2](#0-1) [3](#0-2) 

`contract_call_helper` calls `select_execute_entry_point_func`, which contains only a hint `%{ EnterCall %}` for depth tracking — not a Cairo constraint: [4](#0-3) 

Hints are not enforced by the Cairo verifier. No `assert` or `assert_le` against a `MAX_CALL_DEPTH` constant exists anywhere in the call path. The constants template confirms no such constant is defined: [5](#0-4) 

The `execute_syscalls` dispatcher in `execute_syscalls.cairo` handles `META_TX_V0_SELECTOR` as the final case with no guard: [6](#0-5) 

This means a contract called via `execute_meta_tx_v0` can itself issue another `META_TX_V0` syscall, which `execute_syscalls` will dispatch again, creating unbounded recursive execution. Each recursive level allocates new Cairo memory segments (new `TxInfo`, new `ExecutionInfo`, new `ExecutionContext`, new `deprecated_tx_info` buffer), none of which are bounded by a depth assertion.

---

### Impact Explanation

**High — Network not being able to confirm new transactions (total network shutdown).**

Each recursive `execute_meta_tx_v0` invocation allocates multiple heap segments (`new TxInfo`, `new ExecutionInfo`, `new ExecutionContext`, `alloc()` for `deprecated_tx_info`). These allocations grow the Cairo memory linearly with recursion depth. If `META_TX_V0_GAS_COST` is not calibrated to account for the full per-level memory overhead (segment allocation, Pedersen hash computation for `meta_tx_hash`, revert log entries), a malicious transaction can exhaust prover memory before gas runs out. A prover that cannot complete proof generation for a block halts the network's ability to confirm new transactions.

Additionally, `execute_meta_tx_v0` is semantically designed to simulate a **top-level** transaction (it sets `caller_address=ORIGIN_ADDRESS` and creates a fresh `TxInfo`). Allowing it to be called recursively from within another `execute_meta_tx_v0` context creates nested "top-level" transactions, which is structurally incorrect and could produce malformed OS output commitments, risking a chain split between nodes that accept vs. reject such proofs.

---

### Likelihood Explanation

**Medium.** Any unprivileged transaction sender can deploy a Cairo 1 contract whose `__execute__` function issues a `META_TX_V0` syscall targeting itself or another contract that does the same. The `META_TX_V0_SELECTOR` is reachable from `execute_syscalls` without any privilege check beyond gas. The attack requires only a deployed contract and a submitted invoke transaction — no operator access, no leaked keys, no Sybil attack.

---

### Recommendation

Before calling `contract_call_helper` in `execute_meta_tx_v0`, add a Cairo-level assertion enforcing a hard call stack depth limit, analogous to the fix applied in Tezos MR 15358. Concretely:

1. Add a `call_depth` field to `ExecutionContext` or pass it as an explicit parameter through the call chain.
2. In `execute_meta_tx_v0`, assert `call_depth < MAX_META_TX_CALL_DEPTH` before constructing the new `ExecutionContext`.
3. Define `MAX_META_TX_CALL_DEPTH` as a named constant in `constants_template.txt` / `constants.cairo`.
4. Ensure `META_TX_V0_GAS_COST` accounts for the full per-level memory allocation overhead (Pedersen hash, `TxInfo`, `ExecutionInfo`, `ExecutionContext`, revert log entries).

---

### Proof of Concept

**Attack contract (`__execute__` entry point, pseudocode):**
```
fn __execute__(calls: Array<Call>) -> Array<Span<felt252>> {
    // Issue META_TX_V0 syscall targeting self with same calldata.
    // Each invocation creates a new TxInfo layer with no depth check.
    starknet::syscalls::meta_tx_v0(
        contract_address: get_contract_address(),
        selector: '__execute__',
        calldata: calls.span(),
        signature: array![].span(),
    );
    array![]
}
```

**Call chain in the OS:**

```
execute_syscalls (execute_syscalls.cairo:343)
  └─ execute_meta_tx_v0 (syscall_impls.cairo:286)
       ├─ [gas deducted, no depth check]
       ├─ new TxInfo allocated (syscall_impls.cairo:343-363)
       ├─ new ExecutionContext allocated (syscall_impls.cairo:366-379)
       └─ contract_call_helper (syscall_impls.cairo:389)
            └─ select_execute_entry_point_func (entry_point_utils.cairo:17)
                 └─ execute_entry_point → __execute__ → META_TX_V0 syscall
                      └─ execute_syscalls (execute_syscalls.cairo:343)  ← recurse
                           └─ execute_meta_tx_v0 [no depth check] ...
```

Each level allocates new Cairo memory. With no Cairo assertion bounding depth, the recursion continues until gas is exhausted or prover memory is exceeded — whichever comes first. If `META_TX_V0_GAS_COST` underestimates per-level memory cost, prover memory is exhausted first, halting the network. [7](#0-6) [6](#0-5) [4](#0-3)

    range_check_ptr,
    syscall_ptr: felt*,
    builtin_ptrs: BuiltinPointers*,
    contract_state_changes: DictAccess*,
    contract_class_changes: DictAccess*,
    revert_log: RevertLogEntry*,
    outputs: OsCarriedOutputs*,
}(block_context: BlockContext*, caller_execution_context: ExecutionContext*) {
    alloc_locals;

    let request = cast(syscall_ptr + RequestHeader.SIZE, MetaTxV0Request*);
    local calldata_start: felt* = request.calldata_start;
    local calldata_size = request.calldata_end - calldata_start;

    let specific_base_gas_cost = (
        META_TX_V0_GAS_COST + META_TX_V0_CALLDATA_FACTOR_GAS_COST * calldata_size
    );
    let (success, remaining_gas) = reduce_syscall_base_gas(
        specific_base_gas_cost=specific_base_gas_cost, request_struct_size=MetaTxV0Request.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    local contract_address = request.contract_address;
    local selector = request.selector;
    local caller_execution_info: ExecutionInfo* = caller_execution_context.execution_info;
    local old_tx_info: TxInfo* = caller_execution_info.tx_info;

    if (selector != EXECUTE_ENTRY_POINT_SELECTOR) {
        write_failure_response(remaining_gas=remaining_gas, failure_felt=ERROR_INVALID_ARGUMENT);
        return ();
    }

    // Sanity check: Verify that `signature` is a valid Sierra array.
    assert_nn_le(request.signature_end - request.signature_start, SIERRA_ARRAY_LEN_BOUND - 1);

    let (state_entry: StateEntry*) = dict_read{dict_ptr=contract_state_changes}(
        key=contract_address
    );

    // Compute the meta-transaction hash.
    let pedersen_ptr = builtin_ptrs.selectable.pedersen;
    with pedersen_ptr {
        let meta_tx_hash = compute_meta_tx_v0_hash(
            contract_address=contract_address,
            entry_point_selector=selector,
            calldata=calldata_start,
            calldata_size=calldata_size,
            chain_id=old_tx_info.chain_id,
        );
    }
    update_pedersen_in_builtin_ptrs(pedersen_ptr=pedersen_ptr);

    // Prepare execution context.
    tempvar new_tx_info = new TxInfo(
        version=0,
        account_contract_address=contract_address,
        max_fee=0,
        signature_start=request.signature_start,
        signature_end=request.signature_end,
        transaction_hash=meta_tx_hash,
        chain_id=old_tx_info.chain_id,
        nonce=0,
        resource_bounds_start=cast(0, ResourceBounds*),
        resource_bounds_end=cast(0, ResourceBounds*),
        tip=0,
        paymaster_data_start=cast(0, felt*),
        paymaster_data_end=cast(0, felt*),
        nonce_data_availability_mode=0,
        fee_data_availability_mode=0,
        account_deployment_data_start=cast(0, felt*),
        account_deployment_data_end=cast(0, felt*),
        proof_facts_start=cast(0, felt*),
        proof_facts_end=cast(0, felt*),
    );

    let (deprecated_tx_info_ptr: DeprecatedTxInfo*) = alloc();
    tempvar execution_context: ExecutionContext* = new ExecutionContext(
        entry_point_type=ENTRY_POINT_TYPE_EXTERNAL,
        class_hash=state_entry.class_hash,
        calldata_size=calldata_size,
        calldata=calldata_start,
        execution_info=new ExecutionInfo(
            block_info=caller_execution_info.block_info,
            tx_info=new_tx_info,
            caller_address=ORIGIN_ADDRESS,
            contract_address=contract_address,
            selector=selector,
        ),
        deprecated_tx_info=deprecated_tx_info_ptr,
    );
    fill_deprecated_tx_info(tx_info=new_tx_info, dst=execution_context.deprecated_tx_info);

    // Since we process the revert log backwards, entries before this point belong to the calling
    // contract.
    assert [revert_log] = RevertLogEntry(
        selector=CHANGE_CONTRACT_ENTRY, value=caller_execution_info.contract_address
    );
    let revert_log = &revert_log[1];

    contract_call_helper(
        remaining_gas=remaining_gas,
        block_context=block_context,
        execution_context=execution_context,
    );

    // Entries before this point belong to the callee.
    assert [revert_log] = RevertLogEntry(selector=CHANGE_CONTRACT_ENTRY, value=contract_address);
    let revert_log = &revert_log[1];

    return ();
}
