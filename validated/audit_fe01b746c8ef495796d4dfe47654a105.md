### Title
Stale `feemarketParams` Snapshot in Cosmos Ante Handler Allows Fee Underpayment on Cosmos Transactions - (File: evmd/ante/handler_options.go)

### Summary
`newCosmosAnteHandler` and `newLegacyCosmosAnteHandlerEip712` read `feemarketParams` (including `BaseFee`) once at construction time and pass a pointer to that frozen snapshot into `NewDynamicFeeChecker` and `MinGasPriceDecorator`. Because the EIP-1559 base fee is updated every block in `BeginBlock`, the fee-validation closures permanently operate against a stale base fee, allowing Cosmos-SDK transactions to be admitted and committed with fees below the current required base fee.

### Finding Description

In `evmd/ante/handler_options.go`, both `newCosmosAnteHandler` and `newLegacyCosmosAnteHandlerEip712` snapshot `feemarketParams` at construction time:

```go
feemarketParams := options.FeeMarketKeeper.GetParams(ctx)   // read once
txFeeChecker = evm.NewDynamicFeeChecker(ethCfg, &evmParams, &feemarketParams)
cosmos.NewMinGasPriceDecorator(options.FeeMarketKeeper, evmDenom, &feemarketParams)
``` [1](#0-0) 

The pointer `&feemarketParams` is captured by the returned `sdk.AnteHandler` closure. `feemarketParams` is a local value-type variable; it is never refreshed after the function returns.

Inside `NewDynamicFeeChecker`, every invocation calls:

```go
baseFee := types.GetBaseFee(ctx.BlockHeight(), ethCfg, feemarketParams)
```

func newCosmosAnteHandler(ctx sdk.Context, options HandlerOptions, extra ...sdk.AnteDecorator) sdk.AnteHandler {
	evmParams := options.EvmKeeper.GetParams(ctx)
	feemarketParams := options.FeeMarketKeeper.GetParams(ctx)
	evmDenom := evmParams.EvmDenom
	chainID := options.EvmKeeper.ChainID()
	chainCfg := evmParams.GetChainConfig()
	ethCfg := chainCfg.EthereumConfig(chainID)
	var txFeeChecker ante.TxFeeChecker
	if options.DynamicFeeChecker {
		txFeeChecker = evm.NewDynamicFeeChecker(ethCfg, &evmParams, &feemarketParams)
	}
	decorators := make([]sdk.AnteDecorator, 0, 16+len(extra))
	decorators = append(decorators,
		cosmos.RejectMessagesDecorator{}, // reject MsgEthereumTxs
		// disable the Msg types that cannot be included on an authz.MsgExec msgs field
		cosmos.NewAuthzLimiterDecorator(options.DisabledAuthzMsgs),
		ante.NewSetUpContextDecorator(),
		ante.NewExtensionOptionsDecorator(options.ExtensionOptionChecker),
		ante.NewValidateBasicDecorator(),
		ante.NewTxTimeoutHeightDecorator(),
		cosmos.NewMinGasPriceDecorator(options.FeeMarketKeeper, evmDenom, &feemarketParams),
