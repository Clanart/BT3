Looking at the key code paths in detail:

### Title
Forged Internal-Node Hash Accepted by `build_blob_from_node_list` Without Verification — (`crates/chia-datalayer/src/merkle/blob.rs` / `deltas.rs`)

### Summary
`DeltaReader::new` and `py_add_internal_nodes` accept an `InternalNodesMap` whose keys are arbitrary hashes with no requirement that `key == internal_hash(left, right)`. `inner_build_blob_from_node_list` blindly copies the map key into the stored `InternalNode::hash` field without recomputing it from children. `check_integrity()` does not catch the inconsistency because `calculate_lazy_hashes()` only recomputes hashes for nodes marked `dirty`, and all nodes written by `build_blob_from_node_list` are written with `dirty: false`. The result is a `MerkleBlob` whose `get_root_hash()` returns a value that does not reflect the actual leaf contents.

### Finding Description

**Step 1 — No hash validation on ingestion.**

`DeltaReader::new` inserts caller-supplied entries verbatim:

```rust
for (hash, (left, right)) in internal_nodes {
    nodes.insert(hash, DeltaReaderNode::Internal { left, right });
}
``` [1](#0-0) 

There is no assertion that `hash == internal_hash(left, right)`. The same is true for `py_add_internal_nodes`: [2](#0-1) 

**Step 2 — `inner_build_blob_from_node_list` uses the map key as the stored hash.**

When reconstructing an `Internal` node the function writes `hash: node_hash` — the map key — directly into the blob, without ever calling `internal_hash(left_hash, right_hash)`:

```rust
node: Node::Internal(InternalNode {
    hash: node_hash,   // ← map key, not recomputed
    parent: Parent(None),
    left: left_index,
    right: right_index,
}),
``` [3](#0-2) 

The block is then written with `dirty: false`: [4](#0-3) 

**Step 3 — `check_integrity()` does not catch the inconsistency.**

`check_integrity` calls `calculate_lazy_hashes()` on a clone, but that function only processes nodes where `block.metadata.dirty == true`:

```rust
Some(|block: &Block| block.metadata.dirty),
``` [5](#0-4) 

Because every node written by `inner_build_blob_from_node_list` has `dirty: false`, no hash is ever recomputed. `check_just_integrity()` only validates parent-child pointer consistency, key-to-index cache lengths, and total node counts — it never asserts `internal_node.hash == internal_hash(left_child.hash, right_child.hash)`: [6](#0-5) 

**Step 4 — `get_root_hash()` returns the forged value.**

`py_get_root_hash` calls `get_hash_at_index(TreeIndex(0))`, which reads the stored hash from the blob without recomputing it: [7](#0-6) 

### Impact Explanation

A DataLayer peer serving delta data can supply an `InternalNodesMap` where the key for the root node is the legitimate on-chain root hash `H_real`, but the children point to arbitrary leaf data. The syncing node reconstructs a `MerkleBlob` whose `get_root_hash()` returns `H_real` (matching the on-chain commitment), yet whose leaf contents are entirely attacker-controlled. Any subsequent proof-of-inclusion generated from this blob, or any business logic that reads key-value pairs from it, operates on forged data. This matches the allowed High impact: *DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state.*

### Likelihood Explanation

The Python bindings (`py_init`, `py_add_internal_nodes`) are the realistic entry point: DataLayer sync receives delta node maps from untrusted peers over the network and passes them directly to `DeltaReader`. No privilege is required beyond being a reachable DataLayer peer. The on-chain root hash is public, so the attacker knows exactly which key to use for the root internal node.

### Recommendation

1. In `DeltaReader::new` and `py_add_internal_nodes`, validate each inserted internal node: `assert_eq!(hash, internal_hash(left, right))` and return an error on mismatch.
2. In `inner_build_blob_from_node_list`, recompute the internal hash from the already-resolved child hashes and compare it against `node_hash`, returning `Error` on mismatch.
3. Extend `check_just_integrity()` to verify `internal_node.hash == internal_hash(left_child_hash, right_child_hash)` for every internal node regardless of the dirty flag.

### Proof of Concept

```rust
use std::collections::{HashMap, HashSet};
use chia_datalayer::{
    DeltaReader, InternalNodesMap, LeafNodesMap, KeyId, ValueId, Hash,
};
use chia_datalayer::merkle::blob::internal_hash;

// Legitimate leaf hashes
let h_left  = Hash::from([0x01u8; 32]);
let h_right = Hash::from([0x02u8; 32]);

// Forged root hash — NOT equal to internal_hash(h_left, h_right)
let h_fake  = Hash::from([0xFFu8; 32]);

let mut internal_nodes: InternalNodesMap = HashMap::new();
internal_nodes.insert(h_fake, (h_left, h_right));   // key ≠ internal_hash(left,right)

let mut leaf_nodes: LeafNodesMap = HashMap::new();
leaf_nodes.insert(h_left,  (KeyId(1), ValueId(10)));
leaf_nodes.insert(h_right, (KeyId(2), ValueId(20)));

let mut reader = DeltaReader::new(internal_nodes, leaf_nodes).unwrap();
let blob = reader
    .create_merkle_blob_and_filter_unused_nodes(h_fake, &HashSet::new())
    .unwrap();

// get_root_hash() returns h_fake, not internal_hash(h_left, h_right)
let root = blob.get_hash_at_index(TreeIndex(0)).unwrap().unwrap();
assert_eq!(root, h_fake);                                    // passes — forged hash accepted
assert_ne!(root, internal_hash(&h_left, &h_right));          // passes — hash is inconsistent
blob.check_integrity().unwrap();                             // passes — no hash math checked
```

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
