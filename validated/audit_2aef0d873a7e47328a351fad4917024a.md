### Title
Missing Lower Bound on L2 Gas `max_amount` Causes Unprovable Block When `__validate__` Runs Out of Gas — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/transaction_impls.cairo`)

---

### Summary

`get_initial_user_gas_bound()` returns the user-supplied `resource_bounds[L2_GAS_INDEX].max_amount` directly as the initial gas for transaction execution, with no lower bound enforcement. If a transaction sets this value below `ENTRY_POINT_INITIAL_BUDGET` (10,000), the `__validate__` entry point returns `is_reverted=1` due to out-of-gas. `non_reverting_select_execute_entry_point_func` then unconditionally asserts `is_reverted = 0`, producing a Cairo assertion failure that makes the entire block unprovable.

---

### Finding Description

**Root cause — no minimum gas bound in `get_initial_user_gas_bound()`:** [1](#0-0) 

```cairo
func get_initial_user_gas_bound(common_tx_fields: CommonTxFields*) -> felt {
    assert common_tx_fields.n_resource_bounds = 3;
    return common_tx_fields.resource_bounds[L2_GAS_INDEX].max_amount;
}
```

The function returns the raw user-supplied `max_amount` with no lower bound check. This value becomes `remaining_gas` for the validate and execute phases of every invoke, declare, and deploy-account transaction.

**`cap_remaining_gas` only enforces a maximum, not a minimum:** [2](#0-1) 

```cairo
func cap_remaining_gas{range_check_ptr, remaining_gas: felt}(max_gas: felt) {
    ...
    if (remaining_gas_gt_max != FALSE) {
        assert_nn_le(max_gas, remaining_gas - 1);
        tempvar remaining_gas = max_gas;
    } else {
        assert_nn_le(remaining_gas, max_gas);
        tempvar remaining_gas = remaining_gas;
    }
    return ();
}
```

`remaining_gas = 0` passes through unchanged.

**`execute_entry_point` returns `is_reverted=1` when gas is below `ENTRY_POINT_INITIAL_BUDGET`:** [3](#0-2) 

```cairo
if (is_remaining_gas_lt_initial_budget != FALSE) {
    assert_lt(remaining_gas, ENTRY_POINT_INITIAL_BUDGET);
    ...
    assert retdata[0] = ERROR_OUT_OF_GAS;
    return (is_reverted=1, retdata_size=1, retdata=retdata);
}
let remaining_gas = remaining_gas - ENTRY_POINT_INITIAL_BUDGET;
assert_nn(remaining_gas);
```

`ENTRY_POINT_INITIAL_BUDGET = 10,000`. Any `remaining_gas < 10,000` triggers this path. [4](#0-3) 

**`non_reverting_select_execute_entry_point_func` asserts `is_reverted = 0` — fatal for the block proof:** [5](#0-4) 

```cairo

    let (is_reverted, retdata_size, retdata, is_deprecated) = select_execute_entry_point_func{...}(...);
    assert is_reverted = 0;   // <-- Cairo assertion failure if __validate__ ran out of gas
