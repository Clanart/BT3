### Title
`ExecutionStateReader::get_compiled_class_hash` Ignores `state_number`, Returning Post-Declaration Data at Pre-Declaration Historical Blocks - (File: crates/apollo_rpc_execution/src/state_reader.rs)

### Summary

`ExecutionStateReader::get_compiled_class_hash` completely ignores `self.state_number` — the versioned block anchor that all other state-read methods in the same struct honour — and instead unconditionally returns the compiled class hash from the block at which the class was actually declared. When an RPC caller simulates or estimates fees for a `Declare` transaction against a historical block that precedes the class's declaration, the function returns a non-zero compiled class hash, causing the blockifier to treat the class as already declared and producing a wrong authoritative simulation result.

---

### Finding Description

`ExecutionStateReader` carries a `state_number: StateNumber` field that pins every state read to a specific historical block. Every other `BlockifierStateReader` method in the struct passes `self.state_number` to the underlying storage query:

- `get_storage_at` → `execution_utils::get_storage_at(…, self.state_number, …)` [1](#0-0) 
- `get_nonce_at` → `execution_utils::get_nonce_at(…, self.state_number, …)` [2](#0-1) 
- `get_class_hash_at` → `execution_utils::get_class_hash_at(…, self.state_number, …)` [3](#0-2) 
- `get_compiled_class` → `is_contract_class_declared(…, self.state_number)` / `get_contract_class(…, self.state_number)` [4](#0-3) 

`get_compiled_class_hash`, however, never touches `self.state_number`:

```rust
fn get_compiled_class_hash(&self, class_hash: ClassHash) -> StateResult<CompiledClassHash> {
    // pending data check …

    let maybe_block_number = self
        .storage_reader
        .begin_ro_txn()…
        .get_class_definition_block_number(&class_hash)?;   // ← finds actual declaration block

    let Some(block_number) = maybe_block_number else {
        return Ok(CompiledClassHash::default());
    };

    let state_diff = self
        .storage_reader
        .begin_ro_txn()…
        .get_state_diff(block_number)?…;                    // ← reads from declaration block

    let compiled_class_hash = state_diff
        .class_hash_to_compiled_class_hash
        .get(&class_hash)…;

    Ok(*compiled_class_hash)                                // ← returned regardless of self.state_number
}
``` [5](#0-4) 

`get_class_definition_block_number` returns the block at which the class was first declared, stored in the `declared_classes_block` table as a simple key→block mapping with no version guard. [6](#0-5) 

The missing guard is the one that `get_compiled_class` applies correctly:

```rust
Some(block_number) if state_number.is_before(block_number) => {
    return Ok(None);   // class not yet declared at requested state
}
``` [7](#0-6) 

`get_compiled_class_hash` has no equivalent check, so it returns the compiled class hash from block M even when `self.state_number` is right after block N < M.

The `compiled_class_hash` table itself is block-versioned (keyed by `(class_hash, block_number)`) and the storage-layer helper `get_compiled_class_hash_at` correctly performs a range scan bounded by `state_number`. [8](#0-7)  The bug is that `ExecutionStateReader::get_compiled_class_hash` bypasses this versioned path entirely and reads the raw state diff at the declaration block.

``` [9](#0-8)

### Impact Explanation

`ExecutionStateReader` is the state reader used by `exec_estimate_fee`, `exec_simulate_transactions`, and `starknet_call` in the RPC layer. [10](#0-9) 

When a caller invokes `starknet_simulateTransactions` or `starknet_estimateFee` with a historical `block_id = N` and a `DeclareV2/V3` transaction for a class that was declared at block M > N:

1. `get_compiled_class` correctly returns `UndeclaredClassHash` for the class at state N (the class does not exist yet). [11](#0-10) 
2. `get_compiled_class_hash` incorrectly returns the non-zero compiled class hash from block M.
3. `CachedState` caches this non-zero value as the *initial* compiled class hash for the class. [12](#0-11) 
4. The blockifier's state diff records a transition from the non-zero "initial" value to the new value declared by the transaction, producing a wrong `class_hash_to_compiled_class_hash` entry in the simulated `ThinStateDiff`.
5. Depending on blockifier internals, the transaction may also be rejected as a re-declaration, causing the simulation to return a spurious revert where the real execution at block N would have succeeded.

The corrupted value is the `class_hash_to_compiled_class_hash` mapping inside the simulated state diff, and the wrong result is the simulation/fee-estimation response returned to the caller — an authoritative-looking wrong value from the RPC layer.

This falls under: **High. RPC execution, fee estimation, tracing, simulation, or pending view returns an authoritative-looking wrong value.**

---

### Likelihood Explanation

The trigger requires:
- A class declared at block M to already exist in storage.
- An RPC call that simulates a `Declare` for the same class hash at a historical block N < M.

This is an unusual but fully unprivileged scenario. Any external caller can craft such a request. The condition becomes more likely as the chain grows and more classes accumulate. No special permissions, malicious peers, or privileged operations are required.

---

### Recommendation

Add the missing `state_number` guard in `ExecutionStateReader::get_compiled_class_hash`, mirroring the check already present in `get_contract_class`:

```rust
fn get_compiled_class_hash(&self, class_hash: ClassHash) -> StateResult<CompiledClassHash> {
    // … pending data check unchanged …

    let maybe_block_number = self
        .storage_reader
        .begin_ro_txn()
        .map_err(storage_err_to_state_err)?
        .get_state_reader()
        .map_err(storage_err_to_state_err)?
        .get_class_definition_block_number(&class_hash)
        .map_err(storage_err_to_state_err)?;

    let Some(block_number) = maybe_block_number else {
        return Ok(CompiledClassHash::default());
    };

    // *** ADD THIS GUARD ***
    if self.state_number.is_before(block_number) {
        return Ok(CompiledClassHash::default());
    }

    // … rest unchanged …
}
```

Alternatively, replace the manual state-diff lookup with the versioned storage helper `get_compiled_class_hash_at(state_number, &class_hash)` which already performs the correct range-bounded scan. [8](#0-7) 

Add a regression test for `get_compiled_class_hash` at a historical block preceding the class declaration, analogous to the existing `get_compiled_class` test. [13](#0-12)

### Proof of Concept

**Setup:**
1. Block 0: declare class `C` with compiled class hash `H_C`. Storage now has `declared_classes_block[C] = 0` and `state_diff[0].class_hash_to_compiled_class_hash[C] = H_C`.
2. Block 1: some unrelated block.

**Trigger:**
```
POST /rpc/v0_8
{
  "method": "starknet_V0_8_simulateTransactions",
  "params": {
    "block_id": {"block_number": 0},   // ← historical block BEFORE declaration
    "transactions": [{ "type": "DECLARE", "class_hash": C, ... }],
    "simulation_flags": []
  }
}
```

Wait — block 0 is the declaration block itself. Use block_id pointing to a block *before* block 0 is not possible (genesis). Adjust: declare at block 1, simulate at block 0.

**Adjusted Setup:**
1. Block 0: empty.
2. Block 1: declare class `C` → `declared_classes_block[C] = 1`, `state_diff[1].class_hash_to_compiled_class_hash[C] = H_C`.

**Trigger:**
```
starknet_simulateTransactions(block_id={"block_number": 0}, txs=[DeclareV2(C)])
```

**Expected:** simulation succeeds; state diff shows `C → H_C` (class newly declared).

**Actual:** `get_compiled_class_hash(C)` ignores `state_number = right_after(0)`, finds `declared_classes_block[C] = 1`, reads `state_diff[1]`, returns `H_C` (non-zero). The blockifier's `CachedState` records `H_C` as the initial value. The simulated state diff either shows a wrong `H_C → H_C` no-op transition, or the blockifier rejects the transaction as a re-declaration, returning a spurious revert — both are wrong authoritative simulation results.     fn get_compiled_class_hash(&self, class_hash: ClassHash) -> StateResult<CompiledClassHash> {
        if let Some(pending_data) = &self.maybe_pending_data {
            for DeclaredClassHashEntry { class_hash: other_class_hash, compiled_class_hash } in
                &pending_data.declared_classes
            {
                if class_hash == *other_class_hash {
                    return Ok(*compiled_class_hash);
                }
            }
        }

        let maybe_block_number = self
            .storage_reader
            .begin_ro_txn()
            .map_err(storage_err_to_state_err)?
            .get_state_reader()
            .map_err(storage_err_to_state_err)?
            .get_class_definition_block_number(&class_hash)
            .map_err(storage_err_to_state_err)?;

        // Cairo 0 classes (and undeclared classes) do not have a compiled class hash.
        // According to the trait, return the default value.
        let Some(block_number) = maybe_block_number else {
            return Ok(CompiledClassHash::default());
        };

        let state_diff = self
            .storage_reader
            .begin_ro_txn()
            .map_err(storage_err_to_state_err)?
            .get_state_diff(block_number)
            .map_err(storage_err_to_state_err)?
            .ok_or(StateError::StateReadError(format!(
                "Inner storage error. Missing state diff at block {block_number}."
            )))?;

        let compiled_class_hash = state_diff
            .class_hash_to_compiled_class_hash
            .get(&class_hash)
            .ok_or(StateError::StateReadError(format!(
                "Inner storage error. Missing class declaration at block {block_number}, class \
                 {class_hash}."
            )))?;

        Ok(*compiled_class_hash)
    }

pub(crate) fn get_contract_class(
    txn: &StorageTxn<'_, RO>,
    class_hash: &ClassHash,
    state_number: StateNumber,
) -> Result<Option<RunnableCompiledClass>, ExecutionUtilsError> {
    match txn.get_state_reader()?.get_class_definition_block_number(class_hash)? {
        Some(block_number) if state_number.is_before(block_number) => return Ok(None),
        Some(_block_number) => {
            let (Some(casm), Some(sierra)) = txn.get_casm_and_sierra(class_hash)? else {
                return Err(ExecutionUtilsError::CasmTableNotSynced);
            };
            let sierra_version =
                sierra.get_sierra_version().map_err(ExecutionUtilsError::SierraValidationError)?;
            return Ok(Some(RunnableCompiledClass::V1(CompiledClassV1::try_from((
                casm,
                sierra_version,
            ))?)));
        }
        None => {}
    };

#[test]
fn get_compiled_class() {
    let casm_contract_class = CasmContractClass {
        compiler_version: "0.0.0".to_string(),
        prime: Default::default(),
        bytecode: Default::default(),
        bytecode_segment_lengths: Default::default(),
        hints: Default::default(),
        pythonic_hints: Default::default(),
        entry_points_by_type: Default::default(),
    };
    let expected_result = casm_contract_class.clone();

    let mut mock_class_manager_client = MockClassManagerClient::new();
    mock_class_manager_client.expect_get_executable().returning(move |_| {
        Ok(Some(ContractClass::V1((casm_contract_class.clone(), SierraVersion::default()))))
    });
    let shared_mock_class_manager_client = Arc::new(mock_class_manager_client);

    // Set execution state reader with state_number to be right after block 0.
    let (mut exec_state_reader, mut storage_writer) =
        set_execution_state_reader(shared_mock_class_manager_client.clone(), 0);

    // Prepare storage so class with hash 0x2 is declared in block 0 and class with hash 0x3 is
    // declared in block 1.
    let class_hash_0x2 = class_hash!("0x2");
    let class_hash_0x3 = class_hash!("0x3");

    storage_writer
        .begin_rw_txn()
        .unwrap()
        .append_header(BlockNumber(0), &BlockHeader::default())
        .unwrap()
        .append_state_diff(
            BlockNumber(0),
            ThinStateDiff {
                class_hash_to_compiled_class_hash: indexmap!(class_hash_0x2 => compiled_class_hash!(1_u8)),
                ..Default::default()
            },
        )
        .unwrap()
        .append_header(
            BlockNumber(1),
            &BlockHeader { block_hash: BlockHash(felt!(1_u128)), ..Default::default() },
        )
        .unwrap()
        .append_state_diff(
            BlockNumber(1),
            ThinStateDiff {
                class_hash_to_compiled_class_hash: indexmap!(class_hash_0x3 => compiled_class_hash!(2_u8)),
                ..Default::default()
            },
        )
        .unwrap()
        .commit()
        .unwrap();

    // At state_number right after block 0, class with hash 0x2 is already declared.
    let result = exec_state_reader.get_compiled_class(class_hash_0x2).unwrap();
    assert_eq!(
        result,
        RunnableCompiledClass::V1(
            (expected_result.clone(), SierraVersion::default()).try_into().unwrap()
        )
    );

    // Class with hash 0x3 is not yet declared.
    let result = exec_state_reader.get_compiled_class(class_hash_0x3);
    assert_matches!(result, Err(StateError::UndeclaredClassHash(hash)) if hash == class_hash_0x3);

    // Adjust execution state reader state_number to be right after block 1.
    exec_state_reader.state_number = StateNumber::unchecked_right_after_block(BlockNumber(1));

    // Class with hash 0x2 is still declared.
    let result = exec_state_reader.get_compiled_class(class_hash_0x2).unwrap();
    assert_eq!(
        result,
        RunnableCompiledClass::V1(
            (expected_result.clone(), SierraVersion::default()).try_into().unwrap()
        )
    );

    // And class with hash 0x3 is declared too.
    let result = exec_state_reader.get_compiled_class(class_hash_0x3).unwrap();
    assert_eq!(
        result,
        RunnableCompiledClass::V1((expected_result, SierraVersion::default()).try_into().unwrap())
    );
}
