### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Contract Freezing — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall implementation in the StarkNet OS does not verify that the replacement class hash is actually declared in the contract-class tree before committing the state change. Any contract (an unprivileged actor) can call `replace_class` with an arbitrary, undeclared class hash. Once committed, every subsequent call to that contract will fail at the OS level because the compiled class cannot be resolved, permanently freezing any funds held by the contract.

---

### Finding Description

`execute_replace_class` in `syscall_impls.cairo` reads the requested class hash directly from the syscall request and writes it into `contract_state_changes` without any check that the hash exists in the declared-class tree:

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
```

The TODO comment is the codebase's own acknowledgment that this validation is absent.

When a subsequent transaction calls the affected contract, `execute_entry_point` performs:

```cairo
let (compiled_class_hash: felt) = dict_read{dict_ptr=contract_class_changes}(
    key=execution_context.class_hash
);
...
let (compiled_class_fact: CompiledClassFact*) = find_element(
    array_ptr=compiled_class_facts_bundle.compiled_class_facts,
    elm_size=CompiledClassFact.SIZE,
    n_elms=compiled_class_facts_bundle.n_compiled_class_facts,
    key=compiled_class_hash,
);
```

If `class_hash` is undeclared, `dict_read` returns `0` (the default for an uninitialised dict entry). `find_element` then searches for `compiled_class_hash = 0`, which is not present in `compiled_class_facts_bundle`, causing a Cairo assertion failure. The block cannot be proven, and the contract is permanently uncallable.

The syscall is dispatched unconditionally to any contract in `execute_syscalls.cairo`:

```cairo
if (selector == REPLACE_CLASS_SELECTOR) {
    execute_replace_class(contract_address=execution_context.execution_info.contract_address);
    ...
}
```

No privilege check, no declared-class check.

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Any ETH, STRK, or ERC-20 tokens held in the storage of the affected contract become permanently inaccessible. Because every call to the contract triggers an OS-level proof failure, no withdrawal, transfer, or recovery transaction can ever be included in a valid block for that contract. The state change is committed on-chain and irreversible.

If the affected contract is a shared vault, AMM, or token contract, all depositors lose access to their funds simultaneously.

---

### Likelihood Explanation

**Medium.**

- The `replace_class` syscall is available to every Cairo contract with no access restriction.
- A single buggy upgrade path in any widely-used contract (vault, bridge, token) is sufficient to trigger the freeze.
- The missing check is explicitly flagged in the source with a TODO, confirming the developers are aware it is absent.
- No special key, operator role, or network-level capability is required; a normal contract deployer or user who controls a contract can trigger this.

---

### Recommendation

Inside `execute_replace_class`, before writing the new class hash to `contract_state_changes`, assert that the hash is present in `contract_class_changes` (i.e., it has been declared in the current or a prior block):

```cairo
// Verify the replacement class is declared.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
with_attr error_message("replace_class: class hash is not declared.") {
    assert_not_zero(compiled_class_hash);
}
```

This mirrors the validation already performed implicitly in `execute_entry_point` and makes the invariant explicit and enforced at the point of mutation.

---

### Proof of Concept

1. Attacker deploys contract `V` (a vault holding user deposits).
2. `V` executes a `replace_class` syscall with `class_hash = 0xDEAD` (an undeclared felt).
3. `execute_replace_class` writes `StateEntry(class_hash=0xDEAD, ...)` into `contract_state_changes` with no validation.
4. A depositor submits a withdrawal transaction targeting `V`.
5. `execute_entry_point` calls `dict_read(key=0xDEAD)` → returns `0`.
6. `find_element(..., key=0)` fails: `0` is not in `compiled_class_facts_bundle`.
7. The OS proof for any block containing a call to `V` is invalid.
8. The sequencer permanently excludes calls to `V`; all depositor funds are frozen.

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
