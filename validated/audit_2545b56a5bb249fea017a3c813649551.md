### Title
Missing Internal Node Hash Verification in DataLayer Delta Blob Loading Allows Forged Tree Root Acceptance — (`crates/chia-datalayer/src/merkle/blob.rs`, `crates/chia-datalayer/src/merkle/deltas.rs`)

---

### Summary

`get_internal_terminal` and `collect_and_return_from_merkle_blob` read internal node hashes directly from the zstd-compressed blob without recomputing `internal_hash(left_child_hash, right_child_hash)`. A crafted blob file with a forged internal node hash field is accepted without error, and the forged hash propagates through `DeltaReader::collect_from_merkle_blob` → `create_merkle_blob_and_filter_unused_nodes` → `inner_build_blob_from_node_list` into the final `MerkleBlob`. No existing integrity check catches the discrepancy.

---

### Finding Description

**Root cause — `get_internal_terminal`:** [1](#0-0) 

For each internal node, the function reads `node.hash` directly from the blob and uses it as the map key and as the stored hash for the `DeltaReaderNode::Internal` entry. There is no call to `internal_hash(left_child_hash, right_child_hash)` to verify the stored value.

**Root cause — `collect_and_return_from_merkle_blob`:** [2](#0-1) 

Same pattern: `block.node.hash()` is read from the blob and used as the map key with no recomputation.

**Propagation — `collect_from_merkle_blob`:** [3](#0-2) 

The forged hash is inserted directly into `self.nodes` with no verification step.

**Propagation — `inner_build_blob_from_node_list`:** [4](#0-3) 

When reconstructing the `MerkleBlob`, the internal node is written with `hash: node_hash` — the map key that came from the blob — not from `internal_hash(left, right)`. The node is also written with `dirty: false`.

**Why `check_integrity()` does not catch this:**

`calculate_lazy_hashes` only recalculates hashes for nodes marked `dirty`: [5](#0-4) 

Since `inner_build_blob_from_node_list` sets `dirty: false`, `calculate_lazy_hashes` skips these nodes entirely. `check_just_integrity` only validates parent-child index relationships and cache consistency — it never compares a stored internal hash against `internal_hash(left_child_hash, right_child_hash)`: [6](#0-5) 

---

### Impact Explanation

A malicious DataLayer peer can supply a crafted zstd-compressed blob file where one or more internal node hash fields contain an arbitrary forged value (e.g., the legitimate blockchain root hash R) while the actual child content is different. The victim DataLayer node:

1. Calls `collect_from_merkle_blob(crafted_path, [TreeIndex(0)])` — forged hash enters `DeltaReader.nodes`.
2. Calls `create_merkle_blob_and_filter_unused_nodes(R, ...)` — the resulting `MerkleBlob` has root hash R but wrong tree content.
3. No integrity check rejects the result.

The node now holds a tree state whose root hash matches the committed on-chain value but whose actual key/value content is attacker-controlled. This enables forged inclusion/exclusion proofs against the accepted root, corrupting DataLayer state accepted as valid.

---

### Likelihood Explanation

- Being a DataLayer peer is a low barrier (anyone can run a node).
- The legitimate root hash R is public on-chain.
- Crafting a valid zstd-compressed blob with an arbitrary hash field is trivial — the blob format stores the hash as a raw 32-byte field in each node block.
- The attack is fully local-testable with no cryptographic preimage requirement.

---

### Recommendation

In `get_internal_terminal` and `collect_and_return_from_merkle_blob`, after reading an internal node, recompute the expected hash and reject the blob if it does not match:

```rust
let expected = internal_hash(
    index_to_hash.get(&node.left).unwrap(),
    index_to_hash.get(&node.right).unwrap(),
);
if expected != node.hash {
    return Err(Error::InternalHashMismatch(node.hash, expected));
}
```

Additionally, add a hash-content verification pass to `check_just_integrity` that recomputes `internal_hash(left_child_hash, right_child_hash)` for every internal node and compares it to the stored hash field.

---

### Proof of Concept

```rust
// 1. Build a legitimate two-leaf blob and save it.
let mut blob = MerkleBlob::new(Vec::new()).unwrap();
blob.insert(KeyId(1), ValueId(1), &leaf_hash_a, InsertLocation::AsRoot {}).unwrap();
blob.insert(KeyId(2), ValueId(2), &leaf_hash_b, InsertLocation::Auto {}).unwrap();
blob.calculate_lazy_hashes().unwrap();

// 2. Corrupt the root node's hash field in the raw bytes.
let forged_root = Hash(Bytes32::new([0xde; 32]));
// (patch blob.blob at the hash offset of the root block to forged_root bytes)

// 3. Write the crafted blob to disk (zstd-compressed).
// zstd_encode_to_path(&crafted_blob_bytes, &crafted_path);

// 4. Load via DeltaReader — no error.
let mut delta_reader = DeltaReader::new(HashMap::new(), HashMap::new()).unwrap();
delta_reader.collect_from_merkle_blob(&crafted_path, &vec![TreeIndex(0)]).unwrap();

// 5. Reconstruct — root hash is the forged value.
let result = delta_reader
    .create_merkle_blob_and_filter_unused_nodes(forged_root, &HashSet::new())
    .unwrap();
assert_eq!(result.get_root_hash().unwrap(), forged_root); // passes
result.check_integrity().unwrap();                         // also passes
```

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
