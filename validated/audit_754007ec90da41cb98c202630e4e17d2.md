### Title
`DeltaReader` Accepts Forged Internal Node Hashes ‚Äî No Hash Validity Check in `get_missing_hashes` or `build_blob_from_node_list` ‚Äî (`crates/chia-datalayer/src/merkle/deltas.rs`)

---

### Summary

`DeltaReader::get_missing_hashes` only checks **key presence** in the node map, never verifying that an internal node's stored hash equals `internal_hash(left_child_hash, right_child_hash)`. `MerkleBlob::inner_build_blob_from_node_list` then blindly copies the attacker-supplied hash into the blob with `dirty: false`, and `check_integrity` never recomputes internal hashes (it only recalculates `dirty` nodes). A malicious DataLayer peer can therefore supply a structurally complete but cryptographically invalid delta, causing the victim node to accept and persist a forged tree root.

---

### Finding Description

**`get_missing_hashes` ‚Äî presence-only check:** [1](#0-0) 

The function iterates over internal nodes and checks only whether `left` and `right` hashes exist as keys in `self.nodes`. It never asserts `internal_hash(left, right) == parent_hash`. If an attacker supplies `{H_root: Internal{left=H_L, right=H_R}, H_L: Leaf{...}, H_R: Leaf{...}}` where `H_root ‚âÝ internal_hash(H_L, H_R)`, the function returns an empty `HashSet` ‚Äî signalling a "complete" delta.

**`inner_build_blob_from_node_list` ‚Äî hash copied verbatim, `dirty: false`:** [2](#0-1) 

For an internal node, the blob is written with `hash: node_hash` (the attacker-supplied key) and `dirty: false`. The correct value `internal_hash(left_hash, right_hash)` is never computed here.

**`check_integrity` ‚Äî only recalculates `dirty` nodes:** [3](#0-2) 

`check_integrity` calls `calculate_lazy_hashes()` on a clone, but that function only touches nodes where `block.metadata.dirty == true`: [4](#0-3) 

Since `inner_build_blob_from_node_list` writes `dirty: false`, the forged internal hash is never corrected. `check_just_integrity` then validates only structural properties (parent-child pointers, key counts, free-index counts) ‚Äî it never verifies `stored_hash == internal_hash(left, right)`: [5](#0-4) 

**Attacker entry point ‚Äî blob files from untrusted peers:**

`collect_and_return_from_merkle_blob` reads the hash directly from the blob bytes without recomputing it from children: [6](#0-5) 

A malicious DataLayer peer sends a crafted zstd-compressed blob where the root internal node stores `H_root` (matching the on-chain committed root) but its children satisfy `H_root ‚âÝ internal_hash(H_L, H_R)`. The victim's node loads this via `collect_from_merkle_blob` / `collect_and_return_from_merkle_blobs`, and the forged hash propagates into `NodeHashToDeltaReaderNode` unchecked.

Additionally, the Python bindings expose `py_add_internal_nodes` and `py_add_leaf_nodes` which allow direct injection of arbitrary hash‚Üí(left, right) mappings with no validation: [7](#0-6) 

---

### Impact Explanation

A malicious DataLayer peer can cause a victim node to accept and persist a Merkle blob whose root hash matches the on-chain commitment but whose internal structure is cryptographically invalid. This enables:
- **Forged inclusion proofs**: the victim's `get_proof_of_inclusion` will produce a proof path using the fake internal hash, which will verify against `H_root` on the victim but not recompute correctly from the actual leaves.
- **Corrupted tree root**: subsequent delta operations build on a tree whose internal hashes are wrong, permanently corrupting the victim's DataLayer state.

This matches the allowed High impact: *"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."*

---

### Likelihood Explanation

The attack requires only a network-reachable DataLayer peer. No privileged access, leaked keys, or chain reorgs are needed. The attacker crafts a valid-length zstd blob with a root node whose stored hash matches the target's expected root but whose children are chosen freely. The victim's node processes this blob through the standard delta-sync path with no hash recomputation at any stage.

---

### Recommendation

1. In `get_missing_hashes` or in `DeltaReader::new` / `py_add_internal_nodes`, validate that for every internal node: `internal_hash(left, right) == parent_hash`. Reject the delta if any node fails this check.
2. In `inner_build_blob_from_node_list`, recompute the internal hash from the actual child hashes after recursing, and return an error if it does not match the supplied `node_hash`.
3. In `check_integrity`, add a pass that verifies `stored_hash == internal_hash(left_child_hash, right_child_hash)` for every internal node (after `calculate_lazy_hashes`).

---

### Proof of Concept

```rust
// Construct a DeltaReader where H_root ‚âÝ internal_hash(H_L, H_R)
let mut internal_nodes = InternalNodesMap::new();
let mut leaf_nodes = LeafNodesMap::new();

// H_root is arbitrary ‚Äî does NOT equal internal_hash(H_L, H_R)
internal_nodes.insert(H_root, (H_L, H_R));
leaf_nodes.insert(H_L, (KeyId(0), ValueId(1)));
leaf_nodes.insert(H_R, (KeyId(2), ValueId(3)));

let mut reader = DeltaReader::new(internal_nodes, leaf_nodes).unwrap();

// Step 1: get_missing_hashes returns empty ‚Äî tree appears complete
let missing = reader.get_missing_hashes(H_root);
assert!(missing.is_empty()); // passes ‚Äî only checks key presence

// Step 2: build blob ‚Äî internal node gets hash=H_root, dirty=false
let interested: HashSet<Hash> = reader.nodes.keys().copied().collect();
let blob = reader
    .create_merkle_blob_and_filter_unused_nodes(H_root, &interested)
    .unwrap();

// Step 3: check_integrity passes ‚Äî no hash recomputation for non-dirty nodes
blob.check_integrity().unwrap(); // passes

// Step 4: root hash in blob is H_root, not internal_hash(H_L, H_R)
let root_hash = blob.get_hash_at_index(TreeIndex(0)).unwrap().unwrap();
assert_eq!(root_hash, H_root);                          // forged hash accepted
assert_ne!(root_hash, internal_hash(&H_L, &H_R));      // cryptographic invariant violated
```

    pub fn get_missing_hashes(&self, root_hash: Hash) -> HashSet<Hash> {
        let mut missing_hashes: HashSet<Hash> = HashSet::new();

        for node in self.nodes.values() {
            let DeltaReaderNode::Internal { left, right } = node else {
                continue;
            };

            for hash in [left, right] {
                if !self.nodes.contains_key(hash) {
                    missing_hashes.insert(*hash);
                }
            }
        }

        if !self.nodes.contains_key(&root_hash) {
            missing_hashes.insert(root_hash);
        }

        missing_hashes
    }

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }

    pub fn calculate_lazy_hashes(&mut self) -> Result<(), Error> {
        // OPT: yeah, storing the whole set of blocks via collect is not great

            // OPT: obviously inefficient to re-get/deserialize these blocks inside

            let left_hash = self.get_hash(leaf.left)?;
            let right_hash = self.get_hash(leaf.right)?;
            block.update_hash(&left_hash, &right_hash);
            self.insert_entry_to_blob(index, &block)?;
        }
