### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Contract Freezing — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS accepts any arbitrary felt value as a new class hash without verifying that the hash corresponds to a declared contract class. A contract deployer can call `replace_class` with an undeclared hash, causing the contract's on-chain class hash to be set to an unresolvable value. Any subsequent call to that contract will fail irrecoverably at the OS level, permanently freezing all funds held within it. The OS produces a valid proof for this corrupted state transition, and the L1 verifier accepts it.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function processes the `replace_class` syscall:

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
``` [1](#0-0) 

The `class_hash` value from the syscall request is written directly into the contract state with **no Cairo-level assertion** that it exists in the declared class registry (`contract_class_changes` dict or the compiled class facts bundle). The TODO comment is the developer's own acknowledgment that this check is absent.

When a subsequent transaction calls the affected contract, `execute_entry_point` performs:

```cairo
let (compiled_class_hash: felt) = dict_read{dict_ptr=contract_class_changes}(
    key=execution_context.class_hash
);
...
let (compiled_class_fact: CompiledClassFact*) = find_element(
    array_ptr=compiled_class_facts_bundle.compiled_class_facts,
    elm_size=CompiledClassFact.SIZE,
    n_elms=compiled_class_facts_bundle.n_compiled_class_facts,
    key=compiled_class_hash,
);
``` [2](#0-1) 

If `execution_context.class_hash` is an undeclared value, `dict_read` returns 0 (the default for an uninitialized dict entry), and `find_element` with `key=0` panics (proof is unsatisfiable) unless a compiled class fact with hash 0 happens to exist. The sequencer therefore cannot include any future transaction that calls the contract, making the contract permanently uncallable and all funds within it permanently frozen. The corrupted state (undeclared class hash) is committed to the L1 state root via a valid proof.

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Once a contract's class hash is set to an undeclared value via `replace_class`, the state root committed to L1 reflects this corrupted entry. No future block can include a transaction that calls the contract (the OS proof would be unsatisfiable). Any ERC-20 balances, ETH, or other assets held in the contract's storage are permanently inaccessible. The L1 Starknet core contract accepts the state root as valid because the proof is sound with respect to the (incomplete) OS constraints.

---

### Likelihood Explanation

**Medium.**

Any contract deployer can write a contract whose `__execute__` (or any callable entry point) issues `replace_class(arbitrary_felt)`. Because the OS does not enforce the declared-class invariant, and the TODO comment confirms the check is intentionally deferred, neither the OS proof nor the on-chain verifier rejects the resulting state. A user who mistakenly passes a wrong hash, or a malicious actor who deliberately targets their own contract holding third-party deposits, can trigger this path with a single transaction.

---

### Recommendation

Add a Cairo-level assertion in `execute_replace_class` that the requested `class_hash` is present in `contract_class_changes` (i.e., it was declared in the current or a prior block). Concretely, perform a `dict_read` on `contract_class_changes` for `class_hash` and assert the returned compiled class hash is non-zero before writing the new `StateEntry`. This mirrors the check already performed implicitly in `execute_entry_point` but must be enforced at replacement time to prevent the corrupted state from ever reaching the committed root.

---

### Proof of Concept

1. **Deploy** a contract whose `__execute__` function calls the `replace_class` syscall with `class_hash = 0xdeadbeef` (any value not present in the declared class registry).

2. **Submit** an invoke transaction calling that contract. The OS executes `execute_replace_class`:
   - `class_hash = 0xdeadbeef` is read from the syscall request.
   - No Cairo assertion checks that `0xdeadbeef` is declared.
   - `dict_update` writes `StateEntry(class_hash=0xdeadbeef, ...)` for the contract address.
   - The revert log records the old class hash (for potential revert), but the transaction is not reverted.

3. **Block is proven and submitted to L1.** The state root now encodes the contract with `class_hash = 0xdeadbeef`.

4. **Attempt to call the contract** in any subsequent block. `execute_entry_point` calls `dict_read(contract_class_changes, key=0xdeadbeef)` → returns 0. `find_element(..., key=0)` fails (no compiled class fact with hash 0). The proof is unsatisfiable; the sequencer cannot include this transaction.

5. **Result:** The contract is permanently uncallable. All funds (ERC-20 balances, ETH) stored in the contract are permanently frozen, with no recovery path. [3](#0-2) [4](#0-3)

// Replaces the class.
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
