### Title
Stale `feemarketParams` Captured at Handler-Creation Time in `NewDynamicFeeChecker` Allows Cosmos Transactions to Bypass the Current EIP-1559 Base Fee - (File: `ante/evm/fee_checker.go`, `evmd/ante/handler_options.go`)

---

### Summary

`newCosmosAnteHandler` captures `feemarketParams` once at handler-creation time and passes a pointer to that snapshot into `NewDynamicFeeChecker`. Because `feemarket.BeginBlock` updates `params.BaseFee` in the KV store every block, the pointer held by the fee-checker closure permanently lags behind the live base fee. Any Cosmos SDK transaction routed through the Cosmos ante handler (governance, staking, IBC, etc.) is validated against the genesis-era base fee for the entire lifetime of the node, not the current one.

---

### Finding Description

**Root cause — `evmd/ante/handler_options.go`, `newCosmosAnteHandler`:**

```go
func newCosmosAnteHandler(ctx sdk.Context, options HandlerOptions, ...) sdk.AnteHandler {
    evmParams      := options.EvmKeeper.GetParams(ctx)       // snapshot at creation
    feemarketParams := options.FeeMarketKeeper.GetParams(ctx) // snapshot at creation
    ...
    txFeeChecker = evm.NewDynamicFeeChecker(ethCfg, &evmParams, &feemarketParams)
    ...
}
```

`feemarketParams` is a local value-type variable. Its address is taken and passed to `NewDynamicFeeChecker`, which stores it in a closure. The variable escapes to the heap and remains valid, but its contents are frozen at the moment `newCosmosAnteHandler` was called (app startup / `NewAnteHandler`). It is never refreshed.

**Stale read in the fee checker — `ante/evm/fee_checker.go`, `NewDynamicFeeChecker`:**

```go
func NewDynamicFeeChecker(ethCfg *params.ChainConfig,
    evmParams *types.Params,
    feemarketParams *feemarkettypes.Params,   // ← stale pointer
) authante.TxFeeChecker {
    return func(ctx sdk.Context, tx sdk.Tx) (sdk.Coins, int64, error) {
        ...
        baseFee := types.GetBaseFee(ctx.BlockHeight(), ethCfg, feemarketParams) // reads stale BaseFee
        ...
        if feeCap.LT(baseFeeInt) {
            return nil, 0, errorsmod.Wrapf(errortypes.ErrInsufficientFee, ...)
        }
        effectivePrice := sdkmath.NewIntFromBigInt(
            types.EffectiveGasPrice(baseFeeInt.BigInt(), feeCap.BigInt(), maxPriorityPrice.BigInt()))
        effectiveFee := sdk.Coins{{Denom: denom, Amount: effectivePrice.Mul(...)}}
        return effectiveFee, priority, nil
    }
}
```

`types.GetBaseFee` derives the base fee from `feemarketParams.BaseFee` — the frozen genesis value — not from the live KV store.

**Contrast with the Ethereum ante handler — `evmd/ante/handler_options.go`, `newEthAnteHandler`:**

```go
func newEthAnteHandler(options HandlerOptions) sdk.AnteHandler {
    return func(ctx sdk.Context, tx sdk.Tx, simulate bool) (sdk.Context, error) {
        blockCfg, err := options.EvmKeeper.EVMBlockConfig(ctx, options.EvmKeeper.ChainID())
        ...
        baseFee := blockCfg.BaseFee   // ← fresh read from KV store every call
```

`EVMBlockConfig` reads `feemarketParams.GetBaseFee()` from the KV store on every invocation (or from the per-block object-store cache that is populated after `feemarket.BeginBlock` has already written the new value). The Cosmos path has no equivalent refresh.

**Live base-fee update — `x/feemarket/keeper/abci.go`, `BeginBlock`:**

```go
func (k *Keeper) BeginBlock(ctx sdk.Context) error {
    baseFee := k.CalculateBaseFee(ctx)
    if baseFee == nil { return nil }
    k.SetBaseFee(ctx, baseFee)   // writes new BaseFee into params KV store every block
    ...
}
```

Every block the KV-store value diverges further from the frozen pointer held by `NewDynamicFeeChecker`.

---

### Impact Explanation

When `DynamicFeeChecker` is enabled (the intended production configuration for EIP-1559 fee-market chains), every Cosmos SDK transaction is fee-validated against the genesis `BaseFee` rather than the current one. After a period of sustained high gas usage, the live base fee can be orders of magnitude higher than the genesis value. An attacker can:

1. Submit Cosmos transactions (governance votes, staking delegations, IBC transfers, authz executions, etc.) with fees computed from the genesis base fee.
2. The `DeductFeeDecorator` deducts only `effectivePrice_genesis × gas` from the sender.
3. The fee collector receives far less than `effectivePrice_current × gas`.
4. Validators' fee revenue is systematically under-collected for all Cosmos-path transactions.

This satisfies the allowed High impact: **"fee market, ante handler… bug that permits invalid transactions to commit or valid user funds/fees to be mis-accounted."**

---

### Likelihood Explanation

- `DynamicFeeChecker: true` is the recommended production setting for any Ethermint chain running EIP-1559.
- The base fee diverges from genesis on every block where gas usage differs from the target; on a live chain this is essentially always.
- No special privilege is required: any unprivileged user submitting a standard Cosmos transaction (staking, governance, IBC) can exploit this by simply setting their fee to `genesis_base_fee × gas_limit`.
- The attacker needs only to know the genesis `BaseFee` (a public parameter) and the current block height.

---

### Recommendation

Replace the captured-pointer pattern in `NewDynamicFeeChecker` with a live read from the keeper on every invocation:

```go
// ante/evm/fee_checker.go
func NewDynamicFeeChecker(
    evmKeeper EVMKeeperForFeeChecker,   // add keeper interface
    ethCfg *params.ChainConfig,
    evmParams *types.Params,
) authante.TxFeeChecker {
    return func(ctx sdk.Context, tx sdk.Tx) (sdk.Coins, int64, error) {
        feemarketParams := evmKeeper.GetFeeMarketParams(ctx)  // fresh read
        baseFee := types.GetBaseFee(ctx.BlockHeight(), ethCfg, &feemarketParams)
        ...
    }
}
```

Alternatively, mirror the Ethereum path: call `EVMBlockConfig(ctx, chainID)` at the top of the returned closure, which already performs a correctly-ordered fresh read (or uses the per-block object-store cache that is always populated after `feemarket.BeginBlock`).

---

### Proof of Concept

1. Start an Ethermint chain with `DynamicFeeChecker: true` and `DefaultBaseFee = 1_000_000_000` (1 gwei).
2. Flood the chain with EVM transactions for N blocks so that `feemarket.BeginBlock` raises `params.BaseFee` to, say, `10_000_000_000` (10 gwei).
3. Construct a Cosmos governance `MsgVote` with `fee = 1_000_000_000 × gasLimit` (genesis base fee).
4. Broadcast the transaction. The `NewDynamicFeeChecker` closure reads `feemarketParams.BaseFee = 1_000_000_000` (frozen at genesis), computes `effectiveFee` at 1 gwei, passes the check, and `DeductFeeDecorator` deducts only 1 gwei × gas from the sender — 10× less than the current market rate.

func NewDynamicFeeChecker(ethCfg *params.ChainConfig, evmParams *types.Params, feemarketParams *feemarkettypes.Params) authante.TxFeeChecker {
	return func(ctx sdk.Context, tx sdk.Tx) (sdk.Coins, int64, error) {
		feeTx, ok := tx.(sdk.FeeTx)
		if !ok {
			return nil, 0, fmt.Errorf("tx must be a FeeTx")
		}

		if ctx.BlockHeight() == 0 {
			// genesis transactions: fallback to min-gas-price logic
			return checkTxFeeWithValidatorMinGasPrices(ctx, feeTx)
		}

		denom := evmParams.EvmDenom

		baseFee := types.GetBaseFee(ctx.BlockHeight(), ethCfg, feemarketParams)
		if baseFee == nil {
			// london hardfork is not enabled: fallback to min-gas-prices logic
			return checkTxFeeWithValidatorMinGasPrices(ctx, feeTx)
		}

		// default to `MaxInt64` when there's no extension option.
		maxPriorityPrice := sdkmath.NewInt(math.MaxInt64)

		// get the priority tip cap from the extension option.
		if hasExtOptsTx, ok := tx.(authante.HasExtensionOptionsTx); ok {
			for _, opt := range hasExtOptsTx.GetExtensionOptions() {
				if extOpt, ok := opt.GetCachedValue().(*ethermint.ExtensionOptionDynamicFeeTx); ok {
					maxPriorityPrice = extOpt.MaxPriorityPrice
					break
				}
			}
		}

		if maxPriorityPrice.Sign() == -1 {
			return nil, 0, errorsmod.Wrapf(errortypes.ErrInsufficientFee, "priority fee is negative")
		}

		gas := feeTx.GetGas()
		feeCoins := feeTx.GetFee()
		fee := feeCoins.AmountOf(denom)

		feeCap := fee.Quo(sdkmath.NewIntFromUint64(gas))
		baseFeeInt := sdkmath.NewIntFromBigInt(baseFee)

		if feeCap.LT(baseFeeInt) {
			return nil, 0, errorsmod.Wrapf(errortypes.ErrInsufficientFee, "insufficient gas prices; got: %s required: %s", feeCap, baseFeeInt)
		}

		// calculate the effective gas price using the EIP-1559 logic.
		effectivePrice := sdkmath.NewIntFromBigInt(types.EffectiveGasPrice(baseFeeInt.BigInt(), feeCap.BigInt(), maxPriorityPrice.BigInt()))

func (k *Keeper) BeginBlock(ctx sdk.Context) error {
	baseFee := k.CalculateBaseFee(ctx)

	// return immediately if base fee is nil
	if baseFee == nil {
		return nil
	}

	k.SetBaseFee(ctx, baseFee)

	defer func() {
		telemetry.SetGauge(float32(baseFee.Int64()), "feemarket", "base_fee") //nolint:staticcheck
	}()

	// Store current base fee in event
	ctx.EventManager().EmitEvents(sdk.Events{
		sdk.NewEvent(
			types.EventTypeFeeMarket,
			sdk.NewAttribute(types.AttributeKeyBaseFee, baseFee.String()),
		),
	})
	return nil

func (k *Keeper) EVMBlockConfig(ctx sdk.Context, chainID *big.Int) (*EVMBlockConfig, error) {
	objStore := ctx.ObjectStore(k.objectKey)
	v := objStore.Get(types.KeyPrefixObjectParams)
	if v != nil {
		return v.(*EVMBlockConfig), nil
	}

	params := k.GetParams(ctx)
	ethCfg := params.ChainConfig.EthereumConfig(chainID)

	feemarketParams := k.feeMarketKeeper.GetParams(ctx)

	// get the coinbase address from the block proposer
	coinbase, err := k.GetCoinbaseAddress(ctx)
	if err != nil {
		return nil, errorsmod.Wrap(err, "failed to obtain coinbase address")
	}

	var baseFee *big.Int
	if types.IsLondon(ethCfg, ctx.BlockHeight()) {
		baseFee = feemarketParams.GetBaseFee()
		// should not be nil if london hardfork enabled
		if baseFee == nil {
			baseFee = new(big.Int)
		}
	}
	time := ctx.BlockHeader().Time
	var blockTime uint64
	if !time.IsZero() {
		blockTime, err = ethermint.SafeUint64(time.Unix())
		if err != nil {
			return nil, err
		}
	}
	blockNumber := big.NewInt(ctx.BlockHeight())
	rules := ethCfg.Rules(blockNumber, ethCfg.MergeNetsplitBlock != nil, blockTime)

	// Build the default precompile set once per block.
	contracts := make(map[common.Address]vm.PrecompiledContract)
	for addr, c := range vm.DefaultPrecompiles(rules) {
		contracts[addr] = c
	}

	var zero common.Hash
	cfg := &EVMBlockConfig{
		Params:             params,
		FeeMarketParams:    feemarketParams,
		ChainConfig:        ethCfg,
		CoinBase:           coinbase,
		BaseFee:            baseFee,
		Difficulty:         new(big.Int),
		Random:             &zero,
		BlobBaseFee:        new(big.Int),
		BlockNumber:        blockNumber,
		BlockTime:          blockTime,
		Rules:              rules,
		DefaultPrecompiles: contracts,
	}
	objStore.Set(types.KeyPrefixObjectParams, cfg)
	return cfg, nil
}
