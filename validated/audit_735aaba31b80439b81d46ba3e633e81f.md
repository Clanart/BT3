### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Freezing of Contract Funds — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall implementation in the StarkNet OS accepts any arbitrary felt value as the new class hash without verifying that the hash corresponds to a previously declared (and therefore executable) contract class. An unprivileged contract can call this syscall with an undeclared class hash, permanently replacing its own class hash with one for which no compiled class exists. Any funds held by that contract become permanently inaccessible, satisfying the Critical "Permanent freezing of funds" impact.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function processes the `ReplaceClassRequest` and updates `contract_state_changes` with the caller-supplied `class_hash` without any on-chain Cairo-level assertion that the hash is present in the set of declared classes:

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
```

The developer-inserted TODO comment explicitly acknowledges the missing guard. No Cairo `assert` or dict-lookup against `contract_class_changes` is performed before the state update is committed. [1](#0-0) 

Compare this with the `deploy_contract` path, which enforces `state_entry.class_hash = UNINITIALIZED_CLASS_HASH` before writing a new class hash — a guard that `execute_replace_class` entirely omits: [2](#0-1) 

The OS loads compiled class facts once at startup and validates them post-execution via `validate_compiled_class_facts_post_execution`. If a contract's class hash is set to a value that was never declared, no compiled class entry exists for it. Any future call to that contract will fail at the OS level because the prover cannot supply a valid compiled class fact for the unknown hash. The sequencer will permanently exclude such calls, leaving the contract's storage and balance unreachable. [3](#0-2) 

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Once `replace_class` commits an undeclared class hash to the global state, the contract is bricked:

- Every subsequent call to the contract requires the OS to resolve the class hash to a compiled class. No such compiled class exists.
- The contract cannot self-heal: calling `replace_class` again requires the contract to be callable, which it is not.
- All ERC-20 balances, NFTs, or protocol-controlled value stored in the contract's storage become permanently inaccessible.

---

### Likelihood Explanation

**Medium.**

The `replace_class` syscall is a standard, permissionless syscall available to every Sierra contract. No privileged role is required. The attack requires only:

1. Deploying a contract (permissionless on StarkNet).
2. Invoking `replace_class` with any felt value that is not a declared class hash.

A malicious contract can do this in a single transaction. A legitimate contract that exposes an upgradeable `replace_class` path with insufficient input validation on the caller side is equally vulnerable if an attacker can influence the class hash argument.

---

### Recommendation

Before committing the new class hash to `contract_state_changes`, assert that the hash exists in `contract_class_changes` (i.e., it was declared in the current or a prior block). Concretely, perform a `dict_read` on `contract_class_changes` for `class_hash` and assert the returned compiled class hash is non-zero:

```cairo
// Verify the new class hash is declared.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);
```

This mirrors the guard already present in `execute_declare_transaction`, which uses `prev_value=0` to enforce that a class may only be declared once and must have a non-zero compiled class hash: [4](#0-3) 

---

### Proof of Concept

1. **Attacker deploys** a contract `VictimVault` that holds user funds and exposes a public `upgrade(new_class_hash)` entry point that calls the `replace_class` syscall.
2. **Attacker calls** `upgrade(0xdeadbeef)` where `0xdeadbeef` is an arbitrary felt that has never been declared as a class.
3. **OS execution** of the transaction reaches `execute_replace_class`. The function reads `request.class_hash = 0xdeadbeef`, skips any declaration check (the TODO guard is absent), and writes:
   ```
   contract_state_changes[VictimVault] = StateEntry(class_hash=0xdeadbeef, ...)
   ```
4. **Transaction succeeds**; the state diff is committed. `VictimVault`'s class hash is now `0xdeadbeef`.
5. **Any future call** to `VictimVault` requires the OS to supply a compiled class for `0xdeadbeef`. No such class exists in the compiled class facts bundle. The prover cannot construct a valid proof for any block containing a call to `VictimVault`.
6. **Result**: All funds inside `VictimVault` are permanently frozen. The sequencer will never include calls to `VictimVault` again. [5](#0-4)

// Replaces the class.
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
