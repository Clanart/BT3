### Title
Missing Declared Class Validation in `execute_replace_class` Allows Permanent Freezing of Contract Funds — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler accepts an arbitrary `class_hash` from the caller and writes it directly into the contract state without verifying that the hash corresponds to a previously declared class. This is the direct StarkNet analog of the NFT report's pattern: one field of a two-field validity requirement (existence + declaration) is checked while the other (declaration) is silently skipped. A contract can replace its own class hash with any undeclared value, making itself permanently unexecutable and freezing all funds it holds.

---

### Finding Description

In `execute_replace_class` the OS reads `request.class_hash` and immediately writes it into `contract_state_changes` with no check that the hash exists in `contract_class_changes` (the map of declared class hashes):

```cairo
let class_hash = request.class_hash;
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}
tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);
dict_update{dict_ptr=contract_state_changes}(...);
``` [1](#0-0) 

The TODO comment at line 898 is an explicit acknowledgment by the authors that this check is absent.

When any subsequent call targets the contract, `execute_entry_point` performs:

```cairo
let (compiled_class_hash: felt) = dict_read{dict_ptr=contract_class_changes}(
    key=execution_context.class_hash   // ← the attacker-supplied hash
);
...
let (compiled_class_fact: CompiledClassFact*) = find_element(
    array_ptr=compiled_class_facts_bundle.compiled_class_facts,
    elm_size=CompiledClassFact.SIZE,
    n_elms=compiled_class_facts_bundle.n_compiled_class_facts,
    key=compiled_class_hash,           // ← 0 (default) for undeclared hash
);
``` [2](#0-1) 

Because the attacker-supplied hash was never declared, `dict_read` returns the default value `0`. `find_element` then searches for a compiled class with hash `0` in `compiled_class_facts_bundle`. If no such entry exists (the normal case), `find_element` raises an assertion failure, making the block unprovable and the contract permanently unexecutable.

For comparison, a legitimately declared class is registered via:

```cairo
dict_update{dict_ptr=contract_class_changes}(
    key=[class_hash_ptr], prev_value=0, new_value=compiled_class_hash
);
``` [3](#0-2) 

`execute_replace_class` performs no equivalent lookup before accepting the new hash.

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Any contract that holds token balances (e.g., a multisig, a vault, a DeFi pool) can be targeted. Once `replace_class` is called with an undeclared hash:

1. The contract's `class_hash` in `contract_state_changes` is set to the invalid value.
2. Every future call to the contract reaches `execute_entry_point`, which fails at `find_element`.
3. The contract becomes permanently unexecutable; no withdrawal, transfer, or recovery function can ever run.
4. All funds stored in the contract's storage are frozen forever with no recovery path.

---

### Likelihood Explanation

**High.** The `replace_class` syscall is callable by any contract from within its own execution context — no privileged role, no special key, no operator cooperation is required. The attacker only needs to:

- Deploy (or control) a contract that holds funds.
- Issue a single transaction that calls `replace_class` with an arbitrary undeclared felt value.

The OS accepts the syscall unconditionally. The damage is irreversible.

---

### Recommendation

Before writing the new class hash into `contract_state_changes`, verify that it is present in `contract_class_changes` (i.e., has been declared). The check should mirror the lookup already performed in `execute_entry_point`:

```cairo
// Proposed fix inside execute_replace_class:
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
with_attr error_message("Class hash not declared.") {
    assert_not_zero(compiled_class_hash);
}
```

This ensures that only hashes that have passed the full `execute_declare_transaction` flow — including `finalize_class_hash` verification and `compiled_class_hash` registration — can be used as replacement targets. [4](#0-3) 

---

### Proof of Concept

1. **Deploy** a contract `Vault` that holds user funds and exposes a `withdraw` entry point.
2. **Attacker** (who controls `Vault`, e.g., a malicious owner or the contract itself via a self-call) issues a transaction that calls the `replace_class` syscall with `class_hash = 0xdeadbeef` — a felt that has never been passed through `execute_declare_transaction`.
3. The OS executes `execute_replace_class`: no declared-class check is performed; `contract_state_changes[Vault_address].class_hash` is set to `0xdeadbeef`.
4. In any subsequent block, a user calls `Vault.withdraw(...)`. The OS reaches `execute_entry_point`:
   - `dict_read(contract_class_changes, key=0xdeadbeef)` → returns `0` (undeclared).
   - `find_element(..., key=0)` → assertion failure; block cannot be proven.
5. `Vault` is permanently unexecutable. All funds are frozen with no recovery path.

// Replaces the class.
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
