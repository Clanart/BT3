### Title
Missing Class Existence Validation in `execute_replace_class` Allows Permanent Freezal of Contract Funds - (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall implementation in the StarkNet OS does not verify that the replacement class hash corresponds to a declared contract class. This is explicitly acknowledged with a `TODO` comment in the code. Any unprivileged contract can call `replace_class` with an arbitrary, undeclared class hash, permanently bricking itself and freezing any funds it holds.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function processes the `REPLACE_CLASS` syscall. After deducting gas, it reads the requested class hash directly from the syscall request and writes it into the contract state dictionary without any check that the hash corresponds to a class that has been declared on-chain:

```cairo
func execute_replace_class{...}(contract_address: felt) {
    ...
    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );
    ...
}
```

The critical validation — confirming the new class hash exists in `contract_class_changes` or the global compiled class facts — is entirely absent. The TODO comment at line 898 explicitly acknowledges this gap.

This is the direct analog to the TraitForge finding: a critical control mechanism (`_pause`/`_unpause` there; class-existence enforcement here) is defined conceptually and acknowledged as necessary, but the enforcement function is never called/wired into the execution path, leaving the system in an insecure state. [1](#0-0) 

---

### Impact Explanation

**Impact: Critical — Permanent freezing of funds.**

Once a contract's class hash is replaced with an undeclared hash:

1. The contract's `StateEntry.class_hash` is permanently set to a value that has no corresponding compiled class in the OS's `compiled_class_facts` or `deprecated_compiled_class_facts` bundles.
2. Any subsequent call to that contract (via `call_contract`, `library_call`, or direct invocation) will attempt to look up the class for execution. Since no compiled class exists for that hash, execution cannot proceed and the call will always fail.
3. Any ERC-20 tokens, ETH, STRK, or other assets held in that contract's storage become permanently inaccessible — the contract can never execute a transfer or withdrawal again.
4. The state commitment will include the corrupted class hash, making the damage permanent and provable on-chain. [2](#0-1) 

---

### Likelihood Explanation

**Likelihood: High.**

- The `replace_class` syscall is a standard, publicly accessible syscall available to any Sierra contract.
- No privileged role, operator access, or leaked key is required. Any deployed contract can issue this syscall.
- The attacker-controlled entry path is: deploy a contract → call `replace_class` with an arbitrary felt value as the class hash → the OS accepts it unconditionally.
- The attack can be self-inflicted (attacker bricking their own contract after receiving funds from victims) or can target any contract that exposes a path to `replace_class` in its logic.
- The missing check is explicitly flagged in the source with a `TODO`, confirming the developers are aware it is absent. [3](#0-2) 

---

### Recommendation

After deducting gas and before writing the new state entry, verify that the requested class hash is present in the `contract_class_changes` dictionary (i.e., it was declared in the current block) or in the global `compiled_class_facts` / `deprecated_compiled_class_facts` bundles carried in `OsGlobalContext`. Concretely:

```cairo
// After gas reduction succeeds:
let class_hash = request.class_hash;

// Enforce: the class must have been declared (present in contract_class_changes
// with a non-zero compiled class hash, or in the global compiled class facts).
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);  // Reverts if class is undeclared.
```

This mirrors the enforcement already applied in `execute_declare_transaction` where `assert_not_zero(compiled_class_hash)` is called before committing the class declaration. [4](#0-3) 

---

### Proof of Concept

1. **Attacker deploys** a contract `C` that holds user funds (e.g., acts as a vault). Users deposit assets into `C`.
2. **Attacker triggers** `replace_class` from within `C` (or via a malicious entry point), passing `class_hash = 0xdeadbeef` (any felt not corresponding to a declared class).
3. **OS execution** in `execute_replace_class` deducts gas, reads `request.class_hash = 0xdeadbeef`, skips the missing existence check, and writes:
   ```
   new StateEntry(class_hash=0xdeadbeef, storage_ptr=..., nonce=...)
   ```
   into `contract_state_changes` for address `C`.
4. **State update** in `state_update` commits this corrupted entry into the Merkle tree.
5. **Any subsequent call** to `C` (e.g., `withdraw`) attempts to resolve class `0xdeadbeef` from compiled class facts — it does not exist — execution fails unconditionally.
6. **All funds** in `C`'s storage are permanently frozen with no recovery path. [2](#0-1) [5](#0-4)

func execute_replace_class{
    range_check_ptr,

    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

        key=contract_address,

    );

    let revert_log = &revert_log[1];

    return ();
}
