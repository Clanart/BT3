### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Contract Bricking and Fund Freezing — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in `syscall_impls.cairo` accepts any arbitrary felt value as the new class hash without verifying that the hash corresponds to a previously declared class. A developer-acknowledged TODO comment at line 898 explicitly marks this check as missing. Any contract can call `replace_class` with an undeclared class hash, causing the OS to commit an invalid state transition that permanently bricks the contract and freezes any funds it holds.

---

### Finding Description

In `execute_declare_transaction` (`transaction_impls.cairo`), the OS correctly enforces a "declare once" invariant using `prev_value=0` in `dict_update`:

```cairo
// Note that prev_value=0 enforces that a class may be declared only once.
assert_not_zero(compiled_class_hash);
dict_update{dict_ptr=contract_class_changes}(
    key=[class_hash_ptr], prev_value=0, new_value=compiled_class_hash
);
``` [1](#0-0) 

This correctly prevents re-declaration of an already-declared class. However, the `execute_replace_class` function, which updates a deployed contract's class hash in the contract state tree, performs **no symmetric check** that the incoming class hash is a member of the declared-class set:

```cairo
let class_hash = request.class_hash;

// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
``` [2](#0-1) 

The `class_hash` field is taken directly from the caller-supplied syscall request with no membership check against `contract_class_changes`. The OS then writes this unchecked value into the contract's `StateEntry` and commits it to the Patricia state tree via `compute_contract_state_commitment`. [3](#0-2) 

The `REPLACE_CLASS_SELECTOR` branch in `execute_syscalls` dispatches directly to `execute_replace_class` with no pre-flight validation: [4](#0-3) 

---

### Impact Explanation

Once the OS commits a `StateEntry` whose `class_hash` points to a class that was never declared, every subsequent call to that contract address will fail at class-lookup time — there is no Sierra/CASM bytecode registered for that hash. The contract becomes permanently non-callable. Any ERC-20 balances, ETH, or other assets held by the contract are irreversibly frozen because no entry point can be invoked to transfer them out.

This satisfies the **Critical — Permanent freezing of funds** impact category.

---

### Likelihood Explanation

The `replace_class` syscall is callable by any contract from within its own execution context — no privileged role is required. The attacker-controlled entry path is:

1. An unprivileged user deploys or interacts with any contract that exposes a path to `replace_class`.
2. The attacker supplies an arbitrary felt (e.g., `0xdeadbeef`) as the new class hash.
3. The OS processes the syscall, writes the invalid hash into the contract state, and includes it in the block's state diff.
4. The state diff is committed to the Patricia tree; the contract is permanently bricked.

The TODO comment at line 898 confirms the development team is aware the check is absent. Any contract that holds user funds and exposes a `replace_class` code path — whether through a reentrancy vector, an access-control bug, or an upgrade mechanism — is vulnerable to permanent fund loss with no recovery path.

---

### Recommendation

Inside `execute_replace_class`, before writing the new `StateEntry`, verify that `class_hash` exists in `contract_class_changes` with a non-zero compiled class hash (i.e., it was declared in this block or in a prior block reflected in the Patricia tree). Concretely:

```cairo
// Verify the new class hash is declared.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);
```

If the class was declared in a previous block (not present in the current block's `contract_class_changes`), the hint layer must supply a Merkle proof of membership in the class Patricia tree, analogous to how `execute_get_class_hash_at` reads contract state.

---

### Proof of Concept

1. Declare a legitimate class `C` and deploy contract `V` (holding 1000 ETH) using class `C`.
2. `V` exposes an `upgrade(new_hash: felt)` entry point that calls `replace_class(new_hash)`.
3. Attacker calls `V.upgrade(0xdeadbeef)` where `0xdeadbeef` was never declared.
4. The OS executes `execute_replace_class`:
   - `class_hash = 0xdeadbeef` (from request, attacker-controlled).
   - No membership check is performed (TODO at line 898).
   - `dict_update` writes `StateEntry { class_hash: 0xdeadbeef, ... }` for `V`'s address.
5. The block is proven and the state root is updated; `V`'s leaf in the Patricia tree now encodes `class_hash = 0xdeadbeef`.
6. Any subsequent `call_contract` to `V` fails: the OS reads `class_hash = 0xdeadbeef`, finds no corresponding Sierra/CASM, and the call reverts.
7. The 1000 ETH inside `V` is permanently frozen — no entry point can be reached to withdraw it. [5](#0-4)

// Replaces the class.
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
