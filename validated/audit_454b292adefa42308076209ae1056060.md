### Title
OS `execute_replace_class` Omits Declared-Class Check, Diverging from Blockifier Revert — (`File: crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

### Summary

The Starknet OS Cairo program's `execute_replace_class` syscall handler does not verify that the target class hash is actually declared on-chain. The blockifier (Rust execution engine) enforces this check and reverts the transaction when the class is undeclared. This creates a provable divergence: a transaction that the blockifier marks as reverted will be accepted as successful by the OS, producing a wrong post-block state root in the SNOS proof.

### Finding Description

The blockifier's `replace_class` implementation in `syscall_base.rs` enforces two guards before applying the class replacement:

```rust
pub fn replace_class(&mut self, class_hash: ClassHash) -> SyscallResult<()> {
    // Ensure the class is declared (by reading it), and of type V1.
    let compiled_class = self.state.get_compiled_class(class_hash)?;
    if !is_cairo1(&compiled_class) {
        return Err(SyscallExecutionError::ForbiddenClassReplacement { class_hash });
    }
    self.state.set_class_hash_at(self.call.storage_address, class_hash)?;
    Ok(())
}

The deprecated OS path in `deprecated_execute_syscalls.cairo` has the same omission: [4](#0-3) 

The divergence is structural: the blockifier returns an error from `get_compiled_class` when the class hash is absent, which propagates as a revert. The OS skips that lookup entirely and unconditionally writes the new `StateEntry` into `contract_state_changes`. When the OS later computes the contracts trie root and feeds it into `calculate_block_hash`, the root reflects the undeclared class hash as the live class for that contract address — a value the blockifier never committed. [5](#0-4) 

### Impact Explanation

The OS is the authoritative execution engine for SNOS proofs. When the OS re-executes a block containing a `replace_class` call targeting an undeclared class hash, it does not revert the transaction. The resulting contracts-trie leaf for the affected address carries the undeclared class hash as its `class_hash` field. This corrupted leaf propagates through `commit_state_diff` into the Patricia trie, producing a global root that differs from the one the blockifier computed. That wrong global root is then chained into `calculate_block_hash` and ultimately committed to L1 as the authoritative post-block state root. Any subsequent storage proof, class-hash lookup, or RPC call against that contract address will return the wrong class hash.

**Impact category:** Critical — Wrong state root / wrong class hash value from blockifier/syscall/execution logic for accepted input.

### Likelihood Explanation

Any unprivileged user can deploy a Cairo 1 contract that calls `replace_class` with an arbitrary felt value that is not a declared class hash. The blockifier reverts the inner call; the OS does not. No special privilege, malicious peer, or provider cooperation is required. The TODO comment confirms the check is intentionally deferred, not accidentally omitted, meaning the window is open until the deadline noted in the comment (1/1/2026).

### Recommendation

Add the declared-class check to `execute_replace_class` in `syscall_impls.cairo` before writing the new `StateEntry`. The check should mirror the blockifier: read the class from the class trie (or the in-flight `contract_class_changes` dict) and abort the syscall if the class hash is absent or is a Cairo 0 class. Remove the TODO comment once the check is in place. The deprecated path in `deprecated_execute_syscalls.cairo` should receive the same fix.

### Proof of Concept

1. Declare a Cairo 1 contract `Attacker` whose `__execute__` calls `replace_class(0xdeadbeef)` — an arbitrary felt that is not a declared class hash.
2. Deploy `Attacker` at address `A`.
3. Submit an invoke transaction from `A`.
4. **Blockifier path:** `syscall_base::replace_class` calls `get_compiled_class(0xdeadbeef)`, receives `StateError::UndeclaredClassHash`, returns `SyscallExecutionError`, transaction is marked reverted. The contracts trie leaf for `A` retains the original class hash. The global root `R_blockifier` is computed and stored.
5. **OS path:** `execute_replace_class` skips the declared-class lookup (line 903 TODO), writes `StateEntry { class_hash: 0xdeadbeef, ... }` into `contract_state_changes`, transaction is not reverted. The contracts trie leaf for `A` now holds `0xdeadbeef`. The global root `R_os ≠ R_blockifier` is computed.
6. `calculate_block_hash` chains `R_os` into the block hash.
7. The SNOS proof is generated over `R_os` and submitted to L1. L1 accepts it.
8. Any call to `get_class_hash_at(A)` now returns `0xdeadbeef` — a class hash with no corresponding compiled class — corrupting all future execution that dispatches through `A`.

pub fn calculate_block_hash(
    partial_block_hash_components: &PartialBlockHashComponents,
    state_root: GlobalRoot,
    previous_block_hash: BlockHash,
) -> StarknetApiResult<BlockHash> {
    let block_hash_version: BlockHashVersion =
        partial_block_hash_components.starknet_version.try_into()?;
    let block_commitments = &partial_block_hash_components.header_commitments;
    Ok(BlockHash(
        HashChain::new()
            .chain(&block_hash_version.clone().into())
            .chain(&partial_block_hash_components.block_number.0.into())
            .chain(&state_root.0)
            .chain(&partial_block_hash_components.sequencer.0)
            .chain(&partial_block_hash_components.timestamp.0.into())
            .chain(&block_commitments.concatenated_counts)
            .chain(&block_commitments.state_diff_commitment.0.0)
            .chain(&block_commitments.transaction_commitment.0)
            .chain(&block_commitments.event_commitment.0)
            .chain(&block_commitments.receipt_commitment.0)
            .chain_iter(
                gas_prices_to_hash(
                    &partial_block_hash_components.l1_gas_price,
                    &partial_block_hash_components.l1_data_gas_price,
                    &partial_block_hash_components.l2_gas_price,
                    &block_hash_version,
                )
                .iter(),
            )
            .chain(
                &Felt::try_from(&partial_block_hash_components.starknet_version)
                    .expect("Expect ASCII version"),
            )
            .chain(&Felt::ZERO)
            .chain(&previous_block_hash.0)
            .get_poseidon_hash(),
    ))
}
