### Title
`execute_deploy` Syscall Unconditionally Reports Success Regardless of Constructor Outcome — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_deploy` syscall implementation in the StarkNet OS always writes a success response (`failure_flag=0`) to the calling contract, regardless of whether the constructor execution succeeded or reverted. This is structurally analogous to the external report's vulnerability class: a resource (funds) is committed against an obligation (successful deployment) that is never actually verified, with no mechanism to recover.

---

### Finding Description

In `execute_deploy` (`syscall_impls.cairo`, lines 527–555), after calling `deploy_contract`, the response header is unconditionally written with `failure_flag=0`:

```cairo
with remaining_gas {
    let (retdata_size, retdata) = deploy_contract(
        block_context=block_context, constructor_execution_context=constructor_execution_context
    );
}

let response_header = cast(syscall_ptr, ResponseHeader*);
let syscall_ptr = syscall_ptr + ResponseHeader.SIZE;

// Write the response header.
// TODO(Yoni, 1/1/2026): support failures.
assert [response_header] = ResponseHeader(gas=remaining_gas, failure_flag=0);
``` [1](#0-0) 

The TODO comment explicitly acknowledges that failure cases are not supported. Compare this to `contract_call_helper`, which correctly propagates the revert status:

```cairo
assert [response_header] = ResponseHeader(gas=remaining_gas, failure_flag=is_reverted);
``` [2](#0-1) 

When a constructor reverts, the revert log mechanism rolls back the constructor's state changes. However, the calling contract receives `failure_flag=0` — a success signal — with no way to distinguish a successful deployment from a failed one. The `DeployResponse` struct still contains the `contract_address`, so the caller proceeds as if the contract is live. [3](#0-2) 

---

### Impact Explanation

Any on-chain factory or protocol that:
1. Uses the `deploy` syscall to create user-specific contracts (vaults, escrows, position managers), and
2. Sends user funds to the returned `contract_address` after deployment,

…will commit funds to an address whose contract is in an undefined or uninitialized state (constructor did not complete). Because the constructor's storage writes were reverted, any access-control or initialization logic the constructor was supposed to establish is absent. Funds sent to that address are permanently frozen — there is no OS-level mechanism to recover them, and the calling contract has no signal that anything went wrong.

**Impact: Critical — Permanent freezing of funds.**

---

### Likelihood Explanation

The factory/deploy pattern is ubiquitous in DeFi (lending vaults, AMM pools, escrow accounts). Any such protocol on StarkNet that uses the `deploy` syscall is exposed. A constructor can revert due to:
- Out-of-gas conditions (attacker-controllable via gas bounds)
- Attacker-supplied constructor calldata that triggers a revert path
- Edge-case arithmetic in the constructor

An unprivileged attacker who can interact with a protocol's factory function (a "public protocol input user" per the scope definition) can trigger this path. No privileged role is required.

---

### Recommendation

`execute_deploy` must propagate the constructor's revert status to the caller, mirroring the pattern used in `contract_call_helper`:

```cairo
// Correct pattern (from contract_call_helper):
assert [response_header] = ResponseHeader(gas=remaining_gas, failure_flag=is_reverted);
```

`deploy_contract` should return an `is_reverted` flag (analogous to `select_execute_entry_point_func`), and `execute_deploy` should forward it in the `ResponseHeader`. When `failure_flag=1`, the `contract_address` field in `DeployResponse` should be treated as invalid by callers.

---

### Proof of Concept

1. Attacker declares a class `VaultClass` whose constructor reverts when `calldata[0] == MAGIC_VALUE`.
2. A legitimate factory contract `F` uses `deploy(class_hash=VaultClass, constructor_calldata=[MAGIC_VALUE, ...])` to deploy a vault for a user.
3. The constructor reverts; the revert log rolls back all storage writes. The vault address has the class hash set but no initialized storage (no owner, no access control).
4. `execute_deploy` writes `ResponseHeader(gas=..., failure_flag=0)` and `DeployResponse(contract_address=vault_addr, ...)`.
5. Factory `F` receives success, records `vault_addr`, and transfers user tokens to `vault_addr`.
6. Because the constructor never ran, there is no owner or withdrawal function initialized. Tokens are permanently frozen at `vault_addr`. [4](#0-3)

func execute_deploy{
    range_check_ptr,
    syscall_ptr: felt*,
    builtin_ptrs: BuiltinPointers*,
    contract_state_changes: DictAccess*,
    contract_class_changes: DictAccess*,
    revert_log: RevertLogEntry*,
    outputs: OsCarriedOutputs*,
}(block_context: BlockContext*, caller_execution_context: ExecutionContext*) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, DeployRequest*);
    local constructor_calldata_start: felt* = request.constructor_calldata_start;
    local constructor_calldata_size = request.constructor_calldata_end - constructor_calldata_start;

    let specific_base_gas_cost = DEPLOY_GAS_COST + DEPLOY_CALLDATA_FACTOR_GAS_COST *
        constructor_calldata_size;
    let (success, remaining_gas) = reduce_syscall_base_gas(
        specific_base_gas_cost=specific_base_gas_cost, request_struct_size=DeployRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    local caller_execution_info: ExecutionInfo* = caller_execution_context.execution_info;
    local caller_address = caller_execution_info.contract_address;

    // Verify deploy_from_zero is either 0 (FALSE) or 1 (TRUE).
    tempvar deploy_from_zero = request.deploy_from_zero;
    assert deploy_from_zero * (deploy_from_zero - 1) = 0;
    // Set deployer_address to 0 if request.deploy_from_zero is TRUE.
    let deployer_address = (1 - deploy_from_zero) * caller_address;

    let selectable_builtins = &builtin_ptrs.selectable;
    let hash_ptr = selectable_builtins.pedersen;
    with hash_ptr {
        let (contract_address) = get_contract_address(
            salt=request.contract_address_salt,
            class_hash=request.class_hash,
            constructor_calldata_size=constructor_calldata_size,
            constructor_calldata=constructor_calldata_start,
            deployer_address=deployer_address,
        );
    }
    tempvar builtin_ptrs = new BuiltinPointers(
        selectable=SelectableBuiltins(
            pedersen=hash_ptr,
            range_check=selectable_builtins.range_check,
            ecdsa=selectable_builtins.ecdsa,
            bitwise=selectable_builtins.bitwise,
            ec_op=selectable_builtins.ec_op,
            poseidon=selectable_builtins.poseidon,
            segment_arena=selectable_builtins.segment_arena,
            range_check96=selectable_builtins.range_check96,
            add_mod=selectable_builtins.add_mod,
            mul_mod=selectable_builtins.mul_mod,
        ),
        non_selectable=builtin_ptrs.non_selectable,
    );

    tempvar constructor_execution_context = new ExecutionContext(
        entry_point_type=ENTRY_POINT_TYPE_CONSTRUCTOR,
        class_hash=request.class_hash,
        calldata_size=constructor_calldata_size,
        calldata=constructor_calldata_start,
        execution_info=new ExecutionInfo(
            block_info=caller_execution_info.block_info,
            tx_info=caller_execution_info.tx_info,
            caller_address=caller_address,
            contract_address=contract_address,
            selector=CONSTRUCTOR_ENTRY_POINT_SELECTOR,
        ),
        deprecated_tx_info=caller_execution_context.deprecated_tx_info,
    );

    with remaining_gas {
        let (retdata_size, retdata) = deploy_contract(
            block_context=block_context, constructor_execution_context=constructor_execution_context
        );
    }

    // TODO(Yoni, 1/1/2026): consider sharing code with call_contract_helper.
    let response_header = cast(syscall_ptr, ResponseHeader*);
    let syscall_ptr = syscall_ptr + ResponseHeader.SIZE;

    // Write the response header.
    // TODO(Yoni, 1/1/2026): support failures.
    assert [response_header] = ResponseHeader(gas=remaining_gas, failure_flag=0);

    let response = cast(syscall_ptr, DeployResponse*);
    // Advance syscall pointer to the next syscall.
    let syscall_ptr = syscall_ptr + DeployResponse.SIZE;

    %{ CheckNewDeployResponse %}

    // Write the response.
    relocate_segment(src_ptr=response.constructor_retdata_start, dest_ptr=retdata);
    assert [response] = DeployResponse(
        contract_address=contract_address,
        constructor_retdata_start=retdata,
        constructor_retdata_end=retdata + retdata_size,
    );

    return ();
}
