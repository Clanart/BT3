### Title
Cairo 0 Class Declaration Not Validated Against State Number in RPC Execution State Reader — (`File: crates/apollo_rpc_execution/src/state_reader.rs`)

### Summary

`get_compiled_class` in `ExecutionStateReader` checks the class manager (a content-addressed store that holds all ever-synced classes) as the authoritative source for compiled classes, but only validates that a Cairo 1 class is actually declared at the queried `state_number`. For Cairo 0 (deprecated) classes the declaration check is explicitly skipped with a TODO. This means any Cairo 0 class present in the class manager — including classes declared in future blocks or in reverted blocks — is returned as valid for any historical state query, causing RPC execution, fee estimation, and simulation to produce wrong authoritative-looking results.

### Finding Description

`ExecutionStateReader::get_compiled_class` follows this logic: [1](#0-0) 

For Cairo 1 classes the code calls `is_contract_class_declared` against `state_number` before returning the class. For Cairo 0 classes the check is absent:

```rust
// TODO(shahak): Verify cairo0 as well after get_class_definition_block_number is fixed.
ContractClass::V0(deprecated_contract_class) => {
    Ok(RunnableCompiledClass::V0(deprecated_contract_class.try_into()?))
}
``` [2](#0-1) 

The class manager (`CachedClassStorage` / `FsClassStorage`) is a stateless, content-addressed store that accumulates every class ever synced, with no block-number scoping: [3](#0-2) 

The `deprecated_declared_classes_block_table` in `apollo_storage` records the first block at which each Cairo 0 class was declared and is the correct authority for "is this class declared at state N": [4](#0-3) 

Because `get_compiled_class` never consults this table for Cairo 0 classes, the class manager acts as the sole gate — exactly the pattern of the external bug where `hasRole` was checked without validating against `_roles`.

The analog mapping:

| External (`PermissionsRegistry`) | Sequencer (`ExecutionStateReader`) |
|---|---|
| `_roles` list (valid roles) | `deprecated_declared_classes_block_table` (valid declarations at state N) |
| `hasRole` mapping (user → role) | Class manager store (class_hash → compiled class) |
| `removeRole` removes from list only | Block revert / future-block scenario: class absent from table but present in manager |
| Access check reads `hasRole` directly | `get_compiled_class` reads class manager directly for V0 |

### Impact Explanation

Any caller of `starknet_call`, `starknet_estimateFee`, `starknet_simulateTransactions`, or `starknet_traceTransaction` at block N can supply a transaction that invokes `library_call` with a Cairo 0 class hash that was declared in block N+1 (already synced into the class manager). The RPC layer executes the call successfully and returns a result, while the identical transaction submitted on-chain would revert with `UndeclaredClassHash`. The simulation output is authoritative-looking but wrong.

The same scenario applies after a sequencer revert: the class manager retains the class after `revert_state_diff` removes it from the state tables, so post-revert RPC queries still see the class as executable.

Impact: **High — RPC execution, fee estimation, tracing, simulation returns an authoritative-looking wrong value.**

### Likelihood Explanation

- Unprivileged: any user can call the affected RPC methods.
- Trigger condition: a Cairo 0 class declared in any block after the queried block, or in a reverted block. Both are normal operational states on a live node.
- The class manager is populated eagerly by the sync process, so the window between a class being synced and it being queryable at historical state numbers is always open.

### Recommendation

Apply the same `is_declared` guard to Cairo 0 classes that is already applied to Cairo 1 classes. Replace the TODO branch with a check against `deprecated_declared_classes_block_table`:

```rust
ContractClass::V0(deprecated_contract_class) => {
    let is_declared = is_deprecated_contract_class_declared(
        &self.storage_reader.begin_ro_txn()...,
        &class_hash,
        self.state_number,
    )?;
    if is_declared {
        Ok(RunnableCompiledClass::V0(deprecated_contract_class.try_into()?))
    } else {
        Err(StateError::UndeclaredClassHash(class_hash))
    }
}
```

where `is_deprecated_contract_class_declared` reads `get_deprecated_class_definition_block_number` and compares against `state_number`. [5](#0-4) 

### Proof of Concept

1. Node is synced through block N+1. Block N+1 declares Cairo 0 class `C` (hash `H`). The class manager now holds `H → compiled_C`.

2. Attacker calls `starknet_call` at `block_id = N` with a transaction that invokes `library_call(class_hash=H, ...)`.

3. `ExecutionStateReader` is constructed with `state_number = right_after_block(N)`.

4. The blockifier calls `get_compiled_class(H)`. The class manager returns `ContractClass::V0(compiled_C)`. The TODO branch returns `Ok(RunnableCompiledClass::V0(...))` without consulting `deprecated_declared_classes_block_table`.

5. The RPC returns a successful execution result. The same transaction submitted on-chain at block N would fail with `UndeclaredClassHash(H)` because `H` is not in `deprecated_declared_classes_block_table` at state N. [6](#0-5) [7](#0-6)

    /// Note: when self.class_manager_handle is [Some] this function must be run in a
    /// tokio::spawn_blocking() thread, because of the blocking code.
    fn get_compiled_class(&self, class_hash: ClassHash) -> StateResult<RunnableCompiledClass> {
        if let Some(pending_classes) =
            self.maybe_pending_data.as_ref().map(|pending_data| &pending_data.classes)
        {
            if let Some(api_contract_class) = pending_classes.get_class(class_hash) {
                match api_contract_class {
                    ApiContractClass::ContractClass(sierra) => {
                        if let Some(pending_casm) = pending_classes.get_compiled_class(class_hash) {
                            let sierra_version = sierra.get_sierra_version()?;
                            let runnable_compiled_class = RunnableCompiledClass::V1(
                                CompiledClassV1::try_from((pending_casm, sierra_version))
                                    .map_err(StateError::ProgramError)?,
                            );
                            return Ok(runnable_compiled_class);
                        }
                    }
                    ApiContractClass::DeprecatedContractClass(pending_deprecated_class) => {
                        return Ok(RunnableCompiledClass::V0(
                            CompiledClassV0::try_from(pending_deprecated_class)
                                .map_err(StateError::ProgramError)?,
                        ));
                    }
                }
            }
        }

        if let Some((class_manager_client, run_time_handle)) = &self.class_manager_handle {
            let contract_class = run_time_handle
                .block_on(class_manager_client.get_executable(class_hash))
                .map_err(|e| StateError::StateReadError(e.to_string()))?
                .ok_or(StateError::UndeclaredClassHash(class_hash))?;

            return match contract_class {
                ContractClass::V1(casm_contract_class) => {
                    let is_declared = is_contract_class_declared(
                        &self.storage_reader.begin_ro_txn().map_err(storage_err_to_state_err)?,
                        &class_hash,
                        self.state_number,
                    )
                    .map_err(|e| StateError::StateReadError(e.to_string()))?;

                    if is_declared {
                        Ok(RunnableCompiledClass::V1(casm_contract_class.try_into()?))
                    } else {
                        Err(StateError::UndeclaredClassHash(class_hash))
                    }
                }
                // TODO(shahak): Verify cairo0 as well after get_class_definition_block_number is
                // fixed.
                ContractClass::V0(deprecated_contract_class) => {
                    Ok(RunnableCompiledClass::V0(deprecated_contract_class.try_into()?))
                }
            };
        }

        match get_contract_class(
            &self.storage_reader.begin_ro_txn().map_err(storage_err_to_state_err)?,
            &class_hash,
            self.state_number,
        ) {
            Ok(Some(contract_class)) => Ok(contract_class),
            Ok(None) => Err(StateError::UndeclaredClassHash(class_hash)),
            Err(ExecutionUtilsError::CasmTableNotSynced) => {
                self.missing_compiled_class.set(Some(class_hash));
                Err(StateError::StateReadError("Casm table not fully synced".to_string()))
            }
            Err(ExecutionUtilsError::ProgramError(err)) => Err(StateError::ProgramError(err)),
            Err(ExecutionUtilsError::StorageError(err)) => Err(storage_err_to_state_err(err)),
            Err(ExecutionUtilsError::SierraValidationError(err)) => {
                Err(StateError::StarknetApiError(err))
            }
        }
    }

impl<S: ClassStorage> CachedClassStorage<S> {
    pub fn new(config: CachedClassStorageConfig, storage: S) -> Self {
        Self {
            storage,
            classes: GlobalContractCache::new(config.class_cache_size),
            executable_classes: GlobalContractCache::new(config.class_cache_size),
            executable_class_hashes_v2: GlobalContractCache::new(config.class_cache_size),
            deprecated_classes: GlobalContractCache::new(config.deprecated_class_cache_size),

    }

        self.executable_class_hashes_v2.get(&class_id).is_some()
    }

    pub fn deprecated_class_cached(&self, class_id: ClassId) -> bool {
        self.deprecated_classes.get(&class_id).is_some()
    }
}

    /// Returns the deprecated contract class at a given state number for a given class hash.
    /// If class is not found, returns `None`.
    /// If class is defined but in a block after given state number, returns `None`.
    ///
    /// # Arguments
    /// * state_number - state number to search before.
    /// * class_hash - class hash to search for.
    ///
    /// # Errors
    /// Returns [`StorageError`] if there was an error searching the table.
    pub fn get_deprecated_class_definition_at(
        &self,
        state_number: StateNumber,
        class_hash: &ClassHash,
    ) -> StorageResult<Option<DeprecatedContractClass>> {
        let Some(value) = self.deprecated_declared_classes_table.get(self.txn, class_hash)? else {
            return Ok(None);
        };
        if state_number.is_before(value.block_number) {
            return Ok(None);
        }
        // TODO(shahak): Fix code duplication with ClassStorageReader.
        Ok(Some(
            self.file_handlers.get_deprecated_contract_class_unchecked(value.location_in_file)?,
        ))
