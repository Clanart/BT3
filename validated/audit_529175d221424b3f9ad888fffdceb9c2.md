### Title
Non-EVM Cosmos Transactions Inflate Global Block Gas Counter Used for EIP-1559 Base Fee Calculation - (File: x/feemarket/keeper/abci.go)

### Summary
The `x/feemarket` module's `EndBlock` reads `ctx.BlockGasUsed()` and `ctx.BlockGasWanted()` — global Cosmos SDK counters that accumulate gas from **all** transactions in the block, including non-EVM Cosmos native transactions (bank sends, staking, governance, IBC, etc.). This inflated value is stored as `BlockGasWanted` and used in the next block's `BeginBlock` to compute the EIP-1559 base fee. Because Cosmos native transactions do not pay the EIP-1559 base fee, any unprivileged user can submit cheap Cosmos transactions to artificially inflate the block gas counter, driving the base fee upward and forcing EVM users to overpay.

### Finding Description

In `x/feemarket/keeper/abci.go`, `EndBlock` reads the global block gas counters:

```go
gasWanted := ctx.BlockGasWanted()
gasUsed, err := ethermint.SafeInt64(ctx.BlockGasUsed())
...
gasWanted = sdkmath.LegacyMaxDec(limitedGasWanted, sdkmath.LegacyNewDec(gasUsed)).TruncateInt().Uint64()
k.SetBlockGasWanted(ctx, gasWanted)
``` [1](#0-0) 

`ctx.BlockGasUsed()` and `ctx.BlockGasWanted()` are Cosmos SDK context values that accumulate gas from every transaction processed in the block — EVM and non-EVM alike. For EVM transactions, `ResetGasMeterAndConsumeGas` resets the gas meter to the EVM-specific gas used. For Cosmos native transactions, the SDK gas consumed by those messages is added directly to the same global counter.

The stored `BlockGasWanted` is then consumed in the next block's `BeginBlock` via `CalculateBaseFee`:

```go
parentGasUsed := k.GetBlockGasWanted(ctx)
``` [2](#0-1) 

If `parentGasUsed > parentGasTarget` (where `parentGasTarget = blockGasLimit / ElasticityMultiplier`), the base fee increases:

```go
baseFeeDelta = max(parentBaseFee * gasUsedDelta / parentGasTarget / BaseFeeChangeDenominator, 1)
``` [3](#0-2) 

The existing `MinGasMultiplier` protection only guards against validators setting `gasWanted` artificially low relative to `gasUsed`; it does not prevent `gasUsed` itself from being inflated by Cosmos native transactions. [4](#0-3) 

This is the direct analog to the external report: just as external minters inflate `nft.totalSupply()` and reduce the apparent availability in `NFTMintSale`, external Cosmos transaction senders inflate the global block gas counter and drive up the EIP-1559 base fee that EVM users must pay.

### Impact Explanation

Cosmos native transactions do not pay the EIP-1559 base fee — they pay Cosmos gas prices, which are independent. Therefore, when Cosmos transactions inflate `ctx.BlockGasUsed()`, the resulting base fee increase does not reflect actual EVM demand. EVM users in subsequent blocks are forced to pay a higher base fee than the true EVM-only demand warrants. This is a fee market accounting bug where valid EVM user funds are mis-accounted: users overpay fees due to interference from an unrelated transaction class.

### Likelihood Explanation

Any unprivileged user can submit Cosmos native transactions (e.g., `MsgSend`, `MsgDelegate`, governance votes). If the minimum Cosmos gas price is lower than the EVM base fee, the attacker can inflate the block gas counter at a cost far below the fee increase imposed on EVM users. The attack is sustained by continuously submitting Cosmos transactions across blocks, since the base fee adjusts per-block. No special privileges, validator access, or key compromise is required.

### Recommendation

The `x/feemarket` module should maintain a **local, EVM-only gas counter** rather than relying on the global `ctx.BlockGasUsed()` / `ctx.BlockGasWanted()` counters. Specifically:

1. Track EVM-only gas used in a dedicated transient store key (analogous to `ObjectGasUsedKey` already used in `x/evm/keeper/keeper.go` for per-tx gas accumulation).
2. In `EndBlock`, sum only the EVM-specific gas used (e.g., by reading the per-tx EVM gas from the object store) rather than reading `ctx.BlockGasUsed()`.
3. Use this EVM-only total as the input to `SetBlockGasWanted`, so the EIP-1559 base fee reflects only EVM transaction demand. [5](#0-4) 

### Proof of Concept

1. Chain is running with EIP-1559 enabled (`NoBaseFee = false`, `EnableHeight` reached).
2. Block gas limit is `L`; gas target is `L / ElasticityMultiplier` (e.g., `L / 2`).
3. Attacker submits Cosmos `MsgSend` transactions totaling gas `G_cosmos` such that `G_evm + G_cosmos > L / 2`, where `G_evm` is the actual EVM gas used in the block.
4. `ctx.BlockGasUsed()` at `EndBlock` = `G_evm + G_cosmos`.
5. `SetBlockGasWanted` stores `max((G_evm + G_cosmos) * MinGasMultiplier, G_evm + G_cosmos)`.
6. Next block's `CalculateBaseFee` reads this inflated value as `parentGasUsed`, computes `gasUsedDelta = parentGasUsed - parentGasTarget > 0`, and increases the base fee.
7. EVM users in the next block pay a higher base fee than actual EVM demand warrants.
8. Attacker repeats across blocks to sustain the elevated base fee, paying only Cosmos gas prices (which may be far cheaper than the EVM base fee they are inflating). [6](#0-5) [7](#0-6)

func (k *Keeper) EndBlock(ctx sdk.Context) error {
	gasWanted := ctx.BlockGasWanted()
	gw, err := ethermint.SafeInt64(gasWanted)
	if err != nil {
		return err
	}
	gasUsed, err := ethermint.SafeInt64(ctx.BlockGasUsed())
	if err != nil {
		return err
	}

	// to prevent BaseFee manipulation we limit the gasWanted so that
	// gasWanted = max(gasWanted * MinGasMultiplier, gasUsed)
	// this will be keep BaseFee protected from un-penalized manipulation
	// more info here https://github.com/evmos/ethermint/pull/1105#discussion_r888798925
	minGasMultiplier := k.GetParams(ctx).MinGasMultiplier
	limitedGasWanted := sdkmath.LegacyNewDec(gw).Mul(minGasMultiplier)
	gasWanted = sdkmath.LegacyMaxDec(limitedGasWanted, sdkmath.LegacyNewDec(gasUsed)).TruncateInt().Uint64()
	k.SetBlockGasWanted(ctx, gasWanted)

	defer func() {
		telemetry.SetGauge(float32(gasWanted), "feemarket", "block_gas") //nolint:staticcheck
	}()

	ctx.EventManager().EmitEvent(sdk.NewEvent(
		"block_gas",
		sdk.NewAttribute("height", fmt.Sprintf("%d", ctx.BlockHeight())),
		sdk.NewAttribute("amount", fmt.Sprintf("%d", gasWanted)),
	))
	return nil

	parentGasUsed := k.GetBlockGasWanted(ctx)

	// NOTE: a MaxGas equal to -1 means that block gas is unlimited
	if consParams.Block == nil || consParams.Block.MaxGas <= -1 {
		panic(fmt.Sprintf("get invalid consensus params: %s", consParams))
	}
	gasLimit := big.NewInt(consParams.Block.MaxGas)
	// CONTRACT: ElasticityMultiplier cannot be 0 as it's checked in the params
	// validation
	parentGasTargetBig := new(big.Int).Div(gasLimit, new(big.Int).SetUint64(uint64(params.ElasticityMultiplier)))
	if !parentGasTargetBig.IsUint64() {
		return nil
	}

	parentGasTarget := parentGasTargetBig.Uint64()
	baseFeeChangeDenominator := new(big.Int).SetUint64(uint64(params.BaseFeeChangeDenominator))

	// If the parent gasUsed is the same as the target, the baseFee remains
	// unchanged.
	if parentGasUsed == parentGasTarget {
		return new(big.Int).Set(parentBaseFee)
	}

	if parentGasUsed > parentGasTarget {
		// If the parent block used more gas than its target, the baseFee should
		// increase.
		gasUsedDelta := new(big.Int).SetUint64(parentGasUsed - parentGasTarget)
		x := new(big.Int).Mul(parentBaseFee, gasUsedDelta)
		y := x.Div(x, parentGasTargetBig)
		baseFeeDelta := ethermint.BigMax(
			x.Div(y, baseFeeChangeDenominator),
			common.Big1,
		)

		return x.Add(parentBaseFee, baseFeeDelta)
	}

	// Otherwise if the parent block used less gas than its target, the baseFee
	// should decrease.
	gasUsedDelta := new(big.Int).SetUint64(parentGasTarget - parentGasUsed)
	x := new(big.Int).Mul(parentBaseFee, gasUsedDelta)
	y := x.Div(x, parentGasTargetBig)
	baseFeeDelta := x.Div(y, baseFeeChangeDenominator)

	// Set global min gas price as lower bound of the base fee, transactions below
	// the min gas price don't even reach the mempool.
	minGasPrice := params.MinGasPrice.TruncateInt().BigInt()
	return ethermint.BigMax(x.Sub(parentBaseFee, baseFeeDelta), minGasPrice)

// GetTransientGasUsed returns the gas used by current cosmos tx.
func (k Keeper) GetTransientGasUsed(ctx sdk.Context) uint64 {
	store := ctx.ObjectStore(k.objectKey)
	v := store.Get(types.ObjectGasUsedKey(ctx.TxIndex()))
	if v == nil {
		return 0
	}
	return v.(uint64)
}

// SetTransientGasUsed sets the gas used by current cosmos tx.
func (k Keeper) SetTransientGasUsed(ctx sdk.Context, gasUsed uint64) {
	store := ctx.ObjectStore(k.objectKey)
	store.Set(types.ObjectGasUsedKey(ctx.TxIndex()), gasUsed)
}

// AddTransientGasUsed accumulate gas used by each eth msgs included in current cosmos tx.
func (k Keeper) AddTransientGasUsed(ctx sdk.Context, gasUsed uint64) (uint64, error) {
	result := k.GetTransientGasUsed(ctx) + gasUsed
	if result < gasUsed {
		return 0, errorsmod.Wrap(types.ErrGasOverflow, "transient gas used")
	}
	k.SetTransientGasUsed(ctx, result)
	return result, nil
}
