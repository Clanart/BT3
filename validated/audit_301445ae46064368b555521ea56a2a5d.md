### Title
`inner_build_blob_from_node_list` Writes Attacker-Supplied Internal Node Hash Without Recomputation, Enabling Forged DataLayer Root - (`crates/chia-datalayer/src/merkle/blob.rs`)

### Summary

`MerkleBlob::inner_build_blob_from_node_list` uses the map key (`node_hash`) directly as the stored hash for internal nodes, without ever recomputing `internal_hash(left_child_hash, right_child_hash)`. An attacker who controls the `InternalNodesMap` passed to `DeltaReader::new` (or `py_add_internal_nodes`) can supply a lying hash as the map key, and it will be written verbatim into the resulting `MerkleBlob`. Neither `check_integrity()` nor the blob construction path recomputes or validates internal node hashes against their children.

### Finding Description

**Entry point — `DeltaReader::new` / `py_init`:**

`DeltaReader::new` accepts an `InternalNodesMap` (`HashMap<Hash, (Hash, Hash)>`) and inserts every entry directly into `self.nodes` with zero validation: [1](#0-0) 

There is no check that the map key equals `internal_hash(left, right)`.

**Blob construction — `inner_build_blob_from_node_list`:**

When building the blob for an `Internal` node, the function looks up `node_hash` in the map, recursively builds children, then writes a `Block` whose `hash` field is set to `node_hash` — the attacker-supplied map key — not a freshly computed `internal_hash(left_child_hash, right_child_hash)`: [2](#0-1) 

**`check_integrity()` does not catch this:**

`check_integrity` calls `calculate_lazy_hashes()` on a clone, but that function only recomputes hashes for nodes where `block.metadata.dirty == true`. Every node written by `inner_build_blob_from_node_list` is created with `dirty: false`: [3](#0-2) [4](#0-3) 

`check_just_integrity()` only validates structural properties (parent-child pointer consistency, key counts, free-index accounting) — it never compares a stored internal hash against `internal_hash(left, right)`: [5](#0-4) 

**`get_root_hash()` returns the lying hash verbatim:** [6](#0-5) 

**The existing test inadvertently proves the bug:**

The test `test_delta_reader_create_merkle_blob_works` constructs a `DeltaReader` where `HASH_ZERO` maps to `(HASH_ONE, HASH_TWO)` — values that are clearly not the real `internal_hash(HASH_ONE, HASH_TWO)` — then asserts `check_integrity()` passes. It does pass, confirming the integrity check is blind to hash forgery: [7](#0-6) 

### Impact Explanation

An attacker who can supply data to `DeltaReader` (e.g., a malicious DataLayer peer serving crafted delta files, or any caller of the Python `py_init` / `py_add_internal_nodes` bindings) can produce a `MerkleBlob` whose root hash is an arbitrary value of their choosing, independent of the actual leaf contents. Any downstream consumer that calls `get_root_hash()` on the resulting blob and treats the result as authoritative DataLayer state will accept the forged root. This maps directly to the allowed High impact: **DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state.**

Note: `ProofOfInclusion::valid()` does recompute hashes bottom-up and would detect the forgery if called, but `get_root_hash()` and `check_integrity()` do not, so any code path that skips proof validation is fully exposed.

### Likelihood Explanation

The Python bindings (`py_init`, `py_add_internal_nodes`) are the direct public API surface. Any DataLayer peer node that serves delta files can craft an `InternalNodesMap` with lying keys. No privileged access, key material, or chain reorg is required — only the ability to supply a crafted map to `DeltaReader`.

### Recommendation

In `inner_build_blob_from_node_list`, after recursively building left and right children, recompute the internal hash and assert (or return an error) that it matches `node_hash`:

```rust
let recomputed = internal_hash(&left_child_hash, &right_child_hash);
if recomputed != node_hash {
    return Err(Error::InternalHashMismatch(node_hash, recomputed));
}
```

Alternatively, add a dedicated validation pass in `check_integrity` / `check_just_integrity` that walks every internal node and verifies its stored hash equals `internal_hash(left_child_hash, right_child_hash)`.

### Proof of Concept

```rust
// Construct InternalNodesMap with a lying root hash
let mut internal_nodes = InternalNodesMap::new();
let mut leaf_nodes = LeafNodesMap::new();

let real_left_hash  = Hash([0x01; 32]);
let real_right_hash = Hash([0x02; 32]);
let lying_root_hash = Hash([0xFF; 32]); // != internal_hash(left, right)

internal_nodes.insert(lying_root_hash, (real_left_hash, real_right_hash));
leaf_nodes.insert(real_left_hash,  (KeyId(1), ValueId(10)));
leaf_nodes.insert(real_right_hash, (KeyId(2), ValueId(20)));

let mut reader = DeltaReader::new(internal_nodes, leaf_nodes).unwrap();
let blob = reader
    .create_merkle_blob_and_filter_unused_nodes(lying_root_hash, &HashSet::new())
    .unwrap();

// check_integrity passes — does NOT detect the forgery
blob.check_integrity().unwrap();

// get_root_hash returns the lying hash, not internal_hash(left, right)
let root = blob.get_hash_at_index(TreeIndex(0)).unwrap().unwrap();
assert_eq!(root, lying_root_hash); // passes — forgery accepted
assert_ne!(root, internal_hash(&Hash([0x01;32]), &Hash([0x02;32]))); // confirmed lie
```

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
