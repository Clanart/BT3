### Title
Missing Declared Class Hash Verification in `execute_replace_class` Allows Permanent Contract Freezing - (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS accepts an arbitrary `class_hash` from the caller and writes it directly into the contract state without verifying that the hash corresponds to a previously declared contract class. This is a direct analog to the reported "Missing Type Check" vulnerability class: just as the original bug allowed placing orders with unregistered coin types, this bug allows replacing a contract's class with an undeclared class hash, permanently corrupting the contract's state and freezing any funds it holds.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function processes the `replace_class` syscall:

```cairo
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    ...
    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );
    ...
}
``` [1](#0-0) 

The `class_hash` value comes directly from `request.class_hash`, which is attacker-controlled calldata. The OS developers themselves acknowledge the missing check with the TODO comment at line 898: `// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.` [2](#0-1) 

The correct check would be to verify that `class_hash` exists as a key in `contract_class_changes` (i.e., it was previously declared via a `declare` transaction). The `execute_declare_transaction` function enforces this on the declare side by writing `dict_update{dict_ptr=contract_class_changes}(key=[class_hash_ptr], prev_value=0, new_value=compiled_class_hash)`, but `execute_replace_class` never reads from `contract_class_changes` to confirm the hash is present. [3](#0-2) 

When a subsequent call is made to the contract after the class replacement, `execute_entry_point` reads the compiled class hash from `contract_class_changes` using the (now-invalid) class hash as key:

```cairo
let (compiled_class_hash: felt) = dict_read{dict_ptr=contract_class_changes}(
    key=execution_context.class_hash
);
``` [4](#0-3) 

Since the class hash was never declared, `dict_read` returns 0 (the default/uninitialized value). The subsequent `find_element` call with `key=0` fails to locate a matching `CompiledClassFact`, causing all future calls to the contract to revert. The contract is permanently broken. [5](#0-4) 

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Once a contract's class hash is replaced with an undeclared hash, the contract's `StateEntry.class_hash` in the Patricia Merkle Tree is permanently set to an invalid value. Every subsequent call to the contract will fail at the OS level because no `CompiledClassFact` exists for the bogus hash. Any ERC-20 tokens, ETH, or other assets held in the contract's storage become permanently inaccessible. The state corruption is committed to the global state root and cannot be undone without a protocol-level intervention. [6](#0-5) 

---

### Likelihood Explanation

**High.** The `replace_class` syscall is a standard, publicly documented StarkNet syscall callable by any contract. No privileged role is required. An attacker only needs to:
1. Deploy or control a contract that calls `replace_class` with an arbitrary felt value as the class hash.
2. Submit a valid invoke transaction triggering that code path.

The sequencer's blockifier will execute the syscall, the OS will accept it (no check exists), and the state will be committed. The TODO comment in the source code confirms the developers are aware this check is absent.

---

### Recommendation

In `execute_replace_class`, before writing the new `StateEntry`, verify that `class_hash` is a key present in `contract_class_changes` (i.e., its compiled class hash is non-zero):

```cairo
let (compiled_class_hash: felt) = dict_read{dict_ptr=contract_class_changes}(
    key=class_hash
);
with_attr error_message("Class hash is not declared.") {
    assert_not_zero(compiled_class_hash);
}
```

This mirrors the pattern already used in `execute_entry_point` and enforces that only previously declared classes can be used as replacement targets, consistent with how `execute_declare_transaction` registers classes. [7](#0-6) 

---

### Proof of Concept

1. **Attacker deploys a contract** (class `A`) whose `__execute__` function calls the `replace_class` syscall with an arbitrary felt value `0xdeadbeef` as the new class hash. `0xdeadbeef` has never been declared on-chain.

2. **Attacker submits an invoke transaction** targeting the deployed contract. The sequencer includes it in a block.

3. **OS executes `execute_replace_class`**: `request.class_hash = 0xdeadbeef`. No check against `contract_class_changes` is performed. The OS writes:
   ```
   contract_state_changes[contract_address].class_hash = 0xdeadbeef
   ```
   The block is proven and the state root is updated.

4. **Victim sends funds** (e.g., ERC-20 transfer) to the now-corrupted contract address. The funds are recorded in the contract's storage trie.

5. **Any subsequent call** to the contract reaches `execute_entry_point`. It reads `contract_class_changes[0xdeadbeef]` → returns `0`. Then `find_element(..., key=0)` finds no matching `CompiledClassFact`. The call reverts. The funds are permanently frozen. [8](#0-7)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    let revert_log = &revert_log[1];

    return ();
}
