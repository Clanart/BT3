### Title
`check_integrity` Does Not Verify Internal Node Hash Correctness for Non-Dirty Nodes Built via `DeltaReader` — (`crates/chia-datalayer/src/merkle/blob.rs`)

### Summary
`MerkleBlob::check_integrity` fails to detect cryptographically invalid internal node hashes in blobs constructed via `DeltaReader::create_merkle_blob_and_filter_unused_nodes`. Because `inner_build_blob_from_node_list` marks every node `dirty: false`, and `calculate_lazy_hashes` only recomputes hashes for dirty nodes, the hash-correctness pass inside `check_integrity` is a no-op. `check_just_integrity` only validates structural (parent-child index) consistency, never that `hash == internal_hash(left_child_hash, right_child_hash)`.

### Finding Description
`inner_build_blob_from_node_list` stores the caller-supplied map key as the node's hash verbatim and sets `dirty: false` for every node it creates. `calculate_lazy_hashes` uses `LeftChildFirstIterator::new_with_block_predicate` with predicate `block.metadata.dirty`; when the predicate is false the iterator `continue`s before pushing children, so the entire tree is skipped. `check_just_integrity` checks only structural invariants (parent/child index agreement, cache lengths, total node count) and never compares any stored hash against `internal_hash(left, right)`. The combined effect is that `check_integrity` returns `Ok(())` for any blob produced by `DeltaReader` regardless of whether the stored hashes are correct.

### Impact Explanation
An attacker who can supply crafted `InternalNodesMap` data (e.g., via the Python binding `DeltaReader.py_init`) can inject a `MerkleBlob` with an arbitrary forged root hash that passes `check_integrity`. `get_proof_of_inclusion` reads `combined_hash` directly from the stored (forged) internal node hash, producing proofs anchored to the wrong root. Any system that uses `check_integrity` as a validation gate before accepting or forwarding a DataLayer Merkle blob will accept forged state.

### Likelihood Explanation
The `DeltaReader` is exposed via Python bindings and accepts `InternalNodesMap` directly from callers. Any DataLayer client that reconstructs blobs from delta data and validates them with `check_integrity` is reachable. The exploit requires no privileges beyond the ability to call `DeltaReader` with crafted input.

### Recommendation
In `check_just_integrity`, after traversing each internal node, verify:
```rust
let expected = internal_hash(&left_child_hash, &right_child_hash);
if node.hash != expected {
    return Err(Error::IntegrityHashMismatch(index));
}
```
Alternatively, always mark internal nodes dirty when building from `DeltaReaderNode` data and require `calculate_lazy_hashes` to recompute all hashes before accepting the blob.

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
