### Title
Missing Class Hash Validation in `execute_replace_class` Enables Permanent Fund Freezing — (File: `execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS accepts an arbitrary class hash from a contract without verifying it is actually declared in the system. A contract can call `replace_class` with a non-existent class hash, permanently rendering itself non-functional and freezing any funds it holds. The OS itself acknowledges this missing check via a TODO comment.

---

### Finding Description

In `execute_replace_class`, the `class_hash` field from the syscall request is taken directly from the caller-controlled request struct and written into `contract_state_changes` with no validation that the hash corresponds to a declared class: [1](#0-0) 

The TODO comment at line 898 explicitly acknowledges the missing check:

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
```

After the state update, any future call to the contract enters `execute_entry_point`, which reads the contract's class hash from `contract_state_changes` (now the non-existent hash), then attempts to look up the compiled class hash in `contract_class_changes` via `dict_read`. Since the non-existent hash has no entry, `dict_read` returns `0`. `find_element` is then called with key `0`: [2](#0-1) 

`find_element` panics when the key is absent. The contract is permanently non-functional — no entry point can ever be reached again.

---

### Impact Explanation

Any funds held by a contract whose class hash has been replaced with a non-existent value are **permanently frozen**. There is no recovery path: the OS will panic on every future call to the contract, and no upgrade or withdrawal is possible. This matches the allowed impact: **Critical — Permanent freezing of funds**.

---

### Likelihood Explanation

The attack path requires only standard unprivileged actions:

1. Attacker deploys a contract containing a callable function that invokes `replace_class(non_existent_hash)`.
2. The contract is presented as a legitimate protocol (token, vault, DeFi primitive); users deposit funds.
3. Attacker sends a transaction calling that function.
4. `execute_replace_class` updates the contract's class hash without validation.
5. All subsequent calls to the contract panic in `execute_entry_point`; funds are permanently frozen.

No privileged role, leaked key, or operator access is required. Any unprivileged transaction sender can deploy and trigger this.

---

### Recommendation

Before writing the new class hash into `contract_state_changes`, verify that the hash exists in `contract_class_changes` (i.e., it has been declared). The check should assert that `dict_read{dict_ptr=contract_class_changes}(key=class_hash)` returns a non-zero compiled class hash. The existing TODO at line 898 already identifies exactly this requirement. [3](#0-2) 

---

### Proof of Concept

```
1. Deploy contract C with:
       @external
       func trap() {
           replace_class(0xdeadbeef);  // non-existent class hash
       }

2. Users send funds to C (ETH, ERC-20, etc.).

3. Attacker calls C.trap().
   → execute_replace_class runs, writes class_hash=0xdeadbeef into
     contract_state_changes[C] with no validation.

4. Any subsequent call to C (e.g., withdraw):
   → execute_entry_point reads class_hash=0xdeadbeef from contract_state_changes
   → dict_read(contract_class_changes, 0xdeadbeef) → 0
   → find_element(..., key=0) panics: key not found
   → Transaction fails unconditionally, forever.

5. All funds in C are permanently frozen.
``` [1](#0-0) [4](#0-3)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
