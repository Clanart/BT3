### Title
`replace_class` Syscall Accepts Undeclared Class Hash Without Validation — (`File: crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` function in the StarkNet OS accepts any arbitrary class hash as the replacement class for a contract, without verifying that the supplied hash corresponds to a previously declared class. This is an exact structural analog to the Audius H03 bug class: a sensitive, irreversible state mutation is accepted and committed to the global state root with no enforcement of the invariant that should constrain it. An unprivileged contract deployer can exploit this to permanently freeze funds held by any contract they control.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function processes the `ReplaceClass` syscall:

```cairo

    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    ...
    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );
    ...
}
``` [1](#0-0) 

The `class_hash` value from the syscall request is written directly into `contract_state_changes` with no check that it exists in `contract_class_changes` (the declared class registry). The developer-acknowledged TODO at line 898 confirms this validation is entirely absent.

Contrast this with `execute_declare_transaction`, which enforces `prev_value=0` to guarantee a class is declared at most once: [2](#0-1) 

The `replace_class` path has no symmetric enforcement that the target class hash was ever declared. The updated `contract_state_changes` dict is then squashed and committed into the global state root via `compute_contract_state_commitment`: [3](#0-2) 

This means the corrupted class hash is permanently sealed into the proven state.

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Once a contract's class hash is set to an undeclared value and the block proof is accepted on L1:

1. The state root permanently encodes the invalid class hash for that contract address.
2. Any future transaction that attempts to call the contract requires the OS to look up the class in the compiled class facts bundle. Since the class was never declared, no valid proof can be generated for such a call.
3. All assets (tokens, NFTs, or any value) held in the contract's storage become permanently inaccessible. There is no recovery path: `replace_class` cannot be called again because the contract itself cannot be executed.

---

### Likelihood Explanation

**Medium-High.** The attack requires only:

1. Deploying a contract (permissionless on StarkNet).
2. Including a call to `replace_class` with an arbitrary undeclared felt value as the class hash.
3. Attracting user deposits before executing the replacement.

No privileged role, leaked key, or operator cooperation is needed. The OS will generate a valid proof for the block containing the malicious `replace_class` call because the validation is simply absent. The attack is irreversible once the proof is submitted to L1.

---

### Recommendation

Inside `execute_replace_class`, before writing the new `StateEntry`, verify that `class_hash` exists as a key in `contract_class_changes` (i.e., it has a non-zero compiled class hash). This mirrors the invariant already enforced in `execute_declare_transaction` and in `hash_contract_state_changes` / `compute_class_commitment`, which assume the class tree only contains declared classes.

Concretely, a `dict_read` on `contract_class_changes` for the requested `class_hash` should return a non-zero compiled class hash; otherwise the syscall must fail with an appropriate error response.

---

### Proof of Concept

1. Attacker declares a legitimate Sierra class `C` and deploys contract `A` using class `C`.
2. Users deposit tokens into contract `A` (e.g., `A` is a vault or AMM pool).
3. Attacker calls a function in `A` that internally invokes the `replace_class` syscall with `class_hash = 0xdeadbeef` (never declared).
4. The OS executes `execute_replace_class`:
   - Reads `state_entry` for address `A` (class = `C`).
   - Writes new `StateEntry(class_hash=0xdeadbeef, ...)` into `contract_state_changes`.
   - No validation of `0xdeadbeef` against `contract_class_changes`.
5. `state_update` squashes and commits the change; the global state root now encodes `class_hash(A) = 0xdeadbeef`.
6. The proof is valid and accepted on L1.
7. Any subsequent transaction targeting `A` cannot be proven (class `0xdeadbeef` is absent from the compiled class facts bundle).
8. All funds in `A`'s storage are permanently frozen. [4](#0-3) [5](#0-4)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
