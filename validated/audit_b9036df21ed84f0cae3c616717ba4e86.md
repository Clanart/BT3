### Title
Missing Class Hash Validation in `execute_replace_class` Allows Permanent Freezing of Contract Funds - (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall implementation in the StarkNet OS accepts any arbitrary class hash without verifying that the hash corresponds to a previously declared contract class. An attacker who controls a contract holding user funds can call `replace_class` with an undeclared class hash, permanently rendering the contract non-executable and freezing all funds held within it. The missing check is explicitly acknowledged by a TODO comment in the code.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function processes the `replace_class` syscall. After deducting gas, it reads the caller's current state entry and unconditionally writes the attacker-supplied `class_hash` into `contract_state_changes` with no check that the new class hash has ever been declared on-chain:

```cairo
// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
local state_entry: StateEntry*;
%{ GetContractAddressStateEntry %}

tempvar new_state_entry = new StateEntry(
    class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
);

dict_update{dict_ptr=contract_state_changes}(
    key=contract_address,
    prev_value=cast(state_entry, felt),
    new_value=cast(new_state_entry, felt),
);
``` [1](#0-0) 

The `contract_state_changes` dictionary is later squashed and committed to the global state root via `state_update` in `os.cairo`. Once committed, the contract's class hash in the Merkle state is permanently set to the undeclared value. [2](#0-1) 

When any subsequent transaction attempts to call the affected contract, the OS must look up the compiled class for the stored class hash. Because the class hash was never declared, no compiled class fact exists for it. The contract becomes permanently non-executable, and any ERC-20 tokens, ETH, or other assets held in its storage are irrecoverably frozen.

By contrast, the normal declare path (`execute_declare_transaction`) enforces `prev_value=0` to prevent double-declaration and verifies the class hash pre-image via `finalize_class_hash`, but `execute_replace_class` performs neither check. [3](#0-2) 

---

### Impact Explanation

**Critical â€” Permanent freezing of funds.**

Once the OS commits the state update containing the undeclared class hash, the contract's class hash in the Patricia Merkle Tree is permanently corrupted. No recovery path exists within the protocol: there is no admin override, no rollback mechanism, and no way to re-declare a class at an already-occupied hash slot. All ERC-20 balances, ETH, and any other assets stored in the contract's storage slots are permanently inaccessible.

---

### Likelihood Explanation

**Medium.**

The attack requires only that an attacker deploy (or control) a contract that holds user funds and then invoke the `replace_class` syscall with an arbitrary felt value as the class hash. No privileged role, leaked key, or external dependency is needed. The syscall is available to any Cairo contract. The attack is a single transaction and costs only the gas for the `replace_class` syscall (`REPLACE_CLASS_GAS_COST`). The developers have already identified the gap (the TODO comment targets a fix date of 2026-01-01), confirming the issue is known but unmitigated in the current code.

---

### Recommendation

Before writing the new class hash into `contract_state_changes`, verify that the supplied `class_hash` exists in `contract_class_changes` (i.e., it was declared in the current block) or in the pre-existing state (i.e., it was declared in a prior block). Concretely:

1. Perform a `dict_read` on `contract_class_changes` for `class_hash`; if the result is non-zero, the class was declared in this block and the replacement is valid.
2. If not found in the current block's changes, verify via the state commitment that the class hash maps to a non-zero compiled class hash in the global state.
3. Reject (revert) the syscall with `ERROR_INVALID_ARGUMENT` if neither condition holds, consistent with how `execute_call_contract` rejects the `__execute__` selector. [4](#0-3) 

---

### Proof of Concept

**Setup:**

1. Attacker deploys `VaultContract` â€” a contract that accepts ERC-20 deposits from users and stores balances in its storage. Users deposit 1,000,000 STRK into `VaultContract`.

2. `VaultContract` contains a privileged `upgrade()` entry point that calls the `replace_class` syscall.

**Attack:**

3. Attacker submits an invoke transaction calling `VaultContract.upgrade(class_hash=0xdeadbeef)`, where `0xdeadbeef` is a felt value that has never been declared on-chain.

4. The OS executes `execute_replace_class`:
   - Gas is deducted successfully.
   - `class_hash = 0xdeadbeef` is read from the syscall request.
   - The TODO check is absent; no validation occurs.
   - `dict_update` writes `StateEntry(class_hash=0xdeadbeef, ...)` into `contract_state_changes`. [5](#0-4) 

5. `state_update` squashes `contract_state_changes` and commits the new state root, permanently recording `class_hash=0xdeadbeef` for `VaultContract` in the Patricia Merkle Tree. [6](#0-5) 

6. Any subsequent call to `VaultContract` (e.g., `withdraw()`) causes the OS to look up the compiled class for `0xdeadbeef`. No such class exists in `compiled_class_facts`. Execution fails unconditionally.

7. The 1,000,000 STRK held in `VaultContract`'s storage slots are permanently frozen. No user can withdraw.

    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,

    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(

    );

        key=contract_address,

    );
