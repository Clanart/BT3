### Title
`DeltaReader` Accepts Forged HashŌåÆContent Mappings Without Verification, Enabling Forged DataLayer Inclusion Proofs - (File: `crates/chia-datalayer/src/merkle/deltas.rs`)

### Summary

`DeltaReader::new()`, `py_add_internal_nodes()`, and `py_add_leaf_nodes()` accept caller-supplied `HashMap<Hash, node_content>` mappings and insert them directly into the internal node store without ever verifying that the hash key actually commits to the node content. When `create_merkle_blob_and_filter_unused_nodes()` subsequently builds a `MerkleBlob`, it stores the unverified hash directly into each `LeafNode` and `InternalNode`. The resulting blob's `get_proof_of_inclusion()` then returns a `ProofOfInclusion` whose `node_hash` is the forged key, and `ProofOfInclusion::valid()` returns `true` because the chain of stored hashes is internally self-consistent ŌĆö even though the leaf content is entirely attacker-controlled. A malicious DataLayer peer can exploit this to prove arbitrary key/value pairs are included under any known root hash.

### Finding Description

**Root cause ŌĆö no hash-content binding check in `DeltaReader`:**

`DeltaReader::new()` inserts every supplied `(hash, content)` pair verbatim:

```rust
for (hash, (left, right)) in internal_nodes {
    nodes.insert(hash, DeltaReaderNode::Internal { left, right });
}
for (hash, (key, value)) in leaf_nodes {
    nodes.insert(hash, DeltaReaderNode::Leaf { key, value });
}
```

Neither `calculate_internal_hash(left, right) == hash` nor any leaf-hash preimage check is performed. The same omission exists in `py_add_internal_nodes` and `py_add_leaf_nodes`. [1](#0-0) 

**Forged hash propagates into `MerkleBlob` nodes:**

`inner_build_blob_from_node_list` uses the map key directly as the stored `hash` field of each node, with `dirty: false`, so `calculate_lazy_hashes()` will not overwrite it:

```rust
node: Node::Leaf(LeafNode {
    hash: node_hash,   // ŌåÉ map key, never verified
    key: *key,
    value: *value,
    ...
})
``` [2](#0-1) 

**`ProofOfInclusion::valid()` only checks internal chain consistency:**

`valid()` walks the `layers` vector and verifies that each `combined_hash` equals `calculate_internal_hash(existing_hash, other_hash_side, other_hash)`. For a single-leaf tree with no layers, it simply checks `existing_hash == self.root_hash()`, which is trivially true when `node_hash` was set to the root hash by the attacker. [3](#0-2) 

**Exploit path:**

1. Attacker (malicious DataLayer peer) knows the current root hash `H` (public, committed on-chain).
2. Attacker sends delta response: `leaf_nodes = {H: (attacker_key, attacker_value)}`.
3. Victim calls `DeltaReader.add_leaf_nodes({H: (attacker_key, attacker_value)})` ŌĆö accepted without verification.
4. Victim calls `create_merkle_blob_and_filter_unused_nodes(H, ...)` ŌĆö builds a blob with one leaf whose stored hash is `H` and content is `(attacker_key, attacker_value)`.
5. `get_proof_of_inclusion(attacker_key)` returns `ProofOfInclusion { node_hash: H, layers: [] }`.
6. `proof.valid()` returns `true`; `proof.root_hash()` returns `H` ŌĆö matching the on-chain commitment.
7. The victim's DataLayer now believes `attacker_key ŌåÆ attacker_value` is proven under root `H`.

### Impact Explanation

A malicious DataLayer peer can forge a proof of inclusion for any `(key, value)` pair under any known root hash without breaking SHA-256. The victim's local DataLayer state is corrupted: it accepts forged key/value mappings as proven, and any downstream consumer of `ProofOfInclusion::valid()` is deceived. This directly matches the allowed High impact: **"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."** [4](#0-3) 

### Likelihood Explanation

The `DeltaReader` Python bindings (`add_internal_nodes`, `add_leaf_nodes`) are directly exposed to data received from DataLayer peers during delta synchronization. No privileged role is required ŌĆö any peer that can respond to a delta request can supply forged mappings. The root hash is always public (on-chain), so the attacker has all the information needed to craft the forgery. The attack requires no hash collision and no cryptographic break. [5](#0-4) [6](#0-5) 

### Recommendation

In `DeltaReader::new()`, `py_add_internal_nodes()`, and `py_add_leaf_nodes()`, verify each supplied hash against the actual hash of the node content before inserting:

- For internal nodes: assert `calculate_internal_hash(left, right) == hash`.
- For leaf nodes: assert `hash_leaf(key, value) == hash` (using whatever leaf-hash function the DataLayer uses).

Return an error (or panic) if the check fails. This ensures that no forged hashŌåÆcontent mapping can enter the node store, and that `build_blob_from_node_list` always produces a structurally and cryptographically correct `MerkleBlob`. [1](#0-0) 

### Proof of Concept

```python
from chia_rs.datalayer import DeltaReader, MerkleBlob, KeyId, ValueId

# Suppose the on-chain root hash is H (32 bytes, publicly known)
H = bytes.fromhex("aabbcc" + "00" * 29)  # any known root hash

# Attacker supplies a forged leaf_nodes mapping:
# hash key = H, content = (attacker_key, attacker_value)
attacker_key   = KeyId(9999)
attacker_value = ValueId(8888)

dr = DeltaReader({}, {H: (attacker_key, attacker_value)})
# No error ŌĆö hash is never verified against content

blob = dr.create_merkle_blob_and_filter_unused_nodes(H, {H})
# blob now contains a leaf with stored hash=H but content=(9999, 8888)

proof = blob.get_proof_of_inclusion(attacker_key)
assert proof.root_hash() == H   # matches on-chain commitment
assert proof.valid()            # True ŌĆö forged proof accepted
# Victim now believes attacker_keyŌåÆattacker_value is proven under H
```

    pub fn valid(&self) -> bool {
        let mut existing_hash = self.node_hash;

        for layer in &self.layers {
            let calculated_hash = crate::calculate_internal_hash(
                &existing_hash,
                layer.other_hash_side,
                &layer.other_hash,
            );

            if calculated_hash != layer.combined_hash {
                return false;
            }

            existing_hash = calculated_hash;
        }

        existing_hash == self.root_hash()
    }
