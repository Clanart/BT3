### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Freezal of Contract Funds - (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS program updates a contract's class hash in the global state without verifying that the new class hash corresponds to a previously declared contract class. An unprivileged contract owner can call `replace_class` with an arbitrary, undeclared class hash, permanently replacing the contract's logic pointer with an invalid reference. Any funds held in that contract's storage become permanently inaccessible, as all future calls to the contract will fail to resolve the class.

---

### Finding Description

In `syscall_impls.cairo`, the `execute_replace_class` function reads the caller-supplied `class_hash` from the syscall request and directly writes it into the contract's `StateEntry` in `contract_state_changes`, with no check that the hash corresponds to a declared class:

```cairo
func execute_replace_class{...}(contract_address: felt) {
    ...
    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );
    ...
}
```

The developer-acknowledged TODO comment at line 898 confirms the missing guard: `// TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.` [1](#0-0) 

The `StateEntry` struct pairs a `class_hash` with a `storage_ptr` (the contract's storage) and a `nonce`. After `replace_class` commits an invalid class hash, the storage (and any token balances or assets stored there) remains in the state tree, but the contract's execution logic is permanently severed — no entry point can ever be dispatched again. [2](#0-1) 

The state update is committed unconditionally through `state_update` → `squash_state_changes` → `compute_contract_state_commitment`, which hashes and Patricia-updates whatever `class_hash` is present in the `StateEntry`, with no post-hoc validation against the class tree. [3](#0-2) 

---

### Impact Explanation

Once a contract's class hash is set to an undeclared value and the block is committed:

- Every subsequent call to that contract address will fail at class resolution time, because the OS cannot find a class body for the stored hash.
- The contract's storage — including any ERC-20 balances, NFT ownership records, or other value-bearing state — is permanently locked with no recovery path.
- This constitutes **Critical: Permanent freezing of funds**.

---

### Likelihood Explanation

The `replace_class` syscall is callable by any contract during its own execution — no privileged role is required. A malicious actor who deploys a contract (or exploits a contract that exposes a `replace_class` call path) can trigger this with a single transaction. The attack requires no special access, no leaked keys, and no operator cooperation. The missing check is explicitly flagged in the source as a known TODO, confirming the guard is absent in the current production code.

---

### Recommendation

Before writing the new class hash into `contract_state_changes`, verify that the hash exists in `contract_class_changes` (i.e., it was declared in the current block) or in the committed class tree (i.e., it was declared in a prior block). Concretely, add a lookup analogous to the one used in `execute_get_class_hash_at`:

```cairo
// Verify the new class hash is declared.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
assert_not_zero(compiled_class_hash);
```

If the class was declared in a prior block (not present in the current block's `contract_class_changes`), a hint-assisted lookup against the committed class tree root should be performed and verified via the Patricia proof, consistent with how `SetPreimageForClassCommitments` is used elsewhere.

---

### Proof of Concept

1. Attacker deploys contract `C` that holds token balances (e.g., is an ERC-20 recipient).
2. Attacker sends an invoke transaction calling a function in `C` that internally issues the syscall `replace_class(0xdeadbeef)`, where `0xdeadbeef` is never declared.
3. `execute_replace_class` accepts the call, writes `class_hash=0xdeadbeef` into `C`'s `StateEntry`, and the block is committed via `state_update`.
4. In all subsequent blocks, any call to `C` fails at class resolution — the OS finds `class_hash=0xdeadbeef` in the state but no corresponding class body in the class tree.
5. All token balances and storage values in `C` are permanently inaccessible. [4](#0-3)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
