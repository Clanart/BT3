### Title
Missing Declared-Class Validation in `execute_replace_class` Allows Permanent Contract Freezing — (File: `crates/apollo_starknet_os_program/src/cairo/starkware/starknet/core/os/execution/syscall_impls.cairo`)

---

### Summary

The `execute_replace_class` syscall handler in the StarkNet OS updates a contract's class hash in `contract_state_changes` without verifying that the new class hash has been declared in `contract_class_changes`. A developer-acknowledged TODO comment at line 898 explicitly marks this missing check. Any contract can call `replace_class` with an arbitrary, undeclared class hash, permanently rendering itself uncallable and freezing any funds it holds.

---

### Finding Description

In `syscall_impls.cairo`, `execute_replace_class` reads the requested class hash directly from the syscall request and writes it into `contract_state_changes` with no validation that the hash corresponds to a declared class:

```cairo
func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    ...
    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );
    ...
}
``` [1](#0-0) 

The missing check is: before committing the new class hash, the OS must verify that `contract_class_changes[class_hash]` is non-zero (i.e., the class was previously declared via a `declare` transaction). This is the direct analog to M-02's missing `token.approve(curvePool)` — in both cases, a configuration update is committed without performing the prerequisite step that makes the new configuration usable.

When a subsequent call is made to the contract, the OS resolves the class hash from `contract_state_changes` and attempts to locate the corresponding compiled class in `compiled_class_facts_bundle`. If the class hash was never declared, no compiled class exists, the entry point lookup fails, and the contract becomes permanently uncallable. [2](#0-1) 

The class hash is resolved at call time from `contract_state_changes` and used directly to dispatch execution: [3](#0-2) 

---

### Impact Explanation

**Critical — Permanent freezing of funds.**

Once a contract's class hash is set to an undeclared value, every subsequent call to that contract fails at the OS level because no compiled class can be found for the hash. There is no recovery path: the contract cannot call `replace_class` again (it cannot execute any entry point), and no other contract can restore its class. Any ERC-20 tokens, ETH, or other assets held in the contract's storage are permanently inaccessible.

---

### Likelihood Explanation

The `replace_class` syscall is available to any Sierra contract without privilege restrictions. The attack requires only:

1. Deploying a contract (permissionless on StarkNet).
2. Calling `replace_class` with an arbitrary felt value that has never been declared.

A malicious contract can do this intentionally (e.g., a rug-pull variant where the deployer freezes user deposits). A legitimate contract with a flawed upgrade guard can do this accidentally. The OS provides zero enforcement to prevent it. [4](#0-3) 

---

### Recommendation

Before committing the new class hash in `execute_replace_class`, add a dict-read against `contract_class_changes` to assert the class has been declared (compiled class hash is non-zero):

```cairo
// Verify the new class hash is declared.
let (compiled_class_hash) = dict_read{dict_ptr=contract_class_changes}(key=class_hash);
with_attr error_message("Class hash is not declared.") {
    assert_not_zero(compiled_class_hash);
}
```

This mirrors the check already enforced during `deploy_contract`, where `state_entry.class_hash = UNINITIALIZED_CLASS_HASH` is asserted before writing a new class. [5](#0-4) 

---

### Proof of Concept

1. Attacker declares a legitimate class `C_valid` and deploys contract `V` (a vault) using it. Users deposit funds into `V`.
2. Attacker calls `replace_class(class_hash=0xdeadbeef)` from within `V`, where `0xdeadbeef` has never been declared.
3. The OS executes `execute_replace_class`: gas is deducted, the response header is written with `failure_flag=0`, and `contract_state_changes[V].class_hash` is updated to `0xdeadbeef`. No validation occurs.
4. The block is proven and the state is committed on-chain with `V`'s class hash set to `0xdeadbeef`.
5. Any subsequent `call_contract` or `invoke` targeting `V` reaches `execute_call_contract`, reads `class_hash=0xdeadbeef` from `contract_state_changes`, and attempts to dispatch execution. The OS cannot locate a compiled class for `0xdeadbeef` in `compiled_class_facts_bundle`. The call fails unconditionally.
6. All funds in `V`'s storage are permanently frozen with no recovery path. [6](#0-5)

func execute_replace_class{
    range_check_ptr,
    syscall_ptr: felt*,
    contract_state_changes: DictAccess*,
    revert_log: RevertLogEntry*,
}(contract_address: felt) {
    alloc_locals;
    let request = cast(syscall_ptr + RequestHeader.SIZE, ReplaceClassRequest*);

    // Reduce gas.
    let success = reduce_syscall_gas_and_write_response_header(
        total_gas_cost=REPLACE_CLASS_GAS_COST, request_struct_size=ReplaceClassRequest.SIZE
    );
    if (success == FALSE) {
        // Not enough gas to execute the syscall.
        return ();
    }

    let class_hash = request.class_hash;

    // TODO(Yoni, 1/1/2026): Check that there is a declared contract class with the given hash.
    local state_entry: StateEntry*;
    %{ GetContractAddressStateEntry %}

    tempvar new_state_entry = new StateEntry(
        class_hash=class_hash, storage_ptr=state_entry.storage_ptr, nonce=state_entry.nonce
    );

    dict_update{dict_ptr=contract_state_changes}(
        key=contract_address,
        prev_value=cast(state_entry, felt),
        new_value=cast(new_state_entry, felt),
    );

    assert [revert_log] = RevertLogEntry(selector=CHANGE_CLASS_ENTRY, value=state_entry.class_hash);
    let revert_log = &revert_log[1];

    return ();
}
