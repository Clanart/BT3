### Title
DataLayer `DeltaReader` Accepts Forged Internal Node Hashes Without Verification, Enabling Corrupted Tree Roots and Forged Inclusion Proofs — (`File: crates/chia-datalayer/src/merkle/deltas.rs`)

---

### Summary

`DeltaReader::new()` and `add_internal_nodes()` accept caller-supplied `internal_nodes: HashMap<Hash, (Hash, Hash)>` (a map of `node_hash → (left_child_hash, right_child_hash)`) without ever verifying that `node_hash == internal_hash(left_child_hash, right_child_hash)`. When `create_merkle_blob_and_filter_unused_nodes()` is subsequently called, `inner_build_blob_from_node_list()` stores each internal node with the caller-supplied (potentially forged) hash verbatim. The resulting `MerkleBlob` has a root hash and internal node hashes that do not correspond to the actual tree contents. `check_integrity()` does not catch this because the nodes are stored as non-dirty and `calculate_lazy_hashes()` only recomputes dirty nodes. A `ProofOfInclusion` generated from this blob will pass `valid()` while proving membership against a forged root.

---

### Finding Description

**Root cause — `DeltaReader::new()` / `add_internal_nodes()`:** [1](#0-0) 

The constructor inserts each `(hash, (left, right))` entry directly into `self.nodes` as `DeltaReaderNode::Internal { left, right }` keyed by `hash`. There is no assertion that `hash == internal_hash(left, right)`.

**Propagation — `inner_build_blob_from_node_list()`:** [2](#0-1) 

For an `Internal` variant, the function stores `InternalNode { hash: node_hash, ... }` where `node_hash` is the key that was looked up in the caller-supplied map — i.e., the forged hash — not the value computed from the children. The node is stored with `dirty: false`.

**Why `check_integrity()` does not catch it:** [3](#0-2) 

`check_integrity()` calls `calculate_lazy_hashes()` on a clone, then `check_just_integrity()`. Because the nodes were inserted as non-dirty, `calculate_lazy_hashes()` does not recompute their hashes. `check_just_integrity()` only validates parent-child pointer consistency and node counts — it never checks that `internal_node.hash == internal_hash(left_child.hash, right_child.hash)`. [4](#0-3) 

**Python binding exposure:** [5](#0-4) 

`DeltaReader.__new__`, `add_internal_nodes`, and `create_merkle_blob_and_filter_unused_nodes` are all directly callable from Python with arbitrary untrusted data.

**`ProofOfInclusion.valid()` passes on forged blobs:** [6](#0-5) 

`valid()` checks that each layer's `combined_hash` equals `calculate_internal_hash(existing_hash, ...)`. Since the proof is generated from the forged blob, all `combined_hash` values are internally consistent with the forged hashes, so `valid()` returns `true` even though the root does not correspond to any legitimate DataLayer state.

---

### Impact Explanation

An attacker who controls the `internal_nodes` argument to `DeltaReader` can construct a `MerkleBlob` whose `get_root_hash()` returns an arbitrary forged value. A `ProofOfInclusion` generated from this blob will pass `valid()` while asserting membership against that forged root. This allows proving that a key/value pair belongs to a DataLayer store with a root hash that does not actually contain that pair — i.e., forged inclusion proofs and corrupted tree roots from untrusted input.

This matches the allowed High impact: *"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."*

---

### Likelihood Explanation

The Python bindings expose `DeltaReader` as a first-class public API. Any Python caller (e.g., a DataLayer client or sync peer) that passes `internal_nodes` to `DeltaReader` can trigger this path. No privileged role or key is required — only the ability to call the Python API with attacker-controlled data.

---

### Recommendation

In `DeltaReader::new()` and `add_internal_nodes()`, verify each supplied internal node hash before inserting it:

```rust
let expected = internal_hash(&left, &right);
if hash != expected {
    return Err(Error::InvalidInternalNodeHash(hash, expected));
}
```

Alternatively, enforce the check inside `inner_build_blob_from_node_list()` after recursing into children: compute `internal_hash(left_child_hash, right_child_hash)` and assert it equals `node_hash` before storing the block.

---

### Proof of Concept

```python
from chia_rs.datalayer import DeltaReader, MerkleBlob
from chia_protocol import bytes32

# Legitimate leaf hashes
leaf_hash_a = bytes32(b'\x01' * 32)
leaf_hash_b = bytes32(b'\x02' * 32)

# Forged internal node hash — does NOT equal internal_hash(leaf_hash_a, leaf_hash_b)
forged_root = bytes32(b'\xff' * 32)

# DeltaReader accepts the forged mapping without verification
dr = DeltaReader(
    internal_nodes={forged_root: (leaf_hash_a, leaf_hash_b)},
    leaf_nodes={
        leaf_hash_a: (KeyId(1), ValueId(1)),
        leaf_hash_b: (KeyId(2), ValueId(2)),
    },
)

blob = dr.create_merkle_blob_and_filter_unused_nodes(
    forged_root, {forged_root, leaf_hash_a, leaf_hash_b}
)

# Root hash is the forged value, not internal_hash(leaf_hash_a, leaf_hash_b)
assert blob.get_root_hash() == forged_root  # passes

blob.calculate_lazy_hashes()  # no-op: nodes are non-dirty
proof = blob.get_proof_of_inclusion(KeyId(1))

# Proof validates against the forged root
assert proof.valid()           # passes — forged inclusion proof accepted
assert proof.root_hash() == forged_root  # forged root confirmed
```

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }

    pub fn valid(&self) -> bool {
