### Title
`inner_build_blob_from_node_list` Stores Attacker-Supplied Hash as Internal Node Hash Without Verification — (`crates/chia-datalayer/src/merkle/blob.rs`)

---

### Summary

`DeltaReader::new` and `py_add_internal_nodes` accept an `InternalNodesMap` whose keys are treated as the authoritative hash for each internal node. `inner_build_blob_from_node_list` writes those keys directly into `InternalNode.hash` without ever computing or comparing `internal_hash(left_child_hash, right_child_hash)`. The resulting `MerkleBlob` passes `check_integrity` and is never corrected by `calculate_lazy_hashes`, because all nodes are written with `dirty: false`. A malicious DataLayer peer can therefore deliver delta data that produces a locally-stored blob whose internal hashes are entirely attacker-controlled and inconsistent with the leaf contents.

---

### Finding Description

**`DeltaReader::new` — no hash validation on ingestion** [1](#0-0) 

The function iterates over the caller-supplied `InternalNodesMap` and inserts each `(hash, (left, right))` tuple verbatim into `self.nodes`. There is no assertion or computation that `hash == internal_hash(left, right)`.

**`inner_build_blob_from_node_list` — map key written directly as `InternalNode.hash`** [2](#0-1) 

After recursively building the left and right subtrees, the function constructs the `InternalNode` with `hash: node_hash` — the map key — and `dirty: false`. It never calls `internal_hash(left_child_hash, right_child_hash)` to verify or replace that value.

**`calculate_lazy_hashes` — skips non-dirty nodes** [3](#0-2) 

Only nodes whose `metadata.dirty` flag is `true` are recomputed. Because `inner_build_blob_from_node_list` sets `dirty: false` at line 1345, forged internal hashes are never overwritten.

**`check_just_integrity` — verifies structure, not hash correctness** [4](#0-3) 

`check_just_integrity` validates parent-child index relationships and cache consistency. It does not compare any `InternalNode.hash` against `internal_hash(left_child_hash, right_child_hash)`. A blob with entirely forged internal hashes passes this check without error.

---

### Impact Explanation

A malicious DataLayer mirror/peer supplies a delta where the root hash `H_fake` matches the on-chain committed root but `H_fake != internal_hash(H_left, H_right)`. The victim's node calls `DeltaReader::new({H_fake: (H_left, H_right)}, ...)` then `create_merkle_blob_and_filter_unused_nodes(H_fake, ...)`. [5](#0-4) 

The resulting `MerkleBlob` reports root hash `H_fake` (matching the on-chain commitment) but every internal node hash is attacker-chosen. Consequences:

- **Corrupted local DataLayer state**: the blob's internal structure does not reflect the actual leaf contents.
- **Invalid proofs of inclusion**: `get_proof_of_inclusion` reads `combined_hash` from the stored (forged) parent node; any proof emitted will fail verification by a correct verifier.
- **Silent acceptance**: `check_integrity` passes, `calculate_lazy_hashes` is a no-op on the forged nodes, so the corruption is invisible to all existing guards.

This matches the allowed High impact: *"DataLayer Merkle proof/blob/delta logic accepts forged inclusion/exclusion, corrupts tree roots, or lets untrusted input prove invalid state."*

---

### Likelihood Explanation

`DeltaReader::new` and `py_add_internal_nodes` are public entry points reachable from the Python DataLayer sync layer, which constructs `InternalNodesMap` from peer-provided network data. Any DataLayer mirror (an unprivileged network peer) can supply the forged map. No key material, governance role, or chain reorg is required. The Rust layer provides zero defence; the only possible mitigation would be in the Python caller, which is outside the audited scope.

---

### Recommendation

Inside `inner_build_blob_from_node_list`, after the left and right child indexes are resolved, compute the expected hash and reject mismatches:

```rust
// after left_index and right_index are obtained:
let left_hash  = self.get_hash(left_index)?;
let right_hash = self.get_hash(right_index)?;
let expected   = internal_hash(&left_hash, &right_hash);
if expected != node_hash {
    return Err(Error::InternalHashMismatch(node_hash, expected));
}
```

Alternatively, discard the caller-supplied hash entirely and always derive it from the children, which is the approach used by every other internal-node construction path in the codebase (e.g., `insert_second`, `batch_insert`). [6](#0-5)

### Proof of Concept

```rust
use std::collections::{HashMap, HashSet};
use chia_datalayer::{
    Hash, KeyId, ValueId, InternalNodesMap, LeafNodesMap,
    merkle::{blob::internal_hash, deltas::DeltaReader},
    TreeIndex,
};

// Construct two valid leaf hashes
let h_left  = Hash::from([1u8; 32]);
let h_right = Hash::from([2u8; 32]);

// H_fake is deliberately NOT internal_hash(h_left, h_right)
let h_fake  = Hash::from([0xffu8; 32]);
assert_ne!(h_fake, internal_hash(&h_left, &h_right));

let mut internal_nodes: InternalNodesMap = HashMap::new();
internal_nodes.insert(h_fake, (h_left, h_right));

let mut leaf_nodes: LeafNodesMap = HashMap::new();
leaf_nodes.insert(h_left,  (KeyId(0), ValueId(1)));
leaf_nodes.insert(h_right, (KeyId(2), ValueId(3)));

let mut delta_reader = DeltaReader::new(internal_nodes, leaf_nodes).unwrap();
let blob = delta_reader
    .create_merkle_blob_and_filter_unused_nodes(h_fake, &HashSet::new())
    .unwrap();

// Root hash is h_fake — not the correct internal_hash of the children
let root = blob.get_hash_at_index(TreeIndex(0)).unwrap().unwrap();
assert_eq!(root, h_fake);
assert_ne!(root, internal_hash(&h_left, &h_right));

// check_integrity passes — the corruption is invisible to all existing guards
blob.check_integrity().unwrap();
```

The `check_integrity` call succeeds, confirming the forged blob is accepted as valid by all existing guards.

    fn check_just_integrity(&self) -> Result<(), Error> {
        let mut leaf_count: usize = 0;
        let mut internal_count: usize = 0;
        let mut child_to_parent: HashMap<TreeIndex, TreeIndex> = HashMap::new();

        for item in ParentFirstIterator::new(&self.blob, None) {
            let (index, block) = item?;
            if let Some(parent) = block.node.parent().0 {
                if child_to_parent.remove(&index) != Some(parent) {
                    return Err(Error::IntegrityParentChildMismatch(index));
                }
            }
            match block.node {
                Node::Internal(node) => {
                    internal_count += 1;
                    child_to_parent.insert(node.left, index);
                    child_to_parent.insert(node.right, index);
                }
                Node::Leaf(node) => {
                    leaf_count += 1;
                    let cached_index = self
                        .block_status_cache
                        .get_index_by_key(node.key)
                        .ok_or(Error::IntegrityKeyNotInCache(node.key))?;
                    if *cached_index != index {
                        return Err(Error::IntegrityKeyToIndexCacheIndex(
                            node.key,
                            index,
                            *cached_index,
                        ));
                    }
                    assert!(
                        !self.block_status_cache.is_index_free(index),
                        "{}",
                        format!("active index found in free index list: {index:?}")
                    );
                }
            }
        }

        let key_to_index_cache_length = self.block_status_cache.key_to_index.len();
        if leaf_count != key_to_index_cache_length {
            return Err(Error::IntegrityKeyToIndexCacheLength(
                leaf_count,
                key_to_index_cache_length,
            ));
        }
        let leaf_hash_to_index_cache_length = self.block_status_cache.leaf_hash_to_index.len();
        if leaf_count != leaf_hash_to_index_cache_length {
            return Err(Error::IntegrityLeafHashToIndexCacheLength(
                leaf_count,
                leaf_hash_to_index_cache_length,
            ));
        }
        let total_count = leaf_count + internal_count + self.block_status_cache.free_index_count();
        let extend_index = self.extend_index();
        if total_count != extend_index.0 as usize {
            return Err(Error::IntegrityTotalNodeCount(extend_index, total_count));
        }
        if !child_to_parent.is_empty() {
            return Err(Error::IntegrityUnmatchedChildParentRelationships(
                child_to_parent.len(),
            ));
        }

        Ok(())
    }
